# Projektdefinition

## Lernziele

Nach diesem Kapitel solltest du:
- Ein Projekt von einer routinemäßigen Aufgabe unterscheiden können
- Die fünf Merkmale eines Projekts kennen
- Beispiele für Projekte und Nicht-Projekte nennen können

---

## Kerninhalt

### Was ist ein Projekt?

Ein **Projekt** ist ein Vorhaben mit folgenden charakteristischen Merkmalen:

### Die 5 Merkmale eines Projekts

| Merkmal | Erklärung |
|---------|-----------|
| **Einmaligkeit** | Das Vorhaben ist neu und wird nicht routinemäßig wiederholt |
| **Zielgerichtet** | Es gibt ein klares, definiertes Ziel |
| **Zeitlich begrenzt** | Es gibt einen definierten Anfangs- und Endtermin |
| **Begrenzte Ressourcen** | Budget, Personal und Material sind limitiert |
| **Komplexität** | Das Vorhaben ist nicht trivial und erfordert Koordination |

### Projekt vs. Routineaufgabe

| Projekt | Routineaufgabe |
|---------|----------------|
| Einmalig | Wiederkehrend |
| Definierter Endtermin | Fortlaufend |
| Neuartig | Standardisiert |
| Ergebnisoffen | Vorhersehbares Ergebnis |
| Eigene Organisation | In Linienorganisation integriert |

### Beispiele

**Projekte:**
- Einführung eines neuen ERP-Systems
- Aufbau eines neuen Firmenstandorts
- Entwicklung einer neuen App
- Migration auf eine neue Serverinfrastruktur

**Keine Projekte:**
- Tägliche Datensicherung
- Monatliche Gehaltsabrechnung
- Regelmäßige Wartung von Systemen
- Beantwortung von Support-Anfragen

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Projekt** | Einmaliges, zeitlich begrenztes Vorhaben mit definiertem Ziel |
| **Projektmanagement** | Planen, Steuern und Kontrollieren von Projekten |
| **Stakeholder** | Alle Personen/Gruppen, die vom Projekt betroffen sind oder Einfluss haben |
| **Projektleiter** | Verantwortliche Person für die Projektdurchführung |

---

## Prüfungstipps

**Typische Prüfungsaufgabe:**
> "Überprüfen Sie den folgenden Text auf fehlerhafte Aussagen:
> Ein Projekt ist ein immer wiederkehrendes Vorhaben mit einem klaren Ziel sowie einem Anfangstermin und variablem Endtermin."

**Lösung:** Zwei Fehler:
1. "immer wiederkehrend" - falsch, Projekte sind einmalig
2. "variablem Endtermin" - falsch, Projekte haben einen definierten Endtermin

## Übungsaufgaben

### Aufgabe 1
Entscheiden Sie, ob es sich um ein Projekt handelt:
a) Einführung einer neuen Buchhaltungssoftware
b) Tägliches Backup der Server
c) Umstellung der Firmenwebsite auf ein neues CMS
d) Beantwortung von E-Mails

**Lösung:**
a) ✓ Projekt - einmalig, zeitlich begrenzt, zielgerichtet
b) ✗ Routineaufgabe - wiederkehrend
c) ✓ Projekt - einmalig, zeitlich begrenzt
d) ✗ Routineaufgabe - wiederkehrend

### Aufgabe 2
Nennen Sie drei Merkmale, die ein Projekt von einer Routineaufgabe unterscheiden.

**Lösung:**
1. Einmaligkeit
2. Zeitliche Begrenzung (definierter Endtermin)
3. Neuartigkeit/Komplexität
