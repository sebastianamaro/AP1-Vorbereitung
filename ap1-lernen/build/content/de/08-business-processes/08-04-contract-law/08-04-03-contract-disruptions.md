# Vertragsstörungen und Mängelrechte

## Lernziele

Nach diesem Abschnitt kannst du:
- Arten von Vertragsstörungen unterscheiden
- Mängelrechte des Käufers erklären
- Die Reihenfolge der Mängelrechte nennen
- Gewährleistung und Garantie unterscheiden

---

## Kerninhalt

### Arten von Vertragsstörungen

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VERTRAGSSTÖRUNGEN                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. MANGELHAFTE LIEFERUNG (Schlechtleistung)                │    │
│  │     → Ware hat Mängel                                       │    │
│  │     → Beispiel: Laptop startet nicht                        │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  2. FALSCHLIEFERUNG                                         │    │
│  │     → Falsche Ware geliefert                                │    │
│  │     → Beispiel: Server statt Laptop                         │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  3. MINDERLIEFERUNG                                         │    │
│  │     → Zu wenig geliefert                                    │    │
│  │     → Beispiel: 8 statt 10 Laptops                          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  4. LIEFERUNGSVERZUG                                        │    │
│  │     → Lieferung verspätet                                   │    │
│  │     → Beispiel: Lieferung 2 Wochen nach Termin              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  5. ZAHLUNGSVERZUG                                          │    │
│  │     → Käufer zahlt nicht rechtzeitig                        │    │
│  │     → Beispiel: Rechnung nicht bezahlt nach Fälligkeit      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Mängelarten

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MÄNGELARTEN                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  SACHMANGEL (§ 434 BGB)                                             │
│  → Die Sache entspricht nicht der vereinbarten Beschaffenheit       │
│                                                                      │
│  Ein Sachmangel liegt vor, wenn:                                    │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. Vereinbarte Beschaffenheit fehlt                        │    │
│  │     → Z.B. "16 GB RAM" vereinbart, nur 8 GB verbaut         │    │
│  │                                                              │    │
│  │  2. Gewöhnliche Verwendung nicht möglich                    │    │
│  │     → Z.B. PC startet nicht                                 │    │
│  │                                                              │    │
│  │  3. Beschaffenheit weicht von Werbung ab                    │    │
│  │     → Z.B. "10 Stunden Akku" versprochen, nur 5 Stunden     │    │
│  │                                                              │    │
│  │  4. Montage fehlerhaft (bei vereinbarter Montage)           │    │
│  │     → Z.B. Server falsch eingebaut                          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  RECHTSMANGEL (§ 435 BGB)                                           │
│  → Dritte können Rechte an der Sache geltend machen                 │
│  → Z.B. Software ohne gültige Lizenz, gestohlene Ware               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Mängelrechte des Käufers (§ 437 BGB)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MÄNGELRECHTE                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Bei einem Mangel hat der Käufer folgende Rechte:                   │
│                                                                      │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  1. NACHERFÜLLUNG (§ 439 BGB)         ← VORRANG!            │   │
│  │     a) Nachbesserung (Reparatur)                            │   │
│  │        ODER                                                 │   │
│  │     b) Ersatzlieferung (neue Ware)                          │   │
│  │                                                              │   │
│  │     → Käufer hat die WAHL                                   │   │
│  │     → Verkäufer kann nur ablehnen, wenn unverhältnismäßig   │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│                      │ Erst wenn Nacherfüllung                       │
│                      │ FEHLSCHLÄGT (2 Versuche)                      │
│                      │ oder VERWEIGERT wird                          │
│                      ▼                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  2. RÜCKTRITT (§ 323 BGB)                                   │   │
│  │     → Vertrag rückgängig machen                             │   │
│  │     → Ware zurück, Geld zurück                              │   │
│  │     → Nur bei erheblichen Mängeln!                          │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  3. MINDERUNG (§ 441 BGB)                                   │   │
│  │     → Kaufpreis herabsetzen                                 │   │
│  │     → Ware behalten, weniger zahlen                         │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  4. SCHADENSERSATZ (§ 280 BGB)                              │   │
│  │     → Bei Verschulden des Verkäufers                        │   │
│  │     → Ersatz von Folgeschäden                               │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Reihenfolge der Mängelrechte

```
┌─────────────────────────────────────────────────────────────────────┐
│          STUFENVERHÄLTNIS DER MÄNGELRECHTE                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  STUFE 1: NACHERFÜLLUNG                                             │
│  ─────────────────────────────────────────────────────────────────  │
│  → Immer zuerst!                                                    │
│  → Verkäufer hat das Recht auf Nachbesserung                        │
│  → Käufer muss Frist setzen (i.d.R. 2 Versuche)                     │
│                                                                      │
│           │                                                         │
│           │ Nacherfüllung fehlgeschlagen?                           │
│           │ Verkäufer verweigert?                                   │
│           │ Unzumutbar?                                             │
│           ▼                                                         │
│                                                                      │
│  STUFE 2: RÜCKTRITT oder MINDERUNG                                  │
│  ─────────────────────────────────────────────────────────────────  │
│  → Käufer kann wählen                                               │
│  → Rücktritt nur bei erheblichem Mangel                             │
│  → Minderung auch bei geringem Mangel                               │
│                                                                      │
│  ZUSÄTZLICH: SCHADENSERSATZ                                         │
│  ─────────────────────────────────────────────────────────────────  │
│  → Kann zusätzlich verlangt werden                                  │
│  → Nur bei Verschulden des Verkäufers                               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Gewährleistung vs. Garantie

```
┌─────────────────────────────────────────────────────────────────────┐
│           GEWÄHRLEISTUNG vs. GARANTIE                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  GEWÄHRLEISTUNG                    GARANTIE                         │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  Gesetzlich geregelt               Freiwillig vom Hersteller        │
│  (§§ 434 ff. BGB)                                                   │
│                                                                      │
│  24 Monate ab Übergabe             Vom Hersteller festgelegt        │
│  (bei Neuwaren, B2C)               (z.B. 3 Jahre, 5 Jahre)          │
│                                                                      │
│  Verkäufer haftet                  Hersteller haftet                │
│                                    (oder Verkäufer, je nach Garantie)│
│                                                                      │
│  Mangel muss bei Übergabe          Defekt während Garantiezeit      │
│  vorhanden gewesen sein            entstanden                       │
│                                                                      │
│  Beweislast nach 12 Monaten        Oft einfachere Abwicklung        │
│  beim Käufer (Beweislastumkehr)    (Kulanz)                         │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  MERKE: Garantie ersetzt NICHT die Gewährleistung!                  │
│         Gewährleistung ist Recht, Garantie ist Zusatzleistung       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Fristen

| Frist | Dauer | Bemerkung |
|-------|-------|-----------|
| **Gewährleistungsfrist** | 24 Monate | Bei Neuwaren B2C |
| **Gewährleistungsfrist** | 12 Monate | Bei Gebrauchtwaren B2C |
| **Gewährleistungsfrist B2B** | 12 Monate | Bei Geschäftskunden (verkürzbar) |
| **Beweislastumkehr** | 12 Monate | Danach muss Käufer beweisen |
| **Rügefrist B2B** | Unverzüglich | Kaufmann muss sofort rügen! |

### Verzug

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VERZUG                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  LIEFERUNGSVERZUG (Verkäufer liefert nicht rechtzeitig)             │
│  ──────────────────────────────────────────────────────────────     │
│  Voraussetzungen:                                                   │
│  1. Fälligkeit (Liefertermin überschritten)                         │
│  2. Mahnung (oder Termin war kalendermäßig bestimmt)                │
│  3. Verschulden des Verkäufers                                      │
│                                                                      │
│  Rechte des Käufers:                                                │
│  → Schadensersatz neben der Leistung                                │
│  → Nach Fristsetzung: Schadensersatz statt der Leistung             │
│  → Rücktritt vom Vertrag                                            │
│                                                                      │
│  ZAHLUNGSVERZUG (Käufer zahlt nicht rechtzeitig)                    │
│  ──────────────────────────────────────────────────────────────     │
│  Voraussetzungen:                                                   │
│  1. Fälligkeit (Zahlungsziel überschritten)                         │
│  2. Mahnung (oder 30 Tage nach Rechnungserhalt)                     │
│                                                                      │
│  Rechte des Verkäufers:                                             │
│  → Verzugszinsen (5% über Basiszins, B2B: 9%)                       │
│  → Mahnkosten                                                       │
│  → Schadensersatz                                                   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Sachmangel** | Ware entspricht nicht der Vereinbarung |
| **Rechtsmangel** | Dritte haben Rechte an der Sache |
| **Nacherfüllung** | Reparatur oder Ersatzlieferung |
| **Rücktritt** | Vertrag rückgängig machen |
| **Minderung** | Kaufpreis reduzieren |
| **Gewährleistung** | Gesetzliche Mängelhaftung |
| **Garantie** | Freiwillige Zusatzleistung |
| **Verzug** | Verspätung bei Leistung/Zahlung |

---

## Prüfungstipps

1. **Reihenfolge merken:**
   - **N**acherfüllung zuerst → dann **R**ücktritt/**M**inderung/**S**chadenersatz

2. **Gewährleistung ≠ Garantie:**
   - Gewährleistung = gesetzlich
   - Garantie = freiwillig

3. **Typische Prüfungsfragen:**
   - Welche Rechte hat der Käufer bei einem Mangel?
   - Unterschied Gewährleistung/Garantie?
   - Was ist ein Sachmangel?

## Übungsaufgaben

### Aufgabe 1: Mängelrechte anwenden

ConSystem hat bei einem Händler 10 Laptops gekauft. Bei 3 Laptops funktioniert die Tastatur nicht.

a) Liegt ein Mangel vor?
b) Welches Recht sollte ConSystem zuerst geltend machen?
c) Was passiert, wenn die Nachbesserung zweimal fehlschlägt?

<details>
<summary>Lösung anzeigen</summary>

a) **Ja, ein Sachmangel liegt vor.**
- Die Laptops entsprechen nicht der gewöhnlichen Verwendung
- Eine Tastatur muss funktionieren

b) **Nacherfüllung (Nachbesserung oder Ersatzlieferung)**
- ConSystem sollte den Händler auffordern, die Laptops zu reparieren ODER neue zu liefern
- Frist setzen (z.B. 14 Tage)

c) **Nach 2 fehlgeschlagenen Versuchen:**
- ConSystem kann vom Kaufvertrag (für die 3 Laptops) zurücktreten
- ODER Minderung des Kaufpreises verlangen
- ODER Schadensersatz fordern (z.B. Kosten für Ersatzgeräte)

</details>

### Aufgabe 2: Gewährleistung vs. Garantie

Ein Kunde kauft bei ConSystem einen PC. Nach 14 Monaten geht das Netzteil kaputt.

a) Kann der Kunde Gewährleistung geltend machen?
b) Wer muss beweisen, dass der Mangel schon bei Übergabe vorhanden war?
c) Was wäre anders, wenn der Hersteller 3 Jahre Garantie gibt?

<details>
<summary>Lösung anzeigen</summary>

a) **Ja, die Gewährleistungsfrist (24 Monate) ist noch nicht abgelaufen.**

b) **Der KUNDE muss beweisen.**
- Die Beweislastumkehr gilt nur 12 Monate
- Nach 14 Monaten muss der Kunde nachweisen, dass das Netzteil schon bei Übergabe mangelhaft war (z.B. durch Gutachten)
- Das ist in der Praxis schwierig

c) **Bei 3 Jahren Herstellergarantie:**
- Der Kunde kann sich an den Hersteller wenden
- Garantie ist einfacher abzuwickeln (oft ohne Nachweis)
- Der Hersteller tauscht/repariert nach seinen Garantiebedingungen
- Vorteil: Keine komplizierte Beweisführung

</details>

### Aufgabe 3: Lieferungsverzug

ConSystem hat Server für den 15.01. bestellt. Heute ist der 25.01. und die Server sind noch nicht da. Der Händler reagiert nicht auf Anfragen.

Welche Schritte sollte ConSystem enterprise?

<details>
<summary>Lösung anzeigen</summary>

**Schritt 1: Verzug feststellen**
- Liefertermin war kalendermäßig bestimmt (15.01.)
- Mahnung ist daher nicht nötig (§ 286 Abs. 2 BGB)
- Händler ist seit 16.01. in Verzug

**Schritt 2: Frist setzen**
- Schriftliche Mahnung mit Fristsetzung
- "Wir fordern Sie auf, die Server bis zum 01.02. zu liefern. Nach Ablauf dieser Frist werden wir vom Vertrag zurücktreten."

**Schritt 3: Nach Fristablauf**
- Rücktritt vom Vertrag ODER
- Schadensersatz statt der Leistung ODER
- Ersatzprocurement bei anderem Händler + Schadensersatz

**Schritt 4: Dokumentieren**
- Alle Schreiben aufbewahren
- Mehrkosten für Ersatzprocurement dokumentieren
- Ggf. Verzögerungsschäden (Projekt kann nicht starten) berechnen

</details>
