# Service Level Agreement (SLA)

## Lernziele

Nach diesem Abschnitt kannst du:
- Den Begriff SLA definieren und erklären
- Wichtige SLA-Kennzahlen benennen
- Verfügbarkeit berechnen
- Reaktions- und Lösungszeiten unterscheiden

---

## Kerninhalt

### Was ist ein SLA?

Ein **Service Level Agreement (SLA)** ist eine vertragliche Vereinbarung zwischen Dienstleister und Kunde über die Qualität und den Umfang von IT-Dienstleistungen.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SERVICE LEVEL AGREEMENT                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────┐                      ┌─────────────────┐       │
│  │   DIENSTLEISTER │  ←── Vereinbarung ──→│     KUNDE       │       │
│  │   (IT-Anbieter) │                      │   (Auftraggeber)│       │
│  └─────────────────┘                      └─────────────────┘       │
│                                                                      │
│  SLA REGELT:                                                        │
│  ──────────────────────────────────────────────────────────────     │
│  → WAS wird geleistet? (Leistungsumfang)                           │
│  → WIE GUT wird geleistet? (Qualitätskriterien)                    │
│  → WIE SCHNELL wird reagiert? (Reaktionszeiten)                    │
│  → WAS PASSIERT bei Nichteinhaltung? (Konsequenzen)                │
│                                                                      │
│  ARTEN VON SLAs:                                                    │
│  ──────────────────────────────────────────────────────────────     │
│  → Externer SLA: Mit Kunden/Lieferanten                            │
│  → Interner SLA: Zwischen Abteilungen (OLA)                        │
│  → Underpinning Contract: Mit Zulieferern                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Bestandteile eines SLA

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SLA-BESTANDTEILE                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  1. VERTRAGSPARTEIEN                                                │
│     → Wer ist Dienstleister?                                        │
│     → Wer ist Kunde?                                                │
│     → Ansprechpartner auf beiden Seiten                            │
│                                                                      │
│  2. LEISTUNGSBESCHREIBUNG                                           │
│     → Welche Services sind abgedeckt?                               │
│     → Welche NICHT (Abgrenzung wichtig!)                           │
│     → Servicezeiten (z.B. 24/7, Mo-Fr 8-18 Uhr)                    │
│                                                                      │
│  3. SERVICE LEVEL (Qualitätskennzahlen)                            │
│     → Verfügbarkeit (z.B. 99,9%)                                   │
│     → Reaktionszeit (z.B. 1 Stunde)                                │
│     → Lösungszeit (z.B. 4 Stunden)                                 │
│     → Performance-Werte                                             │
│                                                                      │
│  4. MESSUNG & REPORTING                                             │
│     → Wie wird gemessen?                                            │
│     → Welche Tools werden verwendet?                                │
│     → Wie oft wird berichtet?                                       │
│                                                                      │
│  5. KONSEQUENZEN                                                    │
│     → Pönalen bei Nichteinhaltung                                  │
│     → Bonus bei Übererfüllung                                      │
│     → Eskalationsprozesse                                          │
│                                                                      │
│  6. LAUFZEIT & KÜNDIGUNG                                           │
│     → Vertragsbeginn und -ende                                     │
│     → Kündigungsfristen                                            │
│     → Änderungsverfahren                                           │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Reaktionszeit vs. Lösungszeit

```
┌─────────────────────────────────────────────────────────────────────┐
│           REAKTIONSZEIT vs. LÖSUNGSZEIT                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Zeitachse:                                                         │
│  ═══════════════════════════════════════════════════════════════   │
│                                                                      │
│  Störung     Erste          Problem                                 │
│  gemeldet    Reaktion       gelöst                                  │
│     │           │              │                                    │
│     ▼           ▼              ▼                                    │
│     ●───────────●──────────────●                                    │
│     │           │              │                                    │
│     │◄─────────►│              │                                    │
│     │ REAKTIONS-│              │                                    │
│     │   ZEIT    │              │                                    │
│     │                          │                                    │
│     │◄────────────────────────►│                                    │
│     │      LÖSUNGSZEIT         │                                    │
│                                                                      │
│  REAKTIONSZEIT:                                                     │
│  → Zeit bis zur ersten qualifizierten Rückmeldung                  │
│  → Bestätigung, dass Anfrage eingegangen ist                       │
│  → Mitarbeiter hat sich dem Problem angenommen                     │
│  → NICHT die Zeit bis zur Lösung!                                  │
│                                                                      │
│  LÖSUNGSZEIT:                                                       │
│  → Zeit bis das Problem tatsächlich behoben ist                    │
│  → Oder: Workaround bereitgestellt                                 │
│  → Abhängig von Komplexität und Priorität                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### SLA nach Priorität (Beispiel)

| Priorität | Beschreibung | Reaktionszeit | Lösungszeit |
|-----------|--------------|---------------|-------------|
| **Kritisch** | Totalausfall, Firma steht | 15 Minuten | 4 Stunden |
| **Hoch** | Wichtige Funktion gestört | 1 Stunde | 8 Stunden |
| **Mittel** | Eingeschränkte Nutzung | 4 Stunden | 24 Stunden |
| **Niedrig** | Geringe Auswirkung | 8 Stunden | 5 Werktage |

### Verfügbarkeit berechnen

```
┌─────────────────────────────────────────────────────────────────────┐
│           VERFÜGBARKEIT                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  FORMEL:                                                            │
│  ──────────────────────────────────────────────────────────────     │
│                                                                      │
│                    Betriebszeit - Ausfallzeit                       │
│  Verfügbarkeit = ────────────────────────────── × 100%              │
│                         Betriebszeit                                │
│                                                                      │
│  ODER:                                                              │
│                                                                      │
│                    Gesamtzeit - Ausfallzeit                         │
│  Verfügbarkeit = ────────────────────────────── × 100%              │
│                         Gesamtzeit                                  │
│                                                                      │
│  BEISPIELRECHNUNG:                                                  │
│  ──────────────────────────────────────────────────────────────     │
│                                                                      │
│  Servicezeit: 24/7 = 24 × 365 = 8.760 Stunden/Jahr                 │
│  Ausfallzeit im Jahr: 8,76 Stunden                                 │
│                                                                      │
│                    8.760 - 8,76                                     │
│  Verfügbarkeit = ──────────────── × 100% = 99,9%                   │
│                       8.760                                         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Die "Neunen" der Verfügbarkeit

```
┌─────────────────────────────────────────────────────────────────────┐
│           VERFÜGBARKEITSKLASSEN                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Verfügbarkeit │ Ausfallzeit/Jahr │ Ausfallzeit/Monat │ Einsatz    │
│  ══════════════╪══════════════════╪═══════════════════╪════════════│
│  99%           │ 3,65 Tage        │ 7,2 Stunden       │ Basis      │
│  99,5%         │ 1,83 Tage        │ 3,6 Stunden       │ Standard   │
│  99,9%         │ 8,76 Stunden     │ 43,8 Minuten      │ Gehoben    │
│  99,99%        │ 52,6 Minuten     │ 4,4 Minuten       │ Hoch       │
│  99,999%       │ 5,26 Minuten     │ 26 Sekunden       │ Sehr hoch  │
│                                                                      │
│  MERKHILFE: "Die Neunen"                                           │
│  ──────────────────────────────────────────────────────────────     │
│  99,9%   = "Drei Neunen" = ~8,7 Stunden Ausfall/Jahr               │
│  99,99%  = "Vier Neunen" = ~52 Minuten Ausfall/Jahr                │
│  99,999% = "Fünf Neunen" = ~5 Minuten Ausfall/Jahr                 │
│                                                                      │
│  ACHTUNG: Jede weitere "9" bedeutet ~10× weniger Ausfallzeit       │
│           aber auch ~10× höhere Kosten!                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Servicezeiten beachten

```
┌─────────────────────────────────────────────────────────────────────┐
│           SERVICEZEITEN                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  SLA-Zeiten gelten nur innerhalb der vereinbarten Servicezeiten!   │
│                                                                      │
│  BEISPIEL 1: Servicezeit 24/7                                      │
│  ──────────────────────────────────────────────────────────────     │
│  → 168 Stunden/Woche (24h × 7 Tage)                                │
│  → Reaktionszeit 1h gilt IMMER (auch nachts, Wochenende)           │
│                                                                      │
│  BEISPIEL 2: Servicezeit Mo-Fr 8-18 Uhr                            │
│  ──────────────────────────────────────────────────────────────     │
│  → 50 Stunden/Woche (10h × 5 Tage)                                 │
│  → Reaktionszeit 1h gilt NUR in dieser Zeit                        │
│                                                                      │
│  PRAXISFALL:                                                        │
│  ──────────────────────────────────────────────────────────────     │
│  Störung gemeldet: Freitag 17:30 Uhr                               │
│  Servicezeit: Mo-Fr 8-18 Uhr                                       │
│  Reaktionszeit: 1 Stunde                                           │
│                                                                      │
│  → Verbleibende Zeit Freitag: 30 Minuten                           │
│  → Reaktion muss erfolgen bis: Montag 8:30 Uhr                     │
│    (30 min am Fr + 30 min am Mo = 1 Stunde)                        │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Pönalen (Vertragsstrafen)

```
┌─────────────────────────────────────────────────────────────────────┐
│           PÖNALEN BEI SLA-VERLETZUNG                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Bei Nichteinhaltung des SLA können Vertragsstrafen fällig werden: │
│                                                                      │
│  TYPISCHE REGELUNGEN:                                               │
│  ──────────────────────────────────────────────────────────────     │
│  → Gutschrift auf nächste Rechnung                                 │
│  → Prozentuale Minderung des Monatsentgelts                        │
│  → Kostenlose Zusatzleistungen                                     │
│  → Sonderkündigungsrecht ab bestimmter Schwelle                    │
│                                                                      │
│  BEISPIEL:                                                          │
│  ──────────────────────────────────────────────────────────────     │
│  Vereinbarte Verfügbarkeit: 99,9%                                  │
│  Tatsächliche Verfügbarkeit: 99,5%                                 │
│  Monatsentgelt: 1.000 €                                            │
│                                                                      │
│  → Verfügbarkeit um 0,4% unterschritten                            │
│  → Pönale: 10% Gutschrift pro 0,1% Unterschreitung                 │
│  → 4 × 10% = 40% Gutschrift                                        │
│  → Gutschrift: 400 €                                               │
│                                                                      │
│  WICHTIG:                                                           │
│  → Pönalen sind KEINE Entschädigung für Folgeschäden!             │
│  → Pönalen sind meist gedeckelt (z.B. max. 1 Monatsentgelt)       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **SLA** | Service Level Agreement - Vertragliche Vereinbarung über Servicequalität |
| **OLA** | Operational Level Agreement - Interne Vereinbarung zwischen Abteilungen |
| **Reaktionszeit** | Zeit bis zur ersten qualifizierten Rückmeldung |
| **Lösungszeit** | Zeit bis zur Behebung des Problems |
| **Verfügbarkeit** | Prozentsatz der tatsächlichen Betriebszeit |
| **Pönale** | Vertragsstrafe bei SLA-Verletzung |
| **Servicezeit** | Zeitraum, in dem der Service zur Verfügung steht |

---

## Prüfungstipps

1. **Verfügbarkeit berechnen:**
   - Formel: (Gesamtzeit - Ausfallzeit) / Gesamtzeit × 100%
   - "Drei Neunen" (99,9%) = ca. 8,7 Stunden Ausfall/Jahr

2. **Reaktions- vs. Lösungszeit:**
   - Reaktionszeit = Erste Rückmeldung
   - Lösungszeit = Problem behoben
   - Reaktionszeit ist IMMER kürzer als Lösungszeit

3. **Servicezeiten beachten:**
   - SLA-Zeiten gelten nur innerhalb der Servicezeiten!
   - 24/7 vs. Geschäftszeiten unterscheiden

## Übungsaufgaben

### Aufgabe 1: Verfügbarkeit berechnen

Ein Server hat folgende Daten:
- Servicezeit: 24/7
- Geplante Wartung im Jahr: 12 Stunden
- Ungeplante Ausfälle im Jahr: 4,5 Stunden

Berechnen Sie die Verfügbarkeit. Geplante Wartung zählt NICHT als Ausfall.

<details>
<summary>Lösung anzeigen</summary>

**Berechnung:**

Gesamtzeit/Jahr: 24 × 365 = 8.760 Stunden
Ausfallzeit (nur ungeplant): 4,5 Stunden

Verfügbarkeit = (8.760 - 4,5) / 8.760 × 100%
             = 8.755,5 / 8.760 × 100%
             = **99,95%**

Das entspricht "fast vier Neunen" und liegt über dem typischen SLA von 99,9%.

</details>

### Aufgabe 2: Reaktionszeit prüfen

Ein Incident wird am Donnerstag um 17:00 Uhr gemeldet.
- Servicezeit: Mo-Fr 8-18 Uhr
- Vereinbarte Reaktionszeit: 2 Stunden

Bis wann muss der IT-Support reagieren?

<details>
<summary>Lösung anzeigen</summary>

**Analyse:**

Meldung: Donnerstag 17:00 Uhr
Servicezeit endet: 18:00 Uhr

Verbleibende Servicezeit Donnerstag: 1 Stunde
Fehlende Zeit: 2h - 1h = 1 Stunde

Diese 1 Stunde läuft am Freitag weiter.
Freitag Servicezeit beginnt: 8:00 Uhr

**Antwort: Reaktion muss erfolgen bis Freitag 9:00 Uhr**

</details>

### Aufgabe 3: SLA-Verletzung bewerten

Ein IT-Dienstleister hat folgendes SLA vereinbart:
- Monatliche Verfügbarkeit: 99,5%
- Pönale: 5% Gutschrift pro 0,1% Unterschreitung
- Maximale Pönale: 25% des Monatsentgelts
- Monatsentgelt: 2.000 €

Im März betrug die tatsächliche Verfügbarkeit 99,1%.

Wie hoch ist die Gutschrift?

<details>
<summary>Lösung anzeigen</summary>

**Berechnung:**

Vereinbarte Verfügbarkeit: 99,5%
Tatsächliche Verfügbarkeit: 99,1%
Unterschreitung: 99,5% - 99,1% = 0,4%

Pönale: 0,4% / 0,1% × 5% = 4 × 5% = 20%

Prüfung Maximum: 20% < 25% (OK, nicht gedeckelt)

Gutschrift: 2.000 € × 20% = **400 €**

</details>

### Aufgabe 4: SLA-Parameter zuordnen

Ordnen Sie folgende Aussagen dem richtigen SLA-Parameter zu:

a) "Der Server muss zu 99,9% der Zeit erreichbar sein"
b) "Bei kritischen Störungen meldet sich der Support innerhalb von 15 Minuten"
c) "Kritische Störungen werden innerhalb von 4 Stunden behoben"
d) "Der Support ist von Montag bis Freitag, 7-20 Uhr erreichbar"

<details>
<summary>Lösung anzeigen</summary>

a) **Verfügbarkeit** - Betriebszeit des Systems
b) **Reaktionszeit** - Zeit bis zur ersten Rückmeldung
c) **Lösungszeit** - Zeit bis zur Problembehebung
d) **Servicezeit** - Zeitraum der Support-Erreichbarkeit

</details>
