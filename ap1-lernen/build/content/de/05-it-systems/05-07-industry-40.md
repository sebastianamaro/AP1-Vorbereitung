## Kerninhalt

### Was ist Industrie 4.0?

**Industrie 4.0** bezeichnet die vierte industrielle Revolution: die Vernetzung und Digitalisierung der industriellen Produktion durch moderne Informationstechnologie.

```
┌─────────────────────────────────────────────────────────────────┐
│              DIE VIER INDUSTRIELLEN REVOLUTIONEN                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1.0              2.0              3.0              4.0        │
│  ~1800            ~1900            ~1970            ~2010      │
│                                                                 │
│  ┌────────┐      ┌────────┐      ┌────────┐      ┌────────┐  │
│  │Dampf-  │      │Fließ-  │      │Computer│      │Vernetz-│  │
│  │maschine│      │band    │      │& SPS   │      │ung &   │  │
│  │        │      │        │      │        │      │KI      │  │
│  └────────┘      └────────┘      └────────┘      └────────┘  │
│                                                                 │
│  Mechanisie-     Massen-         Auto-           Cyber-       │
│  rung            produktion      matisierung     Physical     │
│                                                   Systems      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

| Revolution | Zeitraum | Kennzeichen |
|------------|----------|-------------|
| **Industrie 1.0** | ~1800 | Dampfmaschine, Mechanisierung |
| **Industrie 2.0** | ~1900 | Fließband, Massenproduktion, Elektrizität |
| **Industrie 3.0** | ~1970 | Computer, SPS, Automatisierung |
| **Industrie 4.0** | ~2010 | Vernetzung, IoT, KI, Cyber-Physical Systems |

### Internet of Things (IoT)

| Aspekt | Beschreibung |
|--------|--------------|
| **Definition** | Vernetzung physischer Geräte über das Internet |
| **Sensoren** | Erfassen Daten (Temperatur, Druck, Position) |
| **Aktoren** | Führen Aktionen aus (Motor, Ventil) |
| **Kommunikation** | WLAN, Bluetooth, LoRaWAN, 5G |
| **IIoT** | Industrial Internet of Things (Industrie) |

### Cyber-Physical Systems (CPS)

**Cyber-Physical Systems** verbinden die physische Welt mit der digitalen Welt.

```
┌────────────────────────────────────────────────────────────────┐
│               CYBER-PHYSICAL SYSTEM                            │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│         CYBER (Digital)                                        │
│         ┌──────────────────────────────┐                      │
│         │  Software, Algorithmen, KI   │                      │
│         │  Datenanalyse, Steuerung     │                      │
│         └──────────────┬───────────────┘                      │
│                        │                                       │
│         ◄──────────────┼──────────────►                       │
│         Sensoren       │        Aktoren                       │
│         (Daten)        │        (Aktionen)                    │
│                        │                                       │
│         ┌──────────────┴───────────────┐                      │
│         │  Maschinen, Produkte, Umwelt │                      │
│         └──────────────────────────────┘                      │
│         PHYSICAL (Physisch)                                   │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Digital Twin (Digitaler Zwilling)

Ein **Digital Twin** ist ein virtuelles Abbild eines physischen Objekts oder Prozesses.

| Anwendung | Beschreibung |
|-----------|--------------|
| **Simulation** | Testen ohne physisches Objekt |
| **Überwachung** | Echtzeitstatus des realen Objekts |
| **Optimierung** | Verbesserungen erst virtuell testen |
| **Vorhersage** | Ausfälle vorhersagen |

## Übungsaufgaben

### Aufgabe 1
Nennen Sie die vier industriellen Revolutionen und ihre Kennzeichen.

**Lösung:**
1. **Industrie 1.0 (~1800):** Dampfmaschine, mechanische Produktion
2. **Industrie 2.0 (~1900):** Fließband, Elektrizität, Massenproduktion
3. **Industrie 3.0 (~1970):** Computer, SPS, Automatisierung
4. **Industrie 4.0 (~2010):** Vernetzung, IoT, KI, Cyber-Physical Systems

### Aufgabe 2
Was ist ein Digital Twin und wofür wird er eingesetzt?

**Lösung:** Ein **Digital Twin** ist ein virtuelles Abbild eines realen Objekts (Maschine, Produkt, Prozess). Einsatz:
- **Simulation:** Testen neuer Einstellungen ohne Risiko
- **Überwachung:** Echtzeit-Status der realen Maschine
- **Optimierung:** Verbesserungen erst virtuell testen
- **Vorhersage:** Wartungsbedarf oder Ausfälle vorhersagen

### Aufgabe 3
Nennen Sie zwei Chancen und zwei Herausforderungen von Industrie 4.0.

**Lösung:**

**Chancen:**
1. **Effizienzsteigerung:** Optimierte Prozesse durch Echtzeitdaten
2. **Predictive Maintenance:** Wartung vor dem Ausfall, weniger Stillstand

**Herausforderungen:**
1. **IT-Sicherheit:** Vernetzte Produktionsanlagen sind Angriffsziele
2. **Investitionen:** Hohe Anfangskosten für neue Technologien
