# IPv4-Adressierung

## Lernziele

Nach diesem Kapitel solltest du:
- Den Aufbau einer IPv4-Adresse verstehen
- Netzwerk- und Hostanteil unterscheiden können
- Die IP-Klassen kennen
- Private und öffentliche Adressen unterscheiden können
- Subnetzmasken verstehen

---

## Kerninhalt

### Aufbau einer IPv4-Adresse

```
┌────────────────────────────────────────────────────────────────────┐
│                    IPv4-ADRESSE                                    │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   IPv4 = Internet Protocol Version 4                               │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │ Eine IPv4-Adresse ist eine 32-Bit-Zahl, die ein Gerät       │ │
│   │ im Netzwerk eindeutig identifiziert.                        │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   Darstellung:                                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │                                                              │ │
│   │   Dezimal (Dotted Decimal Notation):                        │ │
│   │                                                              │ │
│   │         192  .  168  .  1   .  100                          │ │
│   │          │       │      │      │                            │ │
│   │          └───────┴──────┴──────┘                            │ │
│   │              4 Oktette (je 8 Bit)                           │ │
│   │                                                              │ │
│   │   Binär (32 Bit):                                           │ │
│   │                                                              │ │
│   │   11000000.10101000.00000001.01100100                       │ │
│   │      192      168       1       100                         │ │
│   │                                                              │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   Wertebereich pro Oktett: 0 - 255 (8 Bit = 2^8 = 256 Werte)      │
│                                                                     │
│   Gesamt mögliche Adressen: 2^32 = ca. 4,3 Milliarden             │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

### Netzwerk- und Hostanteil

```
┌────────────────────────────────────────────────────────────────────┐
│                NETZWERK- UND HOSTANTEIL                            │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   Jede IP-Adresse besteht aus zwei Teilen:                        │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │                                                              │ │
│   │   IP-Adresse:    192 . 168 . 1   . 100                      │ │
│   │                  ─────────────   ─────                      │ │
│   │                  NETZWERKANTEIL  HOSTANTEIL                 │ │
│   │                                                              │ │
│   │   Netzwerkanteil: Identifiziert das Netzwerk                │ │
│   │                   (wie Straßenname)                         │ │
│   │                                                              │ │
│   │   Hostanteil:     Identifiziert das Gerät im Netzwerk       │ │
│   │                   (wie Hausnummer)                          │ │
│   │                                                              │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   Die SUBNETZMASKE bestimmt die Aufteilung:                       │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │                                                              │ │
│   │   IP-Adresse:     192.168.1.100                             │ │
│   │   Subnetzmaske:   255.255.255.0                             │ │
│   │                                                              │ │
│   │   Binär:                                                     │ │
│   │   IP:     11000000.10101000.00000001.01100100               │ │
│   │   Maske:  11111111.11111111.11111111.00000000               │ │
│   │           ──────────────────────────  ────────              │ │
│   │           NETZWERK (24 Bit)           HOST (8 Bit)          │ │
│   │                                                              │ │
│   │   1-Bits = Netzwerkanteil                                   │ │
│   │   0-Bits = Hostanteil                                       │ │
│   │                                                              │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

```
│   ───────┼───────────────────┼────────────────┼───────────────────│

│   ───────┼───────────────────┼────────────────┼───────────────────│
│   B      │ 128.0.0.0 -       │ 255.255.0.0    │ 16.384 Netze,     │
│          │ 191.255.255.255   │ /16            │ 65.534 Hosts      │
│   ───────┼───────────────────┼────────────────┼───────────────────│
│   C      │ 192.0.0.0 -       │ 255.255.255.0  │ 2 Mio. Netze,     │
│          │ 223.255.255.255   │ /24            │ 254 Hosts         │
│   ───────┼───────────────────┼────────────────┼───────────────────│
│   D      │ 224.0.0.0 -       │ -              │ Multicast         │
│          │ 239.255.255.255   │                │                   │
│   ───────┼───────────────────┼────────────────┼───────────────────│
│   E      │ 240.0.0.0 -       │ -              │ Reserviert        │
│          │ 255.255.255.255   │                │                   │
│                                                                     │
│   Erkennung der Klasse am ersten Oktett:                          │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │ Klasse A: 1 - 126    (beginnt mit 0xxx)                     │ │
│   │ Klasse B: 128 - 191  (beginnt mit 10xx)                     │ │
│   │ Klasse C: 192 - 223  (beginnt mit 110x)                     │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

### Private und öffentliche Adressen

```
┌────────────────────────────────────────────────────────────────────┐
│              PRIVATE vs. ÖFFENTLICHE ADRESSEN                      │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ÖFFENTLICHE IP-ADRESSEN                                         │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • Weltweit eindeutig                                       │  │
│   │ • Im Internet geroutet                                     │  │
│   │ • Von ISP (Internet Service Provider) vergeben             │  │
│   │ • Begrenzte Anzahl (ca. 4,3 Milliarden)                   │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   PRIVATE IP-ADRESSEN                                             │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • Nur innerhalb lokaler Netzwerke gültig                   │  │
│   │ • Werden NICHT im Internet geroutet                        │  │
│   │ • Können in jedem Netzwerk wiederverwendet werden         │  │
│   │ • Verbindung zum Internet über NAT (Network Address        │  │
│   │   Translation)                                             │  │
│   │                                                             │  │
│   │   PRIVATE ADRESSBEREICHE (RFC 1918):                       │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │ Klasse A:  10.0.0.0    - 10.255.255.255   /8       │  │  │
│   │   │ Klasse B:  172.16.0.0  - 172.31.255.255   /12      │  │  │
│   │   │ Klasse C:  192.168.0.0 - 192.168.255.255  /16      │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   │   ⚠ Diese Bereiche AUSWENDIG LERNEN!                      │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   Beispiel:                                                        │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   Heimnetzwerk            Internet           Webserver     │  │
│   │   192.168.1.x      NAT                      203.0.113.50  │  │
│   │                ┌──────────┐                                │  │
│   │   ┌─────┐     │ Router   │     ┌──────────────────────┐  │  │
│   │   │ PC  │────→│          │────→│     Internet         │  │  │
│   │   │.100 │     │ Öffentl. │     │                      │  │  │
│   │   └─────┘     │   IP     │     └──────────────────────┘  │  │
│   │               └──────────┘                                │  │
│   │   Private IP → wird zu öffentlicher IP übersetzt         │  │
│   │                                                             │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

### Spezielle Adressen

```
┌────────────────────────────────────────────────────────────────────┐
│                   SPEZIELLE IP-ADRESSEN                            │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   127.0.0.1 (Localhost/Loopback)                                  │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • Verweist auf den eigenen Computer                        │  │
│   │ • Zum Testen lokaler Dienste                               │  │
│   │ • Datenpakete verlassen nie das Gerät                      │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   0.0.0.0                                                         │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • "Unbekannte Adresse" oder "alle Schnittstellen"          │  │
│   │ • Wird bei DHCP-Anfrage verwendet                          │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   255.255.255.255 (Broadcast)                                     │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • Sendet an ALLE Geräte im lokalen Netzwerk               │  │
│   │ • Wird nicht über Router hinaus weitergeleitet            │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   Netzwerkadresse (alle Host-Bits = 0)                            │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • z.B. 192.168.1.0 bei /24                                │  │
│   │ • Identifiziert das Netzwerk selbst                        │  │
│   │ • NICHT an Geräte vergebar                                 │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   Broadcast-Adresse (alle Host-Bits = 1)                          │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ • z.B. 192.168.1.255 bei /24                              │  │
│   │ • Sendet an alle Geräte im Subnetz                         │  │
│   │ • NICHT an Geräte vergebar                                 │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

### CIDR-Notation

```
┌────────────────────────────────────────────────────────────────────┐
│                      CIDR-NOTATION                                 │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   CIDR = Classless Inter-Domain Routing                           │
│                                                                     │
│   Schreibweise: IP-Adresse / Präfixlänge                          │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │                                                              │ │
│   │   192.168.1.0/24                                            │ │
│   │              ↑                                               │ │
│   │              └── 24 Bits für Netzwerk, 8 Bits für Host      │ │
│   │                                                              │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   Umrechnung in Subnetzmaske:                                     │
│                                                                     │
│   Präfix │ Subnetzmaske      │ Verfügbare Hosts                   │
│   ───────┼───────────────────┼────────────────────────────────────│
│   /8     │ 255.0.0.0         │ 16.777.214                         │
│   /16    │ 255.255.0.0       │ 65.534                             │
│   /24    │ 255.255.255.0     │ 254                                │
│   /25    │ 255.255.255.128   │ 126                                │
│   /26    │ 255.255.255.192   │ 62                                 │
│   /27    │ 255.255.255.224   │ 30                                 │
│   /28    │ 255.255.255.240   │ 14                                 │
│   /29    │ 255.255.255.248   │ 6                                  │
│   /30    │ 255.255.255.252   │ 2                                  │
│   /32    │ 255.255.255.255   │ 1 (einzelner Host)                 │
│                                                                     │
│   Formel: Hosts = 2^(32-Präfix) - 2                               │
│           (minus 2 für Netzwerk- und Broadcast-Adresse)           │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **IPv4** | Internet Protocol Version 4 (32 Bit) |
| **Oktett** | 8-Bit-Abschnitt einer IP-Adresse |
| **Subnetzmaske** | Trennt Netzwerk- von Hostanteil |
| **CIDR** | Classless Inter-Domain Routing |
| **NAT** | Network Address Translation |
| **Loopback** | Lokale Adresse 127.0.0.1 |
| **Broadcast** | Adresse für alle im Netzwerk |

## Prüfungstipps

### Häufige Prüfungsfragen

- Nennen Sie die privaten IP-Adressbereiche
- Was bedeutet /24?
- Unterschied öffentliche/private IP?

### Wichtig zu merken

- **32 Bit** = 4 Oktette à 8 Bit
- **Private Bereiche:** 10.x.x.x, 172.16-31.x.x, 192.168.x.x
- **127.0.0.1** = Localhost
- **/24** = 255.255.255.0 = 254 Hosts
- Netzwerk- und Broadcast-Adresse NICHT nutzbar
- **Hosts = 2^(Hostbits) - 2**

## Übungsaufgaben

### Aufgabe 1
Ist die IP-Adresse 192.168.5.100 eine private oder öffentliche Adresse?

**Lösung:**
**Private Adresse**

Begründung: 192.168.x.x gehört zum privaten Adressbereich der Klasse C (192.168.0.0 - 192.168.255.255). Diese Adressen werden nicht im Internet geroutet.

### Aufgabe 2
Ein Netzwerk hat die Adresse 192.168.10.0/24. Wie viele Hosts können adressiert werden?

**Lösung:**
**/24** bedeutet 24 Bits für das Netzwerk, 8 Bits für Hosts.

Berechnung: 2^8 - 2 = 256 - 2 = **254 Hosts**

Die 2 werden abgezogen für:
- Netzwerkadresse (192.168.10.0)
- Broadcast-Adresse (192.168.10.255)

### Aufgabe 3
Wandeln Sie die Subnetzmaske 255.255.255.192 in CIDR-Notation um.

**Lösung:**
255.255.255.192 in Binär:
- 255 = 11111111
- 192 = 11000000

Binär: 11111111.11111111.11111111.11000000

Anzahl der 1-Bits: 24 + 2 = **26**

Antwort: **/26**
