# Canales de Distribucion

## Objetivos de Aprendizaje

Despues de esta seccion, podras:
- Distinguir varios canales de distribucion
- Explicar ventajas y desventajas de los metodos de venta
- Nombrar modelos de comercio electronico
- Identificar canales de distribucion adecuados para productos de TI

### Comparacion: Directa vs. Indirecta

| Aspecto | Ventas Directas | Ventas Indirectas |
|---------|-----------------|-------------------|
| **Intermediarios** | Ninguno | Mayorista/Minorista |
| **Margen** | Mayor | Menor (margen del distribuidor) |
| **Contacto con cliente** | Directo | A traves del distribuidor |
| **Alcance** | Limitado | Grande |
| **Inversion** | Alta (fuerza de ventas propia) | Menor |
| **Control** | Alto | Limitado |

## Terminos Clave

| Termino | Explicacion |
|---------|-------------|
| **Ventas directas** | Ventas sin intermediarios |
| **Ventas indirectas** | Ventas a traves de distribuidores |
| **B2B** | Empresa a Empresa (empresas a empresas) |
| **B2C** | Empresa a Consumidor (empresas a clientes finales) |
| **C2C** | Consumidor a Consumidor (particular a particular) |
| **Comercio electronico** | Comercio electronico via internet |
| **Omnicanal** | Integracion de multiples canales de venta |
| **Reseller** | Revendedor autorizado |
| **SaaS** | Software como Servicio (modelo de suscripcion) |

---

## Consejos para el Examen

1. **Recordar abreviaturas:**
   - B2B = **B**usiness to **B**usiness
   - B2C = **B**usiness to **C**onsumer
   - C2C = **C**onsumer to **C**onsumer

2. **Preguntas tipicas de examen:**
   - Que canal de distribucion para servicios de TI?
   - Ventajas/desventajas de ventas directas
   - Asignar modelos de comercio electronico

3. **Importante para TI:**
   - Integrador de sistemas = Ventas indirectas con valor anadido (consultoria)
   - SaaS = Distribucion de software moderna

## Ejercicios Practicos

### Ejercicio 1: Modelos de Comercio Electronico

Asigna los siguientes ejemplos al modelo de comercio electronico correcto:

a) Un fabricante de software vende licencias a empresas
b) Un vendedor particular ofrece su PC usado en eBay
c) Una tienda online vende electronica a consumidores finales
d) Una agencia gubernamental adquiere nuevos ordenadores mediante licitacion

<details>
<summary>Mostrar Solucion</summary>

a) **B2B** (Empresa a Empresa) - Empresa vende a empresas
b) **C2C** (Consumidor a Consumidor) - Particular a particular
c) **B2C** (Empresa a Consumidor) - Empresa a consumidores finales
d) **B2G** (Empresa a Gobierno) - Empresa al sector publico

</details>

### Ejercicio 2: Decision de Canal de Distribucion

ConSystem GmbH es un integrador de sistemas TI de tamano medio y quiere optimizar sus ventas. Actualmente, los clientes son atendidos exclusivamente a traves de ventas de campo.

a) Nombra dos ventajas del enfoque actual de ventas directas.
b) Que canales de venta adicionales podrian ser utiles?
c) Que significa "omnicanal" en este contexto?

<details>
<summary>Mostrar Solucion</summary>

a) **Ventajas de las ventas directas:**
- Contacto directo con el cliente y consultoria personal
- Mayor margen (sin comision de distribuidor)
- Control sobre la experiencia del cliente
- Retroalimentacion directa de los clientes

b) **Canales de venta adicionales:**
- Tienda online para productos estandar (hardware, licencias)
- Programa de socios con otros proveedores de servicios
- Webinars y consultoria online
- Redes sociales para generacion de leads
- Cooperacion con mayoristas

c) **Omnicanal:**
Integracion de todos los canales de venta en una experiencia de cliente unificada. El cliente puede:
- Investigar online
- Recibir consultoria por telefono
- Cerrar el contrato a traves de ventas de campo
- Obtener soporte a traves de todos los canales
- Toda la informacion esta conectada

</details>

### Ejercicio 3: Modelos de Distribucion de Software

Explica la diferencia entre los siguientes modelos de distribucion de software:

a) Compra de licencia (clasica)
b) SaaS (Software como Servicio)
c) Freemium

<details>
<summary>Mostrar Solucion</summary>

a) **Compra de licencia (clasica):**
- Pago unico para uso permanente
- El software se instala en sistemas propios
- Las actualizaciones a menudo cuestan extra
- Ejemplo: Microsoft Office 2021 (version de compra)

b) **SaaS (Software como Servicio):**
- Pago regular (mensual/anual)
- El software se ejecuta en la nube
- Actualizaciones automaticamente incluidas
- Ejemplo: Microsoft 365, Salesforce

c) **Freemium:**
- Version basica gratuita
- Funciones premium de pago
- Objetivo: Convertir usuarios a version de pago
- Ejemplo: Dropbox (2 GB gratis), Spotify

</details>
