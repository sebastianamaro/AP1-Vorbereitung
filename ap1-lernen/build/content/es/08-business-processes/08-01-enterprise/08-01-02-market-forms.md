# Formas de Mercado

## Objetivos de Aprendizaje

Despues de esta seccion, podras:
- Distinguir las tres formas basicas de mercado
- Explicar ventajas y desventajas de cada forma de mercado
- Identificar formas de mercado en el sector de TI
- Describir efectos en precios y competencia

---

## Contenido Principal

### Vision General de las Formas de Mercado

La forma de mercado esta determinada por el **numero de proveedores** en un mercado.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FORMAS DE MERCADO                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Numero de proveedores:  UNO           POCOS          MUCHOS        │
│                           │             │               │            │
│                           ▼             ▼               ▼            │
│               ┌───────────┐   ┌───────────┐   ┌───────────┐         │
│               │ MONOPOLIO │   │ OLIGOPOLIO│   │ POLIPOLO  │         │
│               │           │   │           │   │           │         │
│               │ Un        │   │ Pocos     │   │ Muchos    │         │
│               │ Proveedor │   │ Proveedores│  │ Proveedores│        │
│               └───────────┘   └───────────┘   └───────────┘         │
│                                                                      │
│  Poder de mercado:   ALTO          MEDIO           BAJO             │
│                                                                      │
│  El precio es       por el        a traves de      por el           │
│  determinado:      monopolista    acuerdos*        mercado          │
│                                                                      │
│  *Los acuerdos (carteles) estan prohibidos!                         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 1. Monopolio (Un Proveedor)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MONOPOLIO                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│             ┌────────────────────┐                                  │
│             │                    │                                  │
│             │    MONOPOLISTA     │                                  │
│             │    (1 Proveedor)   │                                  │
│             │                    │                                  │
│             └─────────┬──────────┘                                  │
│                       │                                             │
│          ┌────────────┼────────────┐                                │
│          ▼            ▼            ▼                                │
│       ┌─────┐     ┌─────┐     ┌─────┐                               │
│       │Cliente│   │Cliente│   │Cliente│  ← Sin alternativa         │
│       └─────┘     └─────┘     └─────┘                               │
│                                                                      │
│  Caracteristicas:                                                   │
│  → Sin competencia                                                  │
│  → El proveedor determina el precio                                 │
│  → Los consumidores no tienen opcion                                │
│                                                                      │
│  Ejemplos:                                                          │
│  → Anteriormente: Deutsche Post (monopolio postal, abolido)         │
│  → Operadores de red en una region                                  │
│  → Medicamentos protegidos por patente                              │
│                                                                      │
│  Ejemplo TI:                                                        │
│  → Microsoft en el mercado de sistemas operativos (cuasi-monopolio) │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

| Ventajas | Desventajas |
|----------|-------------|
| Estandares uniformes | Precios altos para consumidores |
| Sin infraestructura duplicada | Sin presion para innovar |
| Seguridad de planificacion | Dependencia del cliente |

### 2. Oligopolio (Pocos Proveedores)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OLIGOPOLIO                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│       ┌─────────┐     ┌─────────┐     ┌─────────┐                   │
│       │Proveedor│     │Proveedor│     │Proveedor│                   │
│       │    A    │     │    B    │     │    C    │                   │
│       └────┬────┘     └────┬────┘     └────┬────┘                   │
│            │               │               │                        │
│            │   se observan mutuamente      │                        │
│            └───────────────┼───────────────┘                        │
│                            │                                        │
│                ┌───────────┼───────────┐                            │
│                ▼           ▼           ▼                            │
│             ┌─────┐     ┌─────┐     ┌─────┐                         │
│             │Cliente│   │Cliente│   │Cliente│  ← Opcion limitada    │
│             └─────┘     └─────┘     └─────┘                         │
│                                                                      │
│  Caracteristicas:                                                   │
│  → Numero limitado de grandes proveedores                           │
│  → Altas barreras de entrada al mercado                             │
│  → Los proveedores se observan mutuamente                           │
│  → Riesgo de acuerdos de precios (ilegal!)                          │
│                                                                      │
│  Ejemplos:                                                          │
│  → Redes moviles (Telekom, Vodafone, O2)                            │
│  → Gasolineras (Shell, Aral, Esso, Total)                           │
│  → Mercado de nube (AWS, Azure, Google Cloud)                       │
│                                                                      │
│  Ejemplos TI:                                                       │
│  → Procesadores (Intel, AMD)                                        │
│  → Tarjetas graficas (NVIDIA, AMD)                                  │
│  → SO de smartphones (Android, iOS)                                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

| Ventajas | Desventajas |
|----------|-------------|
| Mas opcion que el monopolio | Poca competencia real |
| Alta calidad por competencia | Los precios pueden coordinarse |
| Inversiones en innovacion | Altas barreras de entrada para nuevos proveedores |

### 3. Polipolo (Muchos Proveedores)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    POLIPOLO                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐      │
│  │ A │ │ B │ │ C │ │ D │ │ E │ │ F │ │ G │ │ H │ │ I │ │ J │      │
│  └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘      │
│    │     │     │     │     │     │     │     │     │     │          │
│    └─────┴─────┴─────┴─────┼─────┴─────┴─────┴─────┴─────┘          │
│                            │                                        │
│                    ┌───────┼───────┐                                │
│                    ▼       ▼       ▼                                │
│                 ┌─────┐ ┌─────┐ ┌─────┐                             │
│                 │Cliente│ │Cliente│ │Cliente│  ← Gran seleccion     │
│                 └─────┘ └─────┘ └─────┘                             │
│                                                                      │
│  Caracteristicas:                                                   │
│  → Muchos proveedores compiten                                      │
│  → Ningun proveedor puede determinar el precio                      │
│  → El mercado regula el precio                                      │
│  → Favorable para el consumidor                                     │
│                                                                      │
│  Ejemplos:                                                          │
│  → Panaderias                                                       │
│  → Restaurantes                                                     │
│  → Peluquerias                                                      │
│                                                                      │
│  Ejemplos TI:                                                       │
│  → Desarrolladores web / agencias                                   │
│  → Integradores de sistemas TI                                      │
│  → Desarrollo de software (muchas empresas pequenas)                │
│  → Distribuidores de PCs                                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

| Ventajas | Desventajas |
|----------|-------------|
| Gran seleccion para consumidores | Orientacion dificil |
| Precios justos por competencia | Diferencias de calidad |
| Baja barrera de entrada al mercado | Proveedores pequenos a menudo inestables |

### Tabla Comparativa

| Criterio | Monopolio | Oligopolio | Polipolo |
|----------|-----------|------------|----------|
| **Numero de proveedores** | Uno | Pocos (2-10) | Muchos |

| **Formacion de precios** | El proveedor determina | Observacion mutua | Mercado |
| **Competencia** | Ninguna | Limitada | Intensa |

### Derecho de la Competencia

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DERECHO DE LA COMPETENCIA                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ESTA PROHIBIDO:                                                    │
│  ────────────────────────────────────────────────────────────────   │
│  → Acuerdos de precios entre competidores                           │

```

## Terminos Clave

| Termino | Explicacion |
|---------|-------------|
| **Monopolio** | Mercado con un solo proveedor |
| **Oligopolio** | Mercado con pocos grandes proveedores |
| **Polipolo** | Mercado con muchos proveedores |
| **Barrera de entrada al mercado** | Obstaculos para nuevos proveedores |
| **Cartel** | Acuerdo ilegal entre competidores |
| **Poder de mercado** | Capacidad de influir en precios y condiciones |

---

## Consejos para el Examen

1. **Mnemotecnico para formas de mercado:**
   - **Mono** = Uno (un proveedor)
   - **Oligo** = Pocos (griego)
   - **Poli** = Muchos (griego)

2. **Preguntas tipicas de examen:**
   - Asignar ejemplos de mercados de TI
   - Ventajas/desventajas de formas de mercado
   - Por que estan prohibidos los carteles?

3. **Frecuentemente preguntado:**
   - "Oligopolio en procesadores" (Intel, AMD)
   - "Oligopolio en la nube" (AWS, Azure, Google)

## Ejercicios Practicos

### Ejercicio 1: Clasificacion

Asigna los siguientes mercados a la forma de mercado correcta:

a) Sistemas operativos para smartphones
b) Integradores de sistemas TI locales
c) Proveedor de red de fibra optica en una region rural
d) Fabricantes de CPU para PCs de escritorio

<details>
<summary>Mostrar Solucion</summary>

a) **Oligopolio** - Android e iOS dominan (dos proveedores)
b) **Polipolo** - Muchos proveedores pequenos y medianos
c) **Monopolio** - A menudo solo un proveedor en regiones rurales
d) **Oligopolio** - Intel y AMD (duopolio)

</details>

### Ejercicio 2: Formacion de Precios

ConSystem GmbH quiere comprar un nuevo portatil. Explica como la forma de mercado afecta la negociacion de precios:

a) Si hubiera solo un fabricante (monopolio)
b) En la situacion actual (oligopolio: Dell, HP, Lenovo)
c) Con muchos pequenos proveedores locales (polipolo)

<details>
<summary>Mostrar Solucion</summary>

a) **Monopolio:**
- Sin poder de negociacion para ConSystem
- El precio es dictado por el proveedor
- Sin alternativa, "tomalo o dejalo"

b) **Oligopolio (actual):**
- Comparacion entre Dell, HP, Lenovo posible
- Obtener y negociar ofertas
- Los proveedores se observan mutuamente (precios similares)
- Posicion de negociacion media

c) **Polipolo:**
- Gran seleccion, competencia intensa
- Fuerte posicion de negociacion para ConSystem
- Los proveedores se rebajan mutuamente
- Precios mas bajos

</details>

### Ejercicio 3: Derecho de la Competencia

Dos grandes proveedores de nube se reunen y acuerdan aumentar sus precios un 20%.

a) Como se llama este acuerdo?
b) Es este acuerdo legal o ilegal?
c) Que autoridad seria responsable en Alemania?

<details>
<summary>Mostrar Solucion</summary>

a) **Cartel de precios** (o acuerdo de precios)

b) **Ilegal** - Los acuerdos de precios entre competidores estan prohibidos bajo la Ley contra Restricciones de la Competencia (GWB).

c) **Oficina Federal de Carteles** (para importancia nacional) o **Comision Europea** (para impacto en toda la UE)

Posibles consecuencias:
- Altas multas (hasta el 10% de los ingresos anuales)
- Demandas civiles por danos
- Dano a la reputacion

</details>
