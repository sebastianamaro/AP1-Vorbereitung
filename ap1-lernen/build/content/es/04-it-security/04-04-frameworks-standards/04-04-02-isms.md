# Sistema de Gestion de Seguridad de la Informacion (SGSI)

## Objetivos de Aprendizaje

Despues de este capitulo, deberas ser capaz de:
- Explicar el termino SGSI
- Conocer los componentes de un SGSI
- Comprender la conexion con la Proteccion de Linea Base BSI
- Conocer la mejora continua (PDCA)

**Nota:** Los detalles de ISO 2700x YA NO estan en el catalogo de examenes 2025, por lo tanto aqui solo se cubre SGSI en general y la Proteccion de Linea Base BSI.

---

## Contenido Principal

### Que es un SGSI?

Un **SGSI (Sistema de Gestion de Seguridad de la Informacion)** es un enfoque sistematico para gestionar y mejorar continuamente la seguridad de la informacion en una organizacion.

```
+------------------------------------------------------------------+
|                            SGSI                                   |
|              Sistema de Gestion de Seguridad de la Informacion    |
+------------------------------------------------------------------+
|                                                                   |
|  +--------------------------------------------------------------+ |
|  |                      OBJETIVO                                 | |
|  |  Proteccion sistematica de la informacion mediante            | |
|  |  * Politicas                                                  | |
|  |  * Procesos                                                   | |
|  |  * Medidas tecnicas                                           | |
|  |  * Mejora continua                                            | |
|  +--------------------------------------------------------------+ |
|                                                                   |
|                         IMPLEMENTACION                            |
|                            |                                      |
|            +---------------+---------------+                      |
|            v               v               v                      |
|       +-----------+   +-----------+   +-----------+              |
|       |   BSI     |   |   ISO     |   |  Otros    |              |
|       | Proteccion|   |  27001    |   | Marcos    |              |
|       | Linea Base|   |(no en     |   |           |              |
|       |           |   | AP1)      |   |           |              |
|       +-----------+   +-----------+   +-----------+              |
|                                                                   |
+------------------------------------------------------------------+
```

---

### Componentes de un SGSI

| Componente | Descripcion |
|------------|-------------|
| **Politica de Seguridad** | Orientacion basica y objetivos |
| **Organizacion** | Roles y responsabilidades |
| **Gestion de Riesgos** | Identificacion y tratamiento de riesgos |
| **Medidas** | Controles tecnicos y organizativos |
| **Documentacion** | Politicas, procesos, registros |
| **Formacion** | Concienciacion y competencia |
| **Monitorizacion** | Control y medicion |
| **Mejora** | Proceso de mejora continua |

---

### El Ciclo PDCA en el SGSI

El SGSI sigue el **ciclo PDCA** para la mejora continua:

```
+------------------------------------------------------------------+
|                    CICLO PDCA EN EL SGSI                          |
+------------------------------------------------------------------+
|                                                                   |
|                      +-----------+                                |
|                      |   PLAN    |                                |
|                      |           |                                |
|                      |  Planificar|                               |
|                      |  SGSI     |                                |
|                      +-----+-----+                                |
|                            |                                      |
|          +-----------------+-----------------+                    |
|          |                 |                 |                    |
|          v                 v                 v                    |
|     +-----------+                       +-----------+             |
|     |    ACT    |                       |    DO     |             |
|     |           |                       |           |             |
|     |  Mejorar  |<----------------------| Implementar|            |
|     |  SGSI     |                       | SGSI      |             |
|     +-----+-----+                       +-----+-----+             |
|           |                                   |                   |
|           |         +-----------+             |                   |
|           |         |   CHECK   |             |                   |
|           |         |           |             |                   |
|           +-------->|  Revisar  |<------------+                   |
|                     |  SGSI     |                                 |
|                     +-----------+                                 |
|                                                                   |
+------------------------------------------------------------------+
```

| Fase | Descripcion | Actividades |
|------|-------------|-------------|
| **Plan** | Planificar SGSI | Definir objetivos, analizar riesgos, planificar medidas |
| **Do** | Implementar SGSI | Implementar medidas, realizar formacion |
| **Check** | Revisar SGSI | Monitorizar, realizar auditorias, medir efectividad |
| **Act** | Mejorar SGSI | Acciones correctivas, mejora continua |

---

### Roles en el SGSI

| Rol | Tareas |
|-----|--------|
| **Direccion** | Responsabilidad general, proporcionar recursos |
| **Responsable de Seguridad TI (RSI)** | Coordinacion del SGSI, persona de contacto |
| **Delegado de Proteccion de Datos (DPD)** | Cumplimiento de proteccion de datos |
| **Departamento TI** | Implementacion tecnica |
| **Departamentos** | Requisitos profesionales, responsabilidad de datos |
| **Empleados** | Cumplimiento de politicas |

---

### Politica de Seguridad

La **Politica de Seguridad** es el documento central del SGSI:

| Contenido | Descripcion |
|-----------|-------------|
| **Alcance** | Que areas/sistemas estan afectados? |
| **Objetivos** | Que debe lograrse? |
| **Principios** | Que principios aplican? |
| **Responsabilidades** | Quien es responsable de que? |
| **Consecuencias** | Que ocurre en caso de violaciones? |

---

```
+------------------------------------------------------------------+
|              DOCUMENTACION EN EL SGSI                             |
+------------------------------------------------------------------+
|                                                                   |
|  Nivel 1: POLITICA                                                |

|                           |                                       |

|  +--------------------------------------------------------------+ |
|                           |                                       |
|  Nivel 4: REGISTROS       v                                       |
|  +--------------------------------------------------------------+ |
|  |  Evidencia (Que se hizo?)                                     | |

|  +--------------------------------------------------------------+ |
|                                                                   |
+------------------------------------------------------------------+
```

## Ejercicios Practicos

### Ejercicio 1
Que es un SGSI?

**Solucion:** Un **SGSI (Sistema de Gestion de Seguridad de la Informacion)** es un enfoque sistematico para gestionar y mejorar continuamente la seguridad de la informacion en una organizacion. Abarca politicas, procesos, medidas tecnicas y estructuras organizativas para proteger la informacion.

### Ejercicio 2
Explica el ciclo PDCA en el contexto del SGSI.

**Solucion:** El ciclo PDCA es un modelo para la mejora continua:
- **Plan:** Planificar SGSI - Definir objetivos, analizar riesgos, planificar medidas
- **Do:** Implementar SGSI - Implementar medidas, formar empleados
- **Check:** Revisar SGSI - Monitorizar, realizar auditorias, medir efectividad
- **Act:** Mejorar SGSI - Implementar acciones correctivas, optimizar procesos

El ciclo se repite continuamente.

### Ejercicio 3
Que debe contener una politica de seguridad?

**Solucion:** Una politica de seguridad debe contener:
1. **Alcance** - Que areas estan afectadas?
2. **Objetivos** - Que debe lograrse?
3. **Principios** - Que principios de seguridad aplican?
4. **Responsabilidades** - Quien es responsable de que?
5. **Consecuencias** - Que ocurre en caso de violaciones?

### Ejercicio 4
Nombra tres roles en el SGSI.

**Solucion:**
1. **Direccion** - Responsabilidad general, proporcionar recursos
2. **Responsable de Seguridad TI (RSI)** - Coordinacion del SGSI
3. **Delegado de Proteccion de Datos (DPD)** - Cumplimiento de proteccion de datos

(Otros: Departamento TI, Departamentos, Empleados)
