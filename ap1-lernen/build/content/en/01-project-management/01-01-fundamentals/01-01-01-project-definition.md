# Project Definition

## Learning Objectives

After this chapter, you should be able to:
- Distinguish a project from a routine task
- Know the five characteristics of a project
- Give examples of projects and non-projects

---

## Core Content

### What is a Project?

A **project** is an undertaking with the following characteristic features:

### The 5 Characteristics of a Project

| Characteristic | Explanation |
|----------------|-------------|
| **Uniqueness** | The undertaking is new and not routinely repeated |
| **Goal-oriented** | There is a clear, defined goal |
| **Time-limited** | There is a defined start and end date |
| **Limited Resources** | Budget, personnel, and materials are limited |
| **Complexity** | The undertaking is not trivial and requires coordination |

### Project vs. Routine Task

| Project | Routine Task |
|---------|--------------|
| Unique | Recurring |
| Defined end date | Ongoing |
| Novel | Standardized |
| Open-ended result | Predictable result |
| Own organization | Integrated in line organization |

### Examples

**Projects:**
- Introduction of a new ERP system
- Setting up a new company location
- Development of a new app
- Migration to a new server infrastructure

**Not Projects:**
- Daily data backup
- Monthly payroll processing
- Regular system maintenance
- Answering support requests

---

## Key Terms

| Term | Explanation |
|------|-------------|
| **Project** | Unique, time-limited undertaking with a defined goal |
| **Project Management** | Planning, steering, and controlling of projects |
| **Stakeholder** | All persons/groups affected by or influencing the project |
| **Project Manager** | Person responsible for project execution |

---

## Exam Tips

**Typical Exam Question:**
> "Check the following text for incorrect statements:
> A project is a recurring undertaking with a clear goal as well as a start date and variable end date."

**Solution:** Two errors:
1. "recurring" - wrong, projects are unique
2. "variable end date" - wrong, projects have a defined end date

## Practice Exercises

### Exercise 1
Decide whether it is a project:
a) Introduction of new accounting software
b) Daily server backup
c) Conversion of company website to a new CMS
d) Answering emails

**Solution:**
a) Yes, Project - unique, time-limited, goal-oriented
b) No, Routine task - recurring
c) Yes, Project - unique, time-limited
d) No, Routine task - recurring

### Exercise 2
Name three characteristics that distinguish a project from a routine task.

**Solution:**
1. Uniqueness
2. Time limitation (defined end date)
3. Novelty/Complexity
