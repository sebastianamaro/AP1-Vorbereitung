# Copyright in IT

## Learning Objectives

After this chapter, you should be able to:
- Understand copyright in the IT context
- Know the rights of software developers
- Distinguish usage rights from copyrights
- Know consequences of copyright infringement

---

## Core Content

### What is Copyright?

```
┌────────────────────────────────────────────────────────────────────┐
│                       COPYRIGHT                                    │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   Definition:                                                      │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │ Copyright protects intellectual creations (works)           │ │
│   │ and grants the creator exclusive rights.                    │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   In Germany regulated by:                                         │
│   • Copyright Act (UrhG - Urheberrechtsgesetz)                    │
│   • Specifically for software: §§ 69a-69g UrhG                    │
│                                                                     │
│   Basic principle:                                                 │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │              Author = Creator of the work                   │ │
│   │                                                              │ │
│   │         ┌───────────────┐                                   │ │
│   │         │    AUTHOR     │                                   │ │
│   │         │   (natural    │                                   │ │
│   │         │    person)    │                                   │ │
│   │         └───────┬───────┘                                   │ │
│   │                 │                                           │ │
│   │                 ↓ creates                                   │ │
│   │         ┌───────────────┐                                   │ │
│   │         │     WORK      │                                   │ │
│   │         │  (Software,   │                                   │ │
│   │         │   Text, ...)  │                                   │ │
│   │         └───────────────┘                                   │ │
│   │                                                              │ │
│   │   ⚠ Only natural persons can be authors!                   │ │
│   │     (No companies, no AI)                                   │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

```
┌────────────────────────────────────────────────────────────────────┐
│                COPYRIGHT IN EMPLOYMENT                             │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   Special case for employees (§ 69b UrhG):                         │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │                                                              │ │
│   │   ┌─────────────┐  creates  ┌─────────────┐                │ │
│   │   │  Developer  │ ─────────→ │  Software   │                │ │
│   │   │  (employee) │             │   (work)    │                │ │
│   │   └──────┬──────┘             └──────┬──────┘                │ │
│   │          │                           │                      │ │
│   │          │ Author                    │ Usage rights         │ │
│   │          │                           ↓                      │ │
│   │          │                    ┌─────────────┐              │ │
│   │          └───────────────────→│  Employer   │              │ │
│   │                                └─────────────┘              │ │
│   │                                                              │ │
│   │  • Developer ALWAYS remains author                         │ │
│   │  • Employer automatically receives all usage rights        │ │
│   │  • Applies to software created as part of work             │ │
│   │  • No additional compensation required                     │ │
│   │                                                              │ │
│   └─────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   For freelancers/contractors:                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
│   │ • Usage rights must be contractually agreed!               │ │

│                                                                     │
```

```
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐ │
```

## Key Terms

| Term | Explanation |
|------|-------------|
| **Author** | Creator of the work (natural person) |
| **Work** | Intellectual creation (e.g., software, text) |
| **Threshold of originality** | Minimum level of creativity |
| **Usage right** | Transferable right to use |
| **Economic right** | Right to commercial use |
| **Public domain** | Without copyright, freely usable |
| **Decompilation** | Reverse translation of machine code |

## Exam Tips

### Common Exam Questions

- Who is the author of software created in employment?
- What can you do with software without a license?
- How long is software protected by copyright?

### Important to Remember

- **Author** = always the natural person (never a company!)
- **In employment:** Usage rights automatically go to employer
- **Duration:** 70 years after author's death
- **Permitted without license:** Intended use, backup, error correction
- **Ideas and concepts** are NOT protected
- **Source code** is protected

## Practice Exercises

### Exercise 1
A software developer programs a new application for their employer. Who is the author? Who has the usage rights?

**Solution:**
- **Author:** The software developer (employee) ALWAYS remains the author, since only natural persons can be authors.
- **Usage rights:** The employer automatically receives all usage rights under § 69b UrhG when the software is created as part of employment duties.

### Exercise 2
May you create a backup copy of purchased software?

**Solution:**
**Yes**, a backup copy is permitted under § 69d UrhG if:
- It is necessary for backing up the program
- It is only ONE copy
- It is not distributed to third parties

### Exercise 3
How long is a computer program protected by copyright?

**Solution:**
A computer program is protected for **70 years after the author's death**. For multiple authors: 70 years after the death of the last surviving author. After that, the software becomes public domain.
