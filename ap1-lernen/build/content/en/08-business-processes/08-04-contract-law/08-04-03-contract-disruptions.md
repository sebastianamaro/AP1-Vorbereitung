# Contract Disruptions and Warranty Rights

## Learning Objectives

After this section, you will be able to:
- Distinguish types of contract disruptions
- Explain buyer's warranty rights
- Name the sequence of warranty rights
- Distinguish between warranty and guarantee

---

## Core Content

### Types of Contract Disruptions

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CONTRACT DISRUPTIONS                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. DEFECTIVE DELIVERY (Non-conforming performance)        │    │
│  │     → Goods have defects                                    │    │
│  │     → Example: Laptop doesn't start                         │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  2. WRONG DELIVERY                                          │    │
│  │     → Wrong goods delivered                                 │    │
│  │     → Example: Server instead of laptop                     │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  3. SHORT DELIVERY                                          │    │
│  │     → Too little delivered                                  │    │
│  │     → Example: 8 instead of 10 laptops                      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  4. DELIVERY DELAY                                          │    │
│  │     → Delivery late                                         │    │
│  │     → Example: Delivery 2 weeks after deadline              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  5. PAYMENT DELAY                                           │    │
│  │     → Buyer doesn't pay on time                             │    │
│  │     → Example: Invoice not paid after due date              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Types of Defects

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TYPES OF DEFECTS                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  MATERIAL DEFECT (§ 434 BGB)                                        │
│  → The item does not conform to the agreed specification            │
│                                                                      │
│  A material defect exists when:                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. Agreed specification is missing                         │    │
│  │     → E.g. "16 GB RAM" agreed, only 8 GB installed          │    │
│  │                                                              │    │
│  │  2. Normal use not possible                                  │    │
│  │     → E.g. PC doesn't start                                  │    │
│  │                                                              │    │
│  │  3. Specification differs from advertising                   │    │
│  │     → E.g. "10 hours battery" promised, only 5 hours         │    │
│  │                                                              │    │
│  │  4. Assembly faulty (when assembly agreed)                   │    │
│  │     → E.g. server incorrectly installed                      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  LEGAL DEFECT (§ 435 BGB)                                           │
│  → Third parties can assert rights to the item                      │
│  → E.g. software without valid license, stolen goods                │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Buyer's Warranty Rights (§ 437 BGB)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WARRANTY RIGHTS                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  In case of a defect, the buyer has the following rights:          │
│                                                                      │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  1. SUPPLEMENTARY PERFORMANCE (§ 439 BGB)  ← PRIORITY!       │   │
│  │     a) Repair (remediation)                                   │   │
│  │        OR                                                     │   │
│  │     b) Replacement delivery (new goods)                       │   │
│  │                                                               │   │
│  │     → Buyer has the CHOICE                                    │   │
│  │     → Seller can only refuse if disproportionate              │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│                      │ Only if supplementary performance             │
│                      │ FAILS (2 attempts)                            │
│                      │ or is REFUSED                                 │
│                      ▼                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  2. WITHDRAWAL (§ 323 BGB)                                    │   │
│  │     → Reverse the contract                                    │   │
│  │     → Goods back, money back                                  │   │
│  │     → Only for significant defects!                           │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  3. PRICE REDUCTION (§ 441 BGB)                               │   │
│  │     → Reduce purchase price                                   │   │
│  │     → Keep goods, pay less                                    │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  4. DAMAGES (§ 280 BGB)                                       │   │
│  │     → If seller is at fault                                   │   │
│  │     → Compensation for consequential damages                  │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Sequence of Warranty Rights

```
┌─────────────────────────────────────────────────────────────────────┐
│          HIERARCHY OF WARRANTY RIGHTS                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  LEVEL 1: SUPPLEMENTARY PERFORMANCE                                 │
│  ─────────────────────────────────────────────────────────────────  │
│  → Always first!                                                    │
│  → Seller has the right to remedy                                   │
│  → Buyer must set deadline (usually 2 attempts)                     │
│                                                                      │
│           │                                                         │
│           │ Supplementary performance failed?                       │
│           │ Seller refuses?                                         │
│           │ Unreasonable?                                           │
│           ▼                                                         │
│                                                                      │
│  LEVEL 2: WITHDRAWAL or PRICE REDUCTION                             │
│  ─────────────────────────────────────────────────────────────────  │
│  → Buyer can choose                                                 │
│  → Withdrawal only for significant defect                           │
│  → Price reduction also for minor defect                            │
│                                                                      │
│  ADDITIONALLY: DAMAGES                                              │
│  ─────────────────────────────────────────────────────────────────  │
│  → Can be claimed additionally                                      │
│  → Only if seller is at fault                                       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Warranty vs. Guarantee

```
┌─────────────────────────────────────────────────────────────────────┐
│           WARRANTY vs. GUARANTEE                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  WARRANTY (Statutory)              GUARANTEE (Voluntary)            │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  Legally regulated                 Voluntary from manufacturer      │
│  (§§ 434 ff. BGB)                                                   │
│                                                                      │
│  24 months from delivery           Set by manufacturer              │
│  (new goods, B2C)                  (e.g. 3 years, 5 years)          │
│                                                                      │
│  Seller is liable                  Manufacturer is liable           │
│                                    (or seller, depending on guarantee)│
│                                                                      │
│  Defect must have existed          Defect occurred during           │
│  at delivery                       guarantee period                  │
│                                                                      │
│  Burden of proof shifts            Often simpler handling           │
│  after 12 months to buyer          (goodwill)                       │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  REMEMBER: Guarantee does NOT replace warranty!                     │
│            Warranty is a right, guarantee is an extra               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Time Limits

| Limit | Duration | Note |
|-------|----------|------|
| **Warranty period** | 24 months | For new goods B2C |
| **Warranty period** | 12 months | For used goods B2C |
| **Warranty period B2B** | 12 months | For business customers (can be shortened) |
| **Burden of proof reversal** | 12 months | After that, buyer must prove |
| **Complaint period B2B** | Immediately | Merchant must complain immediately! |

### Default

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DEFAULT                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  DELIVERY DEFAULT (Seller doesn't deliver on time)                  │
│  ──────────────────────────────────────────────────────────────     │
│  Requirements:                                                      │
│  1. Due date (delivery date exceeded)                               │
│  2. Reminder (or date was calendar-specific)                        │
│  3. Seller's fault                                                  │
│                                                                      │
│  Buyer's rights:                                                    │
│  → Damages in addition to performance                               │
│  → After deadline: Damages instead of performance                   │
│  → Withdrawal from contract                                         │
│                                                                      │
│  PAYMENT DEFAULT (Buyer doesn't pay on time)                        │
│  ──────────────────────────────────────────────────────────────     │
│  Requirements:                                                      │
│  1. Due date (payment deadline exceeded)                            │
│  2. Reminder (or 30 days after invoice receipt)                     │
│                                                                      │
│  Seller's rights:                                                   │
│  → Default interest (5% above base rate, B2B: 9%)                   │
│  → Reminder costs                                                   │
│  → Damages                                                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Key Terms

| Term | Explanation |
|------|-------------|
| **Material defect** | Goods do not conform to agreement |
| **Legal defect** | Third parties have rights to the item |
| **Supplementary performance** | Repair or replacement delivery |
| **Withdrawal** | Reverse the contract |
| **Price reduction** | Reduce purchase price |
| **Warranty** | Statutory defect liability |
| **Guarantee** | Voluntary additional service |
| **Default** | Delay in performance/payment |

---

## Exam Tips

1. **Remember the sequence:**
   - **S**upplementary performance first → then **W**ithdrawal/**R**eduction/**D**amages

2. **Warranty ≠ Guarantee:**
   - Warranty = statutory
   - Guarantee = voluntary

3. **Typical exam questions:**
   - What rights does the buyer have in case of a defect?
   - Difference warranty/guarantee?
   - What is a material defect?

## Practice Exercises

### Exercise 1: Apply Warranty Rights

ConSystem bought 10 laptops from a dealer. The keyboard doesn't work on 3 laptops.

a) Is there a defect?
b) Which right should ConSystem assert first?
c) What happens if repair fails twice?

<details>
<summary>Show solution</summary>

a) **Yes, there is a material defect.**
- The laptops do not conform to normal use
- A keyboard must work

b) **Supplementary performance (repair or replacement delivery)**
- ConSystem should request the dealer to repair OR deliver new laptops
- Set deadline (e.g. 14 days)

c) **After 2 failed attempts:**
- ConSystem can withdraw from the purchase contract (for the 3 laptops)
- OR demand price reduction
- OR claim damages (e.g. cost of replacement devices)

</details>

### Exercise 2: Warranty vs. Guarantee

A customer buys a PC from ConSystem. After 14 months the power supply fails.

a) Can the customer claim warranty?
b) Who must prove that the defect existed at delivery?
c) What would be different if the manufacturer gives 3 years guarantee?

<details>
<summary>Show solution</summary>

a) **Yes, the warranty period (24 months) has not yet expired.**

b) **The CUSTOMER must prove.**
- The burden of proof reversal only applies for 12 months
- After 14 months, the customer must prove that the power supply was already defective at delivery (e.g. by expert opinion)
- This is difficult in practice

c) **With 3-year manufacturer guarantee:**
- The customer can contact the manufacturer
- Guarantee is easier to handle (often without proof)
- The manufacturer exchanges/repairs according to their guarantee terms
- Advantage: No complicated proof required

</details>

### Exercise 3: Delivery Default

ConSystem ordered servers for January 15. Today is January 25 and the servers haven't arrived. The dealer doesn't respond to inquiries.

What steps should ConSystem take?

<details>
<summary>Show solution</summary>

**Step 1: Establish default**
- Delivery date was calendar-specific (Jan 15)
- Reminder is therefore not necessary (§ 286 para. 2 BGB)
- Dealer has been in default since Jan 16

**Step 2: Set deadline**
- Written reminder with deadline
- "We request you to deliver the servers by February 1. After this deadline expires, we will withdraw from the contract."

**Step 3: After deadline expires**
- Withdraw from contract OR
- Claim damages instead of performance OR
- Substitute procurement from another dealer + damages

**Step 4: Document**
- Keep all correspondence
- Document additional costs for substitute procurement
- Calculate delay damages if applicable (project cannot start)

</details>
