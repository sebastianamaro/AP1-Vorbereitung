# Angebotsvergleich

## Lernziele

Nach diesem Abschnitt kannst du:
- Quantitative und qualitative Vergleichsmethoden anwenden
- Den Bezugspreis berechnen
- Angebote systematisch bewerten
- Die beste Beschaffungsalternative ermitteln

### Bezugspreisberechnung

Der **Bezugspreis** ist der tatsächliche Einkaufspreis, den ein Unternehmen für eine Ware zahlt - inklusive aller Abzüge und Zuschläge.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BEZUGSPREISSCHEMA                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│      Listeneinkaufspreis (netto)                                    │
│    - Lieferantenrabatt                                              │
│    ─────────────────────────────                                    │
│    = Zieleinkaufspreis                                              │
│    - Lieferantenskonto                                              │
│    ─────────────────────────────                                    │
│    = Bareinkaufspreis                                               │
│    + Bezugskosten                                                   │
│      (Verpackung, Transport, Versicherung, Zoll)                    │
│    ─────────────────────────────                                    │
│    = BEZUGSPREIS (Einstandspreis)                                   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Beispielrechnung

```
┌─────────────────────────────────────────────────────────────────────┐
│          BEZUGSPREISBERECHNUNG - BEISPIEL                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Die ConSystem GmbH bestellt 10 Laptops:                            │
│                                                                      │
│  Listeneinkaufspreis (netto):    1.000,00 € × 10 = 10.000,00 €     │
│  - Mengenrabatt 15%:                             -  1.500,00 €      │
│  ─────────────────────────────────────────────────────────────      │
│  = Zieleinkaufspreis:                              8.500,00 €       │
│  - Skonto 2% (bei Zahlung innerhalb 10 Tagen):   -   170,00 €      │
│  ─────────────────────────────────────────────────────────────      │
│  = Bareinkaufspreis:                               8.330,00 €       │
│  + Versandkosten:                                +   150,00 €       │
│  + Verpackung:                                   +    50,00 €       │
│  ─────────────────────────────────────────────────────────────      │
│  = BEZUGSPREIS:                                    8.530,00 €       │
│                                                                      │
│  Bezugspreis pro Laptop: 8.530 € ÷ 10 = 853,00 €                   │
│                                                                      │
│  Vergleich:                                                         │
│  Listenpreis:   1.000 € pro Laptop                                  │
│  Bezugspreis:     853 € pro Laptop                                  │
│  Ersparnis:       147 € pro Laptop (14,7%)                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Begriffe bei Zahlungsbedingungen

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ZAHLUNGSBEDINGUNGEN                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  RABATT                                                             │
│  → Preisnachlass beim Kauf                                          │
│  → Z.B. Mengenrabatt, Treuerabatt, Sonderrabatt                     │
│  → Wird vom Listenpreis abgezogen                                   │
│                                                                      │
│  SKONTO                                                             │
│  → Preisnachlass bei schneller Zahlung                              │
│  → Z.B. "2% Skonto bei Zahlung innerhalb 10 Tagen"                  │
│  → Wird vom Zieleinkaufspreis abgezogen                             │
│                                                                      │
│  ZAHLUNGSZIEL                                                       │
│  → Zeitraum bis zur Zahlung                                         │
│  → Z.B. "30 Tage netto" = Zahlung innerhalb 30 Tagen ohne Abzug     │
│                                                                      │
│  Typische Formulierung:                                             │
│  "2% Skonto bei Zahlung innerhalb 10 Tagen, 30 Tage netto"          │
│  → Innerhalb 10 Tagen: 2% Skonto                                    │
│  → Innerhalb 30 Tagen: Kein Skonto, aber keine Mahnung              │
│  → Nach 30 Tagen: Verzug                                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Qualitativer Vergleich

| Kriterium | Anbieter A | Anbieter B | Anbieter C |
|-----------|------------|------------|------------|
| Bezugspreis | 8.530 € | 8.443,50 € | 8.400 € |
| Lieferzeit | 3 Tage | 7 Tage | 14 Tage |
| Garantie | 3 Jahre | 2 Jahre | 5 Jahre |
| Support | Telefon + Vor-Ort | Nur Telefon | Telefon + Chat |
| Umtauschrecht | 14 Tage | 30 Tage | 14 Tage |

**Fazit:** Der günstigste Anbieter (C) hat die längste Lieferzeit. Die Entscheidung hängt von den Prioritäten ab!

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Listeneinkaufspreis** | Ursprünglicher Preis ohne Abzüge |
| **Rabatt** | Preisnachlass (z.B. Mengenrabatt) |
| **Zieleinkaufspreis** | Listenpreis nach Rabatt |
| **Skonto** | Preisnachlass bei schneller Zahlung |
| **Bareinkaufspreis** | Preis nach Skonto |
| **Bezugskosten** | Transport, Verpackung, Versicherung |
| **Bezugspreis** | Tatsächlicher Einkaufspreis (Einstandspreis) |
| **Zahlungsziel** | Zeitraum bis zur Zahlung |

---

## Prüfungstipps

1. **Schema auswendig lernen:**
   - Listenpreis - Rabatt = Zieleinkaufspreis
   - Zieleinkaufspreis - Skonto = Bareinkaufspreis
   - Bareinkaufspreis + Bezugskosten = **Bezugspreis**

2. **Reihenfolge beachten:**
   - Erst Rabatt, dann Skonto!
   - Skonto berechnet sich vom ZIELEINKAUFSPREIS (nicht vom Listenpreis!)

3. **Typische Prüfungsfehler:**
   - Skonto vom Listenpreis berechnen (FALSCH!)
   - Bezugskosten vergessen
   - Prozentrechnung falsch (15% von 1000 = 150, nicht 850!)

4. **Prüfungsformat:**
   - Oft: Tabelle mit Lücken ausfüllen
   - Oder: Günstigsten Anbieter ermitteln

## Übungsaufgaben

### Aufgabe 1: Bezugspreisberechnung

Die ConSystem GmbH erhält folgendes Angebot für 20 Monitore:

- Listenpreis: 300 € pro Stück
- Mengenrabatt: 10%
- Skonto: 3% bei Zahlung innerhalb 14 Tagen
- Lieferkosten: 80 € pauschal
- Verpackung: 2 € pro Stück

Berechnen Sie den Bezugspreis und den Stückpreis.

<details>
<summary>Lösung anzeigen</summary>

**Berechnung:**

Listeneinkaufspreis: 300 € × 20 = **6.000,00 €**
- Rabatt 10%: **- 600,00 €**
───────────────────────────────────────
= Zieleinkaufspreis: **5.400,00 €**
- Skonto 3%: **- 162,00 €**
───────────────────────────────────────
= Bareinkaufspreis: **5.238,00 €**
+ Lieferkosten: **+ 80,00 €**
+ Verpackung (20 × 2 €): **+ 40,00 €**
───────────────────────────────────────
= **Bezugspreis: 5.358,00 €**

**Stückpreis:** 5.358 € ÷ 20 = **267,90 €**

</details>

### Aufgabe 2: Angebotsvergleich

Vergleichen Sie zwei Angebote für 50 Tastaturen:

| Position | Anbieter X | Anbieter Y |
|----------|------------|------------|
| Listenpreis/Stück | 45 € | 42 € |
| Rabatt | 20% | 15% |
| Skonto | 2% | 3% |
| Versandkosten | 50 € | kostenlos |

Welcher Anbieter ist günstiger?

<details>
<summary>Lösung anzeigen</summary>

**Anbieter X:**
- Listenpreis: 45 € × 50 = 2.250 €
- Rabatt 20%: - 450 €
- Zieleinkaufspreis: 1.800 €
- Skonto 2%: - 36 €
- Bareinkaufspreis: 1.764 €
- Versand: + 50 €
- **Bezugspreis X: 1.814 €**

**Anbieter Y:**
- Listenpreis: 42 € × 50 = 2.100 €
- Rabatt 15%: - 315 €
- Zieleinkaufspreis: 1.785 €
- Skonto 3%: - 53,55 €
- Bareinkaufspreis: 1.731,45 €
- Versand: 0 €
- **Bezugspreis Y: 1.731,45 €**

**Ergebnis: Anbieter Y ist um 82,55 € günstiger.**

(Obwohl der Rabatt geringer ist, macht der höhere Skonto und der kostenlose Versand den Unterschied.)

</details>

### Aufgabe 3: Kombinierte Bewertung

Die ConSystem GmbH hat folgende Angebote für Server erhalten:

| Kriterium | Anbieter 1 | Anbieter 2 |
|-----------|------------|------------|
| Bezugspreis | 12.500 € | 11.800 € |
| Lieferzeit | 5 Tage | 21 Tage |
| Garantie | 5 Jahre | 3 Jahre |
| Support | 24/7 Vor-Ort | Telefon werktags |

Der Server wird dringend benötigt (Projekt startet in 10 Tagen). Bei Ausfall entstehen hohe Kosten.

a) Welcher Anbieter ist quantitativ günstiger?
b) Welche qualitativen Faktoren sprechen für Anbieter 1?
c) Welche Empfehlung geben Sie?

<details>
<summary>Lösung anzeigen</summary>

a) **Quantitativ:** Anbieter 2 ist 700 € günstiger (11.800 € vs. 12.500 €)

b) **Qualitative Vorteile Anbieter 1:**
- Lieferzeit 5 Tage (Projekt kann starten!)
- 5 Jahre Garantie (2 Jahre länger)
- 24/7 Vor-Ort-Support (kritisch bei Serverausfall)

c) **Empfehlung: Anbieter 1**

Begründung:
- Anbieter 2 kann nicht rechtzeitig liefern (21 Tage > 10 Tage bis Projektstart)
- Der Mehrpreis von 700 € ist gering im Vergleich zu Projektkosten bei Verzögerung
- Besserer Support reduziert Ausfallrisiken
- Längere Garantie spart langfristig Kosten

Der günstigere Preis von Anbieter 2 kann die Nachteile nicht aufwiegen.

</details>
