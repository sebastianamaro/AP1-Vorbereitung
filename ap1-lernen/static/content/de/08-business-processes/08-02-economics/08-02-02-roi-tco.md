# ROI und TCO

## Lernziele

Nach diesem Abschnitt kannst du:
- Return on Investment (ROI) berechnen und interpretieren
- Total Cost of Ownership (TCO) ermitteln
- Beide Kennzahlen für IT-Investitionen anwenden
- Wirtschaftlichkeit von IT-Projekten bewerten

---

## Kerninhalt

### Return on Investment (ROI)

Der **ROI** misst die Rendite einer Investition - wie viel Gewinn wird im Verhältnis zum eingesetzten Kapital erzielt?

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ROI - RETURN ON INVESTMENT                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                         Gewinn                                       │
│         ROI  =  ─────────────────────  ×  100%                       │
│                    Investition                                       │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  Oder alternativ:                                                    │
│                                                                      │
│               Ertrag  -  Investition                                 │
│         ROI  =  ─────────────────────────  ×  100%                   │
│                      Investition                                     │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  BEISPIEL:                                                          │
│                                                                      │
│  Investition in neue Server: 50.000 €                               │
│  Einsparung durch Effizienz: 15.000 €/Jahr                          │
│  Betrachtungszeitraum: 3 Jahre                                      │
│                                                                      │
│  Gewinn = (15.000 € × 3) - 50.000 € = 45.000 € - 50.000 € = -5.000 €│
│                                                                      │
│  ODER: Ertrag gesamt = 45.000 €                                     │
│                                                                      │
│         45.000 € - 50.000 €     -5.000 €                            │
│  ROI = ─────────────────────  = ────────── = -10%                   │
│              50.000 €            50.000 €                            │
│                                                                      │
│  → Investition rechnet sich in 3 Jahren noch NICHT                  │
│                                                                      │
│  Bei 4 Jahren: (15.000 × 4) - 50.000 = 10.000 €                     │
│  ROI = 10.000 / 50.000 = +20% ✓                                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### ROI-Interpretation

| ROI-Wert | Bedeutung |
|----------|-----------|
| **> 0%** | Investition bringt Gewinn |
| **= 0%** | Investition ist kostenneutral |
| **< 0%** | Investition bringt Verlust |
| **> 20%** | Gute Rendite (abhängig von Branche) |

### Amortisationszeit (Payback)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    AMORTISATIONSZEIT                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                        Investition                                   │
│  Amortisation  =  ─────────────────────────                          │
│                   Jährliche Einsparung                               │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  BEISPIEL:                                                          │
│                                                                      │
│  Investition: 50.000 €                                              │
│  Jährliche Einsparung: 15.000 €                                     │
│                                                                      │
│                 50.000 €                                             │
│  Amortisation = ───────── = 3,33 Jahre                              │
│                 15.000 €                                             │
│                                                                      │
│  → Nach 3 Jahren und 4 Monaten hat sich die Investition bezahlt    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Total Cost of Ownership (TCO)

**TCO** erfasst ALLE Kosten, die über die gesamte Nutzungsdauer eines Produkts oder Systems anfallen.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TCO - TOTAL COST OF OWNERSHIP                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  TCO = Anschaffungskosten + Betriebskosten + Entsorgungskosten      │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │     ANSCHAFFUNG          BETRIEB             ENDE           │    │
│  │     ────────────         ───────             ────           │    │
│  │                                                              │    │
│  │     • Kaufpreis          • Wartung           • Entsorgung   │    │
│  │     • Installation       • Updates           • Datenlöschung│    │
│  │     • Schulung           • Support           • Abbau        │    │
│  │     • Konfiguration      • Energie           • Migration    │    │
│  │     • Datenmigration     • Verbrauchsmaterial              │    │
│  │                          • Reparaturen                      │    │
│  │                          • Lizenzen                         │    │
│  │                                                              │    │
│  │  ←────────────────── Nutzungsdauer ─────────────────────→   │    │
│  │         (typisch 3-5 Jahre für IT-Hardware)                 │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  WICHTIG: Der Kaufpreis ist oft nur 20-30% der TCO!                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### TCO-Beispiel: PC-Arbeitsplatz

```
┌─────────────────────────────────────────────────────────────────────┐
│              TCO BEISPIEL: PC-ARBEITSPLATZ (5 Jahre)                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ANSCHAFFUNGSKOSTEN (einmalig):                                     │
│  ├── Hardware (PC, Monitor, Peripherie)      1.200 €                │
│  ├── Software-Lizenzen                         300 €                │
│  ├── Installation und Konfiguration            150 €                │
│  └── Schulung Mitarbeiter                      200 €                │
│                                          ──────────                 │
│                               Summe:        1.850 €                 │
│                                                                      │
│  BETRIEBSKOSTEN (pro Jahr × 5 Jahre):                               │
│  ├── Wartung und Support                       300 €                │
│  ├── Software-Updates                          100 €                │
│  ├── Energiekosten                              80 €                │
│  ├── Verbrauchsmaterial                         50 €                │
│  └── Anteilige IT-Personalkosten               400 €                │
│                                          ──────────                 │
│                        Pro Jahr:            930 €                   │
│                        × 5 Jahre:         4.650 €                   │
│                                                                      │
│  ENTSORGUNGSKOSTEN:                                                 │
│  ├── Datenlöschung                              50 €                │
│  └── Entsorgung/Recycling                       30 €                │
│                                          ──────────                 │
│                               Summe:          80 €                  │
│                                                                      │
│  ════════════════════════════════════════════════════════════════   │
│                                                                      │
│  TOTAL COST OF OWNERSHIP:                                           │
│  Anschaffung:       1.850 €                                         │
│  Betrieb (5 J.):    4.650 €                                         │
│  Entsorgung:           80 €                                         │
│                   ──────────                                        │
│  TCO GESAMT:        6.580 €                                         │
│                                                                      │
│  → Anschaffung ist nur 28% der TCO!                                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### ROI vs. TCO

| Kennzahl | Fragestellung | Berechnet |
|----------|---------------|-----------|
| **ROI** | Lohnt sich die Investition? | Rendite / Gewinn |
| **TCO** | Was kostet es insgesamt? | Alle Kosten |

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ROI vs. TCO                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ROI beantwortet:                TCO beantwortet:                   │
│  "Wie rentabel ist             "Was kostet das                      │
│   die Investition?"              insgesamt?"                        │
│                                                                      │
│  Verwendet für:                  Verwendet für:                     │
│  → Investitionsentscheidung     → Budgetplanung                    │
│  → Projektbewertung             → Vergleich von Alternativen       │
│  → Wirtschaftlichkeitsnachweis  → Kostenprognose                   │
│                                                                      │
│  TCO kann IN ROI einfließen:                                        │
│                                                                      │
│             Einsparung durch neue Lösung - TCO neue Lösung          │
│  ROI  =    ────────────────────────────────────────────── × 100%   │
│                          TCO neue Lösung                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **ROI** | Return on Investment - Rendite einer Investition |
| **TCO** | Total Cost of Ownership - Gesamtbetriebskosten |
| **Amortisation** | Zeitraum bis zur Kostendeckung |
| **Payback** | Englisch für Amortisationszeit |
| **Break-Even** | Punkt, ab dem Gewinn erzielt wird |
| **Nutzungsdauer** | Geplante Einsatzzeit (z.B. 5 Jahre) |

---

## Prüfungstipps

1. **ROI-Formel merken:**
   - ROI = Gewinn / Investition × 100%
   - Positiver ROI = Investition lohnt sich

2. **TCO nicht vergessen:**
   - Kaufpreis ist NICHT alles!
   - Betriebskosten oft höher als Anschaffung

3. **Typische Prüfungsaufgaben:**
   - ROI berechnen
   - TCO-Positionen benennen
   - Amortisationszeit ermitteln

4. **Häufiger Fehler:**
   - Bei TCO die Betriebskosten × Nutzungsdauer rechnen!

## Übungsaufgaben

### Aufgabe 1: ROI-Berechnung

Die ConSystem GmbH investiert 30.000 € in ein neues Ticketsystem. Durch effizientere Abläufe werden jährlich 8.000 € eingespart.

a) Berechnen Sie den ROI nach 3 Jahren.
b) Berechnen Sie die Amortisationszeit.
c) Lohnt sich die Investition?

<details>
<summary>Lösung anzeigen</summary>

a) **ROI nach 3 Jahren:**
- Einsparung: 8.000 € × 3 = 24.000 €
- Gewinn: 24.000 € - 30.000 € = -6.000 €
- ROI = -6.000 € / 30.000 € × 100% = **-20%**

b) **Amortisationszeit:**
- 30.000 € / 8.000 € = **3,75 Jahre**

c) **Bewertung:**
- Nach 3 Jahren: ROI negativ → lohnt sich noch nicht
- Nach 4 Jahren: (8.000 × 4 - 30.000) / 30.000 = 2.000 / 30.000 = **+6,7%**
- **Ab dem 4. Jahr lohnt sich die Investition.**

</details>

### Aufgabe 2: TCO-Vergleich

Vergleichen Sie zwei Alternativen für ein ERP-System:

| Position | Lösung A (On-Premise) | Lösung B (Cloud) |
|----------|----------------------|------------------|
| Anschaffung | 50.000 € | 0 € |
| Installation | 10.000 € | 5.000 € |
| Jährliche Lizenz | 5.000 € | 15.000 € |
| Wartung/Jahr | 8.000 € | 0 € (inkl.) |
| Hardware/Jahr | 2.000 € | 0 € |

Nutzungsdauer: 5 Jahre

Welche Lösung hat die niedrigere TCO?

<details>
<summary>Lösung anzeigen</summary>

**Lösung A (On-Premise):**
- Anschaffung: 50.000 €
- Installation: 10.000 €
- Lizenzen (5 Jahre): 5.000 € × 5 = 25.000 €
- Wartung (5 Jahre): 8.000 € × 5 = 40.000 €
- Hardware (5 Jahre): 2.000 € × 5 = 10.000 €
- **TCO Lösung A: 135.000 €**

**Lösung B (Cloud):**
- Anschaffung: 0 €
- Installation: 5.000 €
- Lizenzen (5 Jahre): 15.000 € × 5 = 75.000 €
- Wartung: 0 € (inklusive)
- Hardware: 0 €
- **TCO Lösung B: 80.000 €**

**Ergebnis: Lösung B (Cloud) ist um 55.000 € günstiger über 5 Jahre.**

</details>

### Aufgabe 3: Kombinierte Berechnung

Ein Unternehmen überlegt, seine alte Server-Infrastruktur (TCO: 80.000 €/Jahr) durch eine Cloud-Lösung (TCO: 50.000 €/Jahr) zu ersetzen. Die Migration kostet einmalig 60.000 €.

a) Wie hoch ist die jährliche Einsparung?
b) Nach wie vielen Jahren amortisiert sich die Migration?
c) Wie hoch ist der ROI nach 5 Jahren?

<details>
<summary>Lösung anzeigen</summary>

a) **Jährliche Einsparung:**
80.000 € - 50.000 € = **30.000 €/Jahr**

b) **Amortisationszeit:**
60.000 € / 30.000 € = **2 Jahre**

c) **ROI nach 5 Jahren:**
- Einsparung gesamt: 30.000 € × 5 = 150.000 €
- Gewinn: 150.000 € - 60.000 € = 90.000 €
- ROI = 90.000 € / 60.000 € × 100% = **150%**

**Fazit:** Sehr gute Investition - hoher ROI, schnelle Amortisation.

</details>
