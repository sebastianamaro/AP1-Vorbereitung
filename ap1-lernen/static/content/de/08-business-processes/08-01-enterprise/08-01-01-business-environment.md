# Unternehmensumfeld

Nach diesem Abschnitt kannst du:

- Stakeholder eines Unternehmens benennen
- Das Unternehmensumfeld analysieren
- Die Rolle von IT-Unternehmen im Wirtschaftssystem einordnen

### IT-Unternehmen im Wirtschaftssystem

| Sektor | IT-Beispiele |
|--------|--------------|
| **Primär** | IT in der Landwirtschaft (Smart Farming) |
| **Sekundär** | Hardwarehersteller, Chip-Produktion |
| **Tertiär** | Systemhäuser, Cloud-Provider, IT-Beratung |

Die meisten IT-Unternehmen gehören zum **tertiären Sektor** (Dienstleistungen).

### Stakeholder (Anspruchsgruppen)

**Stakeholder** sind alle Personen und Gruppen, die ein Interesse am Unternehmen haben oder von seinen Handlungen betroffen sind.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    STAKEHOLDER                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                         ┌───────────────┐                           │
│           Eigentümer    │               │    Kunden                 │
│         (Gesellschafter)│  UNTERNEHMEN  │  (Abnehmer)               │
│                 ↘       │               │       ↙                   │
│                         │   ConSystem   │                           │
│    Mitarbeiter ────────►│     GmbH      │◄──────── Lieferanten     │
│                         │               │                           │
│                 ↗       │               │       ↖                   │
│            Staat        │               │    Banken                 │
│          (Behörden)     └───────────────┘  (Kapitalgeber)           │
│                                 ↑                                    │
│                           Gesellschaft                               │
│                         (Öffentlichkeit)                             │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Stakeholder-Interessen

| Stakeholder | Interessen |
|-------------|------------|
| **Eigentümer** | Gewinn, Wertsteigerung, Unternehmenserfolg |
| **Mitarbeiter** | Sicherer Arbeitsplatz, gutes Gehalt, Entwicklung |
| **Kunden** | Qualität, faire Preise, guter Service |
| **Lieferanten** | Zahlungssicherheit, langfristige Beziehung |
| **Banken** | Zinsen, Rückzahlung, Kreditwürdigkeit |
| **Staat** | Steuern, Arbeitsplätze, Einhaltung von Gesetzen |
| **Gesellschaft** | Umweltschutz, soziale Verantwortung |

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Primärsektor** | Urproduktion (Landwirtschaft, Bergbau) |
| **Sekundärsektor** | Verarbeitendes Gewerbe, Industrie |
| **Tertiärsektor** | Dienstleistungen |
| **Stakeholder** | Alle Interessengruppen eines Unternehmens |
| **Shareholder** | Anteilseigner/Eigentümer |
| **Makroumfeld** | Externe, nicht beeinflussbare Faktoren |
| **Mikroumfeld** | Nahes Umfeld, teilweise beeinflussbar |

---

## Prüfungstipps

1. **Sektoren-Zuordnung:**
   - IT-Dienstleister = Tertiär (Dienstleistung)
   - Hardwarehersteller = Sekundär (Produktion)

2. **Stakeholder merken:**
   - **E**igentümer, **M**itarbeiter, **K**unden, **L**ieferanten, **B**anken, **S**taat
   - "**E**in **M**itarbeiter **K**auft **L**eckere **B**rötchen **S**onntags"

3. **Typische Prüfungsfragen:**
   - Welche Stakeholder hat ein IT-Unternehmen?
   - Unterschied Shareholder/Stakeholder?
   - Zuordnung zu Wirtschaftssektoren

## Übungsaufgaben

### Aufgabe 1: Sektoren zuordnen

Ordnen Sie folgende Unternehmen dem richtigen Sektor zu:

a) Ein Cloud-Dienstleister
b) Ein Chiphersteller (Intel, AMD)
c) Ein Rechenzentrum, das Serverhosting anbietet
d) Ein Kupferbergwerk für Kabelherstellung

<details>
<summary>Lösung anzeigen</summary>

a) **Tertiärer Sektor** - Cloud ist eine Dienstleistung
b) **Sekundärer Sektor** - Produktion/Herstellung von Hardware
c) **Tertiärer Sektor** - Hosting ist eine Dienstleistung
d) **Primärer Sektor** - Rohstoffgewinnung

</details>

### Aufgabe 2: Stakeholder-Analyse

Die ConSystem GmbH plant die Einführung eines neuen ERP-Systems. Welche Stakeholder sind betroffen und welche Interessen haben sie?

<details>
<summary>Lösung anzeigen</summary>

| Stakeholder | Interesse am ERP-System |
|-------------|-------------------------|
| **Geschäftsführung** | Effizienzsteigerung, Kostensenkung, bessere Daten |
| **Mitarbeiter** | Benutzerfreundlichkeit, Schulung, keine Mehrarbeit |
| **IT-Abteilung** | Wartbarkeit, Integration, Support |
| **Kunden** | Schnellere Auftragsabwicklung |
| **Lieferanten** | Bessere Bestellprozesse |
| **Banken** | ROI der Investition |

</details>

### Aufgabe 3: Fallbeispiel

Ein neues Datenschutzgesetz (DSGVO-Änderung) wird verabschiedet. Zu welchem Teil des Unternehmensumfelds gehört dies und warum kann das Unternehmen darauf nicht direkt Einfluss nehmen?

<details>
<summary>Lösung anzeigen</summary>

**Einordnung:** Makroumfeld (Politik/Recht)

**Begründung:**
- Gesetze werden von staatlichen Organen verabschiedet
- Einzelne Unternehmen können Gesetze nicht direkt beeinflussen
- Das Unternehmen muss sich an das Gesetz anpassen (Compliance)

**Mögliche indirekte Einflussnahme:**
- Über Branchenverbände (z.B. BITKOM)
- Lobbyarbeit
- Stellungnahmen zu Gesetzesvorhaben

</details>
