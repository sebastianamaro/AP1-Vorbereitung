## Kerninhalt

### Was ist Projektmanagement?

**Projektmanagement** umfasst das Planen, Steuern und Kontrollieren von Projekten. Es beinhaltet die Anwendung von Methoden, Hilfsmitteln, Techniken sowie Kompetenzen in einem Projekt (nach DIN ISO 21500).

### Managementbereiche im Projektmanagement

Das Projektmanagement umfasst oder verwendet folgende Bereiche:

| Bereich | Beschreibung |
|---------|-------------|
| **Terminmanagement** | Zeit- und Ablaufplanung |
| **Kostenmanagement** | Budgetplanung und -kontrolle |
| **Qualitätsmanagement** | Sicherung der Ergebnisqualität |
| **Risikomanagement** | Identifikation und Umgang mit Risiken |
| **Kommunikationsmanagement** | Informationsfluss im Projekt |
| **Personalmanagement** | Teamzusammensetzung und -führung |
| **Beschaffungsmanagement** | Einkauf von Ressourcen und Leistungen |
| **Inhalts- und Umfangsmanagement** | Definition des Projektumfangs |
| **Integrationsmanagement** | Koordination aller Bereiche |

---

### Das magische Dreieck des Projektmanagements

Die drei zentralen Größen eines Projekts stehen in einem Spannungsverhältnis:

```
           QUALITÄT
              /\
             /  \
            /    \
           /      \
          /________\
     KOSTEN      ZEIT
```

**Zusammenhang:** Ändert sich eine Größe, hat das Auswirkungen auf die anderen beiden. Beispiel: Weniger Zeit führt oft zu höheren Kosten oder geringerer Qualität.

---

### Normen und Standards

| Standard | Beschreibung |
|----------|-------------|
| **DIN 69901** | Anforderungen an Projektmanagementsysteme |
| **DIN ISO 21500** | Leitfaden zum Projektmanagement |
| **PRINCE2** | Prozessorientierte, skalierbare Vorgehensweise (weltweit verbreitet) |
| **Agiles Manifest** | Grundlage für agiles Projektmanagement |
