# Qualitätsmanagementsysteme (QMS)

## Lernziele

Nach diesem Kapitel solltest du:
- Wissen, was ein Qualitätsmanagementsystem ist
- Die Bestandteile eines QMS kennen
- Die Schritte zur Einführung eines QMS beschreiben können
- Den Unterschied zwischen QMS-Modellen kennen

---

## Kerninhalt

### Was ist ein QMS?

Ein **Qualitätsmanagementsystem (QMS)** ist ein strukturierter Ansatz zur Sicherstellung und kontinuierlichen Verbesserung der Qualität in Unternehmen. Es umfasst:

- **Prozesse:** Definierte Arbeitsabläufe
- **Richtlinien:** Regeln und Vorgaben
- **Verantwortlichkeiten:** Klare Zuständigkeiten
- **Dokumentation:** Nachvollziehbare Aufzeichnungen

---

### Bestandteile eines QMS

```
┌─────────────────────────────────────────────────────────────┐
│            QUALITÄTSMANAGEMENTSYSTEM (QMS)                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────┐    ┌─────────────────┐                │
│  │ QUALITÄTSPOLITIK│    │ QUALITÄTSZIELE  │                │
│  │ (Was wollen wir │    │ (Messbare Ziele)│                │
│  │  erreichen?)    │    │                 │                │
│  └────────┬────────┘    └────────┬────────┘                │
│           │                      │                          │
│           └──────────┬───────────┘                          │
│                      ▼                                      │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              AUFBAUORGANISATION                      │   │
│  │  (Strukturen, Rollen, Verantwortlichkeiten)         │   │
│  └─────────────────────────────────────────────────────┘   │
│                      │                                      │
│                      ▼                                      │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              ABLAUFORGANISATION                      │   │
│  │  (Prozesse, Verfahren, Arbeitsanweisungen)          │   │
│  └─────────────────────────────────────────────────────┘   │
│                      │                                      │
│                      ▼                                      │
│  ┌─────────────────────────────────────────────────────┐   │
│  │             RESSOURCENMANAGEMENT                     │   │
│  │  (Personal, Infrastruktur, Arbeitsmittel)           │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### Die fünf Bausteine im Detail

| Baustein | Beschreibung | Beispiel |
|----------|--------------|----------|
| **Qualitätspolitik** | Grundsätzliche Ausrichtung und Absichten | "Wir streben 100% Kundenzufriedenheit an" |
| **Qualitätsziele** | Konkrete, messbare Ziele | "Fehlerquote unter 0,1%" |
| **Aufbauorganisation** | Strukturen und Verantwortlichkeiten | Organigramm, QM-Beauftragter |
| **Ablauforganisation** | Definierte Prozesse und Abläufe | Prozessbeschreibungen, Arbeitsanweisungen |
| **Ressourcenmanagement** | Bereitstellung notwendiger Ressourcen | Personal, IT-Systeme, Budget |

### Einführung eines QMS

Die Einführung eines QMS erfolgt in mehreren Schritten:

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 1. IST-     │ ─► │ 2. ZIEL-    │ ─► │ 3. QM-HAND- │
│    ANALYSE  │    │    ZUSTAND  │    │    BUCH     │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 6. ZERTI-   │ ◄─ │ 5. INTERNE  │ ◄─ │ 4. SCHULUNG │
│    FIZIERUNG│    │    AUDITS   │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
```

| Schritt | Aktivitäten |
|---------|-------------|
| **1. Ist-Analyse** | Aktuelle Prozesse und Strukturen erfassen |
| **2. Ziel-Zustand definieren** | Qualitätsziele und Standards festlegen |
| **3. QM-Handbuch erstellen** | Prozesse dokumentieren, Verantwortlichkeiten festlegen |
| **4. Mitarbeiter schulen** | Alle Beteiligten mit dem QMS vertraut machen |
| **5. Interne Audits** | Einhaltung der Vorgaben prüfen |
| **6. Zertifizierung** | Externe Prüfung durch Zertifizierungsstelle |

---

### QMS-Modelle im Vergleich

| Aspekt | DIN EN ISO 9001 | EFQM-Modell |
|--------|-----------------|-------------|
| **Herkunft** | Internationale Norm | European Foundation for Quality Management |
| **Ansatz** | Anforderungsorientiert | Selbstbewertung und Excellence |
| **Zertifizierung** | Ja, durch Auditoren | Nein, Selbstbewertung |
| **Fokus** | Prozesse und Dokumentation | Ganzheitliche Unternehmensführung |
| **Philosophie** | Mindestanforderungen erfüllen | Excellence anstreben |

---

### Das QM-Handbuch

Das **QM-Handbuch** ist das zentrale Dokument eines QMS:

**Typische Inhalte:**
- Qualitätspolitik und -ziele
- Organisationsstruktur
- Prozessbeschreibungen
- Verantwortlichkeiten
- Verfahrensanweisungen
- Formulare und Checklisten

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **QMS** | Qualitätsmanagementsystem - strukturierter Ansatz zur Qualitätssicherung |
| **Qualitätspolitik** | Grundsätzliche Ausrichtung eines Unternehmens bezüglich Qualität |
| **Qualitätsziele** | Konkrete, messbare Vorgaben zur Qualität |
| **QM-Handbuch** | Zentrales Dokument mit allen QMS-Informationen |
| **Audit** | Systematische Überprüfung der QMS-Einhaltung |
| **Aufbauorganisation** | Hierarchische Struktur (wer ist wofür zuständig) |
| **Ablauforganisation** | Prozessuale Struktur (wie werden Aufgaben erledigt) |

## Prüfungstipps

### Häufige Prüfungsfragen
- Welche Bestandteile hat ein QMS?
- In welcher Reihenfolge erfolgt die Einführung eines QMS?
- Was ist der Unterschied zwischen DIN EN ISO 9001 und EFQM?

### Wichtig für die Prüfung
- Die Reihenfolge der QMS-Einführung merken: **I**st-**Z**ustand → **Z**iel → **H**andbuch → **S**chulung → **A**udits → **Z**ertifizierung

---

## Übungsaufgaben

### Aufgabe 1
Bringen Sie die folgenden Schritte zur Einführung eines QMS in die richtige Reihenfolge:
- Zertifizierung
- Schulung der Mitarbeiter
- Ist-Analyse
- Interne Audits
- QM-Handbuch verfassen

**Lösung:**
1. Ist-Analyse
2. QM-Handbuch verfassen
3. Schulung der Mitarbeiter
4. Interne Audits
5. Zertifizierung

### Aufgabe 2
Definieren Sie die Begriffe Qualitätsplanung und Qualitätspolitik.

**Lösung:**
- **Qualitätsplanung:** Festlegung der Qualitätsziele und Maßnahmen zur Erreichung dieser Ziele
- **Qualitätspolitik:** Die grundsätzlichen Absichten und Ausrichtung einer Organisation bezüglich Qualität, formuliert von der obersten Leitung

### Aufgabe 3
Nennen Sie drei Bestandteile eines QMS und erklären Sie diese kurz.

**Lösung:**
1. **Qualitätspolitik:** Grundsätzliche Qualitätsausrichtung des Unternehmens
2. **Aufbauorganisation:** Strukturen, Rollen und Verantwortlichkeiten
3. **Ablauforganisation:** Definierte Prozesse und Arbeitsabläufe
