# Market Forms

## Learning Objectives

After this section, you will be able to:
- Distinguish the three basic market forms
- Explain advantages and disadvantages of each market form
- Identify market forms in the IT sector
- Describe effects on pricing and competition

---

## Core Content

### Overview of Market Forms

The market form is determined by the **number of suppliers** in a market.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MARKET FORMS                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Number of suppliers:  ONE           FEW            MANY            │
│                         │             │               │              │
│                         ▼             ▼               ▼              │
│               ┌───────────┐   ┌───────────┐   ┌───────────┐         │
│               │  MONOPOLY │   │  OLIGOPOLY│   │  POLYPOLY │         │
│               │           │   │           │   │           │         │
│               │ One       │   │ Few       │   │ Many      │         │
│               │ Supplier  │   │ Suppliers │   │ Suppliers │         │
│               └───────────┘   └───────────┘   └───────────┘         │
│                                                                      │
│  Market power:   HIGH          MEDIUM           LOW                 │
│                                                                      │
│  Price is        by the        through          by the              │
│  determined:   monopolist     agreements*       market              │
│                                                                      │
│  *Agreements (cartels) are prohibited!                              │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 1. Monopoly (One Supplier)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MONOPOLY                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│             ┌────────────────────┐                                  │
│             │                    │                                  │
│             │    MONOPOLIST      │                                  │
│             │    (1 Supplier)    │                                  │
│             │                    │                                  │
│             └─────────┬──────────┘                                  │
│                       │                                             │
│          ┌────────────┼────────────┐                                │
│          ▼            ▼            ▼                                │
│       ┌─────┐     ┌─────┐     ┌─────┐                               │
│       │Cust.│     │Cust.│     │Cust.│  ← No alternative            │
│       └─────┘     └─────┘     └─────┘                               │
│                                                                      │
│  Characteristics:                                                   │
│  → No competition                                                   │
│  → Supplier determines the price                                    │
│  → Consumers have no choice                                         │
│                                                                      │
│  Examples:                                                          │
│  → Previously: Deutsche Post (mail monopoly, abolished)             │
│  → Network operators in a region                                    │
│  → Patent-protected medications                                     │
│                                                                      │
│  IT Example:                                                        │
│  → Microsoft in the operating system market (quasi-monopoly)        │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

| Advantages | Disadvantages |
|------------|---------------|
| Uniform standards | High prices for consumers |
| No duplicate infrastructure | No pressure to innovate |
| Planning security | Customer dependency |

### 2. Oligopoly (Few Suppliers)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OLIGOPOLY                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│       ┌─────────┐     ┌─────────┐     ┌─────────┐                   │
│       │Supplier │     │Supplier │     │Supplier │                   │
│       │    A    │     │    B    │     │    C    │                   │
│       └────┬────┘     └────┬────┘     └────┬────┘                   │
│            │               │               │                        │
│            │   observe each other          │                        │
│            └───────────────┼───────────────┘                        │
│                            │                                        │
│                ┌───────────┼───────────┐                            │
│                ▼           ▼           ▼                            │
│             ┌─────┐     ┌─────┐     ┌─────┐                         │
│             │Cust.│     │Cust.│     │Cust.│  ← Limited choice       │
│             └─────┘     └─────┘     └─────┘                         │
│                                                                      │
│  Characteristics:                                                   │
│  → Limited number of large suppliers                                │
│  → High market entry barriers                                       │
│  → Suppliers observe each other                                     │
│  → Risk of price agreements (illegal!)                              │
│                                                                      │
│  Examples:                                                          │
│  → Mobile networks (Telekom, Vodafone, O2)                          │
│  → Gas stations (Shell, Aral, Esso, Total)                          │
│  → Cloud market (AWS, Azure, Google Cloud)                          │
│                                                                      │
│  IT Examples:                                                       │
│  → Processors (Intel, AMD)                                          │
│  → Graphics cards (NVIDIA, AMD)                                     │
│  → Smartphone OS (Android, iOS)                                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

| Advantages | Disadvantages |
|------------|---------------|
| More choice than monopoly | Little real competition |
| High quality through competition | Prices can be coordinated |
| Investments in innovation | High entry barriers for new suppliers |

### 3. Polypoly (Many Suppliers)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    POLYPOLY                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐      │
│  │ A │ │ B │ │ C │ │ D │ │ E │ │ F │ │ G │ │ H │ │ I │ │ J │      │
│  └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘      │
│    │     │     │     │     │     │     │     │     │     │          │
│    └─────┴─────┴─────┴─────┼─────┴─────┴─────┴─────┴─────┘          │
│                            │                                        │
│                    ┌───────┼───────┐                                │
│                    ▼       ▼       ▼                                │
│                 ┌─────┐ ┌─────┐ ┌─────┐                             │
│                 │Cust.│ │Cust.│ │Cust.│  ← Large selection          │
│                 └─────┘ └─────┘ └─────┘                             │
│                                                                      │
│  Characteristics:                                                   │
│  → Many suppliers compete                                           │
│  → No single supplier can determine the price                       │
│  → The market regulates the price                                   │
│  → Consumer-friendly                                                │
│                                                                      │
│  Examples:                                                          │
│  → Bakeries                                                         │
│  → Restaurants                                                      │
│  → Hair salons                                                      │
│                                                                      │
│  IT Examples:                                                       │
│  → Web developers / agencies                                        │
│  → IT system integrators                                            │
│  → Software development (many small companies)                      │
│  → PC dealers                                                       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

| Advantages | Disadvantages |
|------------|---------------|
| Large selection for consumers | Difficult orientation |
| Fair prices through competition | Quality differences |
| Low market entry | Small suppliers often unstable |

### Comparison Table

| Criterion | Monopoly | Oligopoly | Polypoly |
|-----------|----------|-----------|----------|
| **Number of suppliers** | One | Few (2-10) | Many |
| **Market power** | Very high | High | Low |
| **Price formation** | Supplier determines | Mutual observation | Market |
| **Competition** | None | Limited | Intense |
| **Entry barriers** | Very high | High | Low |
| **Innovation** | Low | Medium | High |

## Key Terms

| Term | Explanation |
|------|-------------|
| **Monopoly** | Market with only one supplier |
| **Oligopoly** | Market with few large suppliers |
| **Polypoly** | Market with many suppliers |
| **Market entry barrier** | Obstacles for new suppliers |
| **Cartel** | Illegal agreement between competitors |
| **Market power** | Ability to influence prices and conditions |

## Practice Exercises

### Exercise 1: Classification

Assign the following markets to the correct market form:

a) Operating systems for smartphones
b) Local IT system integrators
c) Fiber optic network provider in a rural region
d) CPU manufacturers for desktop PCs

<details>
<summary>Show Solution</summary>

a) **Oligopoly** - Android and iOS dominate (two suppliers)
b) **Polypoly** - Many small and medium-sized suppliers
c) **Monopoly** - Often only one provider in rural regions
d) **Oligopoly** - Intel and AMD (duopoly)

</details>

### Exercise 2: Price Formation

ConSystem GmbH wants to purchase a new laptop. Explain how the market form affects price negotiation:

a) If there were only one manufacturer (monopoly)
b) In the current situation (oligopoly: Dell, HP, Lenovo)
c) With many small local suppliers (polypoly)

<details>
<summary>Show Solution</summary>

a) **Monopoly:**
- No negotiating power for ConSystem
- Price is dictated by the supplier
- No alternative, "take it or leave it"

b) **Oligopoly (current):**
- Comparison between Dell, HP, Lenovo possible
- Obtain and negotiate offers
- Suppliers observe each other (similar prices)
- Medium negotiating position

c) **Polypoly:**
- Large selection, intense competition
- Strong negotiating position for ConSystem
- Suppliers undercut each other
- Lowest prices

</details>

### Exercise 3: Antitrust Law

Two large cloud providers meet and agree to increase their prices by 20%.

a) What is this agreement called?
b) Is this agreement legal or illegal?
c) Which authority would be responsible in Germany?

<details>
<summary>Show Solution</summary>

a) **Price cartel** (or price agreement)

b) **Illegal** - Price agreements between competitors are prohibited under the Act against Restraints of Competition (GWB).

c) **Federal Cartel Office** (for national significance) or **European Commission** (for EU-wide impact)

Possible consequences:
- High fines (up to 10% of annual revenue)
- Civil damage claims
- Reputation damage

</details>
