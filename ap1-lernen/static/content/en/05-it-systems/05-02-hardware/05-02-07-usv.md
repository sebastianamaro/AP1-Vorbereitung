## Core Content

### What is a UPS?

A **UPS (Uninterruptible Power Supply)** protects IT systems from power failures and voltage fluctuations through batteries that take over when needed.

```
┌─────────────────────────────────────────────────────────────────┐
│                    UPS FUNCTION                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Normal operation:                                             │
│                                                                 │
│   ┌────────┐      ┌──────────┐      ┌──────────┐              │
│   │ Power  │ ───► │   UPS    │ ───► │  Server  │              │
│   │ Grid   │      │(charging)│      │          │              │
│   └────────┘      └──────────┘      └──────────┘              │
│                                                                 │
│   During power failure:                                         │
│                                                                 │
│   ┌────────┐      ┌──────────┐      ┌──────────┐              │
│   │ Power  │  ✗   │   UPS    │ ───► │  Server  │              │
│   │ Grid   │      │(battery) │      │(continues│              │
│   └────────┘      └──────────┘      │ running) │              │
│                                      └──────────┘              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

### Tasks of a UPS

| Task | Description |
|------|-------------|
| **Bridging** | Power supply during grid failure |
| **Voltage Smoothing** | Compensation for fluctuations |
| **Surge Protection** | Protection against voltage spikes |
| **Safe Shutdown** | Time for orderly shutdown |

### UPS Types

```
┌────────────────────────────────────────────────────────────────┐
│                    UPS TYPES                                   │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  1. OFFLINE/STANDBY                                            │
│     ┌───────┐                                                  │
│     │  AC   │──────────────────────►  Load                    │
│     └───────┘        ↑                                         │
│                      │ (only during failure)                   │
│                 ┌────┴────┐                                    │
│                 │ Battery │                                    │
│                 └─────────┘                                    │
│     → Affordable, short transfer time (5-12 ms)               │
│                                                                │
│  2. LINE-INTERACTIVE                                           │
│     ┌───────┐    ┌──────────┐                                 │
│     │  AC   │───►│ Voltage  │────────►  Load                  │
│     └───────┘    │ Regulator│                                 │
│                  └────┬─────┘                                  │
│                       │                                        │
│                  ┌────┴────┐                                   │
│                  │ Battery │                                   │
│                  └─────────┘                                   │
│     → Voltage regulation, fast transfer (2-4 ms)              │
│                                                                │
│  3. ONLINE/DOUBLE CONVERSION                                   │
│     ┌───────┐    ┌────────┐    ┌────────┐                     │
│     │  AC   │───►│Rectifier│───►│Inverter│───► Load           │
│     └───────┘    └────┬────┘    └────────┘                     │
│                       │                                        │
│                  ┌────┴────┐                                   │
│                  │ Battery │ (always in line)                  │
│                  └─────────┘                                   │
│     → Best quality, no transfer time (0 ms)                   │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Comparison of UPS Types

| Type | Transfer Time | Protection | Efficiency | Price | Usage |
|------|---------------|------------|------------|-------|-------|
| **Offline/Standby** | 5-12 ms | Basic | High | Affordable | Home PC |
| **Line-Interactive** | 2-4 ms | Medium | Medium | Medium | Office, Server |
| **Online/Double Conversion** | 0 ms | Maximum | Lower | High | Data Center |

---

### Important Metrics

| Metric | Description | Example |
|--------|-------------|---------|
| **VA (Volt-Ampere)** | Apparent power | 1500 VA |
| **Watt** | Real power (~60-70% of VA) | 900 W |
| **Autonomy Time** | Runtime during power failure | 5-30 minutes |
| **Power Factor** | Ratio Watt/VA | 0.6 - 0.9 |

**Rule of Thumb:** Watt ≈ VA × 0.6 to 0.7

│                                                                │

### Power Grid Disturbances

| Disturbance | Description | UPS Protection |
|-------------|-------------|----------------|
| **Power Failure** | Complete failure | All types |
| **Voltage Sag** | Voltage too low | Line-Interactive, Online |
| **Overvoltage** | Voltage too high | Line-Interactive, Online |
| **Voltage Spikes** | Short spikes | All types |
| **Frequency Fluctuation** | Grid frequency unstable | Online only |
| **Noise** | Electrical interference | Online |

### Use Cases

| Scenario | UPS Type | Reasoning |
|----------|----------|-----------|
| **Home PC** | Offline/Standby | Cost-effective, simple protection |
| **Office Workstation** | Line-Interactive | Good compromise |
| **Single Server** | Line-Interactive | Fast transfer |
| **Server Room** | Online | Maximum protection |
| **Data Center** | Online + Diesel | Highest availability |

## Practice Exercises

### Exercise 1
Name the three UPS types and their transfer times.

**Solution:**
1. **Offline/Standby:** 5-12 ms transfer time
2. **Line-Interactive:** 2-4 ms transfer time
3. **Online/Double Conversion:** 0 ms transfer time

### Exercise 2
A server requires 400 Watts. What UPS capacity (VA) is recommended?

**Solution:**
1. Safety margin: 400 W × 1.3 = 520 W
2. VA calculation: 520 W ÷ 0.6 = 867 VA
3. **Recommendation:** UPS with at least **1000 VA**

### Exercise 3
Why is an Online UPS better suited for a data center than an Offline UPS?

**Solution:**
- **No transfer time:** Online UPS delivers without interruption (0 ms), Offline UPS has 5-12 ms transfer time
- **Constant voltage:** Double converter always produces clean power
- **Complete protection:** Protects against all types of disturbances (including frequency fluctuations, noise)
- **Critical systems:** Data centers require highest availability
