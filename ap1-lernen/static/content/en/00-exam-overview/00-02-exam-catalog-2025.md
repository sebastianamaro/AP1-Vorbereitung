# Exam Catalog 2025 - Changes and Focus Areas

## Learning Objectives

After this chapter, you should be able to:
- Know the most important changes in the exam catalog from 2025
- Know which topics have been NEWLY added
- Understand which topics have been REMOVED
- Properly prioritize exam focus areas

---

## Core Content

### Important Notice

Starting January 2025, a **new exam catalog** applies to AP1 for IT professions. This learning material has been updated accordingly and contains **only exam-relevant topics** according to the current catalog.

---

### NEW Topics from 2025 (High Priority)

These topics have been **newly added** and should be studied intensively:

#### Artificial Intelligence
- **AI Fundamentals:** Machine Learning, Deep Learning, Neural Networks
- **AI Applications:** ChatGPT, LLMs, practical use cases
- **AI Code Assistants:** GitHub Copilot and similar tools
- **EU AI Act:** Risk categories, regulations, terminology

#### IT Security (expanded)
- **Two-Factor Authentication (2FA):** Methods and application
- **OS Hardening:** Operating system hardening, security baselines
- **Hash Procedures:** SHA-256, integrity verification
- **Anonymization and Pseudonymization:** Differences and application

#### Project Management
- **SMART Principle:** Specific, Measurable, Achievable, Relevant, Time-bound
- **Project Economics:** Economic viability assessments for projects

#### Networks (expanded)
- **IPv4 and IPv6:** Configuration, notation, transition mechanisms
- **Data Transmission Rates:** Calculations and conversions

#### Software
- **UML Activity Diagrams:** Swimlanes, Fork/Join, activity flows
- **Desk Checking:** Manual code verification
- **Code Debugging:** Identifying errors in given code
- **Database Redundancy:** Normal forms (was tested in Spring 2025)

#### IT Systems
- **HDD vs. SSD:** Comparison and use cases
- **NAS:** Network Attached Storage (replaces RAID/SAN in exam context)
- **Cloud Computing (expanded):** IaaS, PaaS, SaaS, DaaS
- **Enterprise Software:** ERP, CRM, SCM systems
- **Social Media Platforms:** Business use
- **Domain Integration:** Adding PCs to domains

#### Accessibility
- **Website Accessibility:** WCAG guidelines, accessible design

---

### REMOVED Topics (No Longer Exam-Relevant)

These topics have been **removed from the exam catalog** and are NO longer part of AP1:

| Topic | Note |
|-------|------|
| **SQL Queries** | Moved to AP2 |
| **RAID Levels and Configurations** | Replaced by NAS |
| **SAN Systems** | Replaced by NAS |
| **LTE and 5G** | Mobile technologies removed |
| **Kanban, XP and other agile methods** | Only Scrum remains relevant |
| **SWOT Analysis** | Removed |
| **Object-Oriented Inheritance** | Only OOP basics, no inheritance |
| **Struktogramme (Nassi-Shneiderman diagrams)** | Removed |
| **Program Flowcharts (PAP/Flow diagrams)** | Removed |
| **Non-relational Databases** | NoSQL removed |
| **ISO 2700x Standards** | Only BSI Basic Protection remains |
| **Physical Formulas** | P = U × I, W = P × t removed |
| **Software Quality Criteria Documentation** | Simplified |

> **Important:** These topics are NOT covered in this learning material as they are no longer exam-relevant.

### Exam Focus Areas (based on Spring 2025)

The first exam under the new catalog (Spring 2025) showed the following focus areas:

#### Top Point Earners
1. **Utility Analysis** - Highest point value
2. **Network Planning Technique** - Second highest point value
3. **Subnetting with IPv4**
4. **IT Security and Cryptography**
5. **Email Protocols** (IMAP vs. POP3)
6. **Artificial Intelligence** (~12 points)

#### Other Tested Topics
- Hardware interfaces and connections
- Cost calculations
- Static vs. dynamic websites
- Pseudocode/Desk checking (14 points)
- Data protection and GDPR
- Public Key Infrastructure (PKI)
- Open source vs. proprietary software
- Hash verification
- SSH protocol selection

---

### Assessment of Spring 2025 Exam

Feedback on the first exam under the new catalog:
- **"Absolutely manageable exam"** without major surprises
- Questions corresponded to the new catalog
- Many tasks required **logical thinking** rather than deep expert knowledge
- AI questions emphasized **practical application** over theory
- **Fair exam** despite controversy over the new catalog

## Exam Tips

### Learning Prioritization

1. **Highest Priority:** New 2025 topics (AI, 2FA, SMART, IPv6, Activity diagrams)
2. **High Priority:** Classic point earners (Utility analysis, Network plan, Subnetting)
3. **Medium Priority:** Standard topics (CIA triad, Backups, Protocols)
4. **Ignore:** Removed topics (SQL, RAID, Struktogramme, etc.)

### Time Savings

- **Don't learn removed topics** - it only costs time
- Focus on the **practical application** of AI
- Practice **calculations** (Subnetting, Utility analysis, Costs)
