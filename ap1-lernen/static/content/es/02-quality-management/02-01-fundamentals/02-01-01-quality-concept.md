# Concepto de Calidad

## Objetivos de Aprendizaje

Después de este capítulo usted debería ser capaz de:
- Definir el término calidad
- Distinguir entre calidad del producto y calidad del proceso
- Explicar el modelo Kano
- Comprender las características inherentes

---

## Contenido Principal

### ¿Qué es la Calidad?

**Calidad** se refiere al grado en que un conjunto de características inherentes de un objeto cumple con los requisitos (según DIN EN ISO 9000).

Dicho de forma más simple:
> Calidad = Grado de cumplimiento de los requisitos

**Característica inherente:** Una característica que es propia de un producto o servicio (ej., el color de un automóvil, el tiempo de respuesta de un software).

---

### Calidad del Producto vs. Calidad del Proceso

| Aspecto | Calidad del Producto | Calidad del Proceso |
|---------|---------------------|---------------------|
| **Definición** | Calidad del producto final | Calidad de los procesos de fabricación |
| **Enfoque** | Propiedades, funciones | Eficiencia, operación sin errores |
| **Ejemplo en TI** | Software sin errores | Proceso de desarrollo eficiente |
| **Medición** | Satisfacción del cliente, tasa de errores | Tiempo de producción, tasa de rechazos |

**Importante:** ¡Una buena calidad de proceso conduce a una buena calidad de producto!

---

### El Modelo Kano

El **Modelo Kano** (Profesor Noriaki Kano, Tokio 1978) muestra la relación entre las características de calidad realizadas y la satisfacción del cliente.

```
Satisfacción del Cliente
        ▲
completamente│                    ✱ Características de Entusiasmo
 satisfecho  │                 ✱
             │              ✱
             │           ✱      ╱ Características de Rendimiento
             │        ✱       ╱
             │     ✱        ╱
             │──────────────────────────────► Características
             │            ╱              de Calidad Realizadas
             │          ╱
             │        ╱ ─────── Características Básicas
             │      ╱
completamente│    ╱
insatisfecho
```

#### Las Tres Categorías de Características

| Característica | Descripción | Efecto |
|----------------|-------------|--------|
| **Características Básicas** | Propiedades evidentes por sí mismas | La ausencia lleva a fuerte insatisfacción, la presencia no se nota positivamente |
| **Características de Rendimiento** | Demandadas conscientemente por el cliente | Cuantas más, mayor satisfacción |
| **Características de Entusiasmo** | Extras sorprendentes | Generan alta satisfacción, pero no se esperan |

#### Ejemplo Práctico: Smartphone

| Característica | Tipo | Explicación |
|----------------|------|-------------|
| Función de teléfono | Básica | Esperada, imprescindible |
| Duración de batería | Rendimiento | Más = mejor |
| Característica innovadora | Entusiasmo | Sorprendentemente positiva |

---

### Dimensiones de la Calidad

La calidad puede verse en diferentes dimensiones:

| Dimensión | Descripción | Ejemplo en TI |
|-----------|-------------|---------------|
| **Funcionalidad** | Cumplimiento de las funciones requeridas | El software hace lo que debe |
| **Fiabilidad** | Rendimiento consistente a lo largo del tiempo | Tiempo de actividad del 99.9% |
| **Usabilidad** | Facilidad de uso | Interfaz de usuario intuitiva |
| **Eficiencia** | Consumo de recursos | Baja carga de CPU |
| **Mantenibilidad** | Facilidad de modificación/reparación | Código limpio y documentado |
| **Portabilidad** | Transferibilidad | Funciona en diferentes sistemas |

## Términos Clave

| Término | Explicación |
|---------|-------------|
| **Calidad** | Grado de cumplimiento de los requisitos |
| **Característica inherente** | Propiedad propia de un producto |
| **Calidad del producto** | Calidad del producto final |
| **Calidad del proceso** | Calidad de los procesos de fabricación |
| **Características básicas** | Propiedades evidentes, esperadas |
| **Características de rendimiento** | Propiedades demandadas conscientemente |
| **Características de entusiasmo** | Extras sorprendentes e inesperados |

## Ejercicios Prácticos

### Ejercicio 1
Defina el término calidad.

**Solución:** La calidad es el grado en que un conjunto de características inherentes de un objeto cumple con los requisitos.

### Ejercicio 2
Asigne las siguientes características de una tienda online a las categorías Kano:
a) La tienda es accesible
b) Tiempos de carga rápidos
c) Recomendaciones personalizadas de productos basadas en IA

**Solución:**
- a) Característica básica (esperada como evidente)
- b) Característica de rendimiento (cuanto más rápido, mayor satisfacción)
- c) Característica de entusiasmo (sorprendente, no esperada)

### Ejercicio 3
¿Cuál es la diferencia entre calidad del producto y calidad del proceso?

**Solución:**
- **Calidad del producto** se refiere a las características del producto terminado
- **Calidad del proceso** se refiere a la calidad de los procesos de fabricación y desarrollo
- Una buena calidad del proceso es un requisito previo para una buena calidad del producto
