# Principio SMART

## Objetivos de Aprendizaje

Después de este capítulo, usted debería ser capaz de:
- Explicar el principio SMART
- Conocer las cinco letras del acrónimo
- Formular objetivos según el principio SMART
- Reconocer y mejorar objetivos no SMART

---

## Contenido Principal

### ¿Qué es el Principio SMART?

El **principio SMART** es un método para formular objetivos claros y concretos. SMART es un acrónimo que representa cinco características que un buen objetivo debe tener.

> **NUEVO desde 2025:** ¡El principio SMART fue añadido recientemente al catálogo del examen!

---

### Los Cinco Criterios SMART

| Letra | Significado | Explicación |
|-------|-------------|-------------|
| **S** | **Específico** | El objetivo está formulado de manera clara e inequívoca |
| **M** | **Medible** | El éxito puede medirse objetivamente |
| **A** | **Alcanzable** (o Aceptado/Atractivo) | El objetivo es motivador y alcanzable |
| **R** | **Realista** | El objetivo es alcanzable bajo las condiciones dadas |
| **T** | **Temporal** | Existe un marco de tiempo concreto |

---

### Explicación de los Criterios

#### S - Específico
- Descripción precisa de lo que se debe lograr
- Sin formulaciones vagas
- Concreto en lugar de general

**No específico:** "Los sistemas de TI deberían mejorarse."
**Específico:** "El tiempo de respuesta del servidor debería ser menor a 200ms."

#### M - Medible
- Criterios cuantificables
- Objetivamente verificable
- Indicadores definidos

**No medible:** "La satisfacción del cliente debería aumentar."
**Medible:** "La satisfacción del cliente debería aumentar de 3.5 a 4.2 puntos según la encuesta."

#### A - Alcanzable/Aceptado
- El objetivo es aceptado por los involucrados
- Existe motivación para lograr el objetivo
- Formulación positiva

**No alcanzable:** "Cometer menos errores."
**Alcanzable:** "Lograr una tasa de error inferior al 1%."

#### R - Realista
- Alcanzable con los recursos disponibles
- Ni demasiado fácil ni imposible
- Considera las restricciones

**No realista:** "Renovar toda la red en una semana."
**Realista:** "Modernizar la red paso a paso durante seis meses."

#### T - Temporal
- Fecha de fin clara
- Posiblemente fechas intermedias (hitos)
- Marco de tiempo definido

**No temporal:** "En algún momento en el futuro."
**Temporal:** "Para el 31 de marzo de 2026."

---

### Ejemplos de Objetivos SMART

#### Ejemplo 1: Proyecto de TI
**No SMART:** "El nuevo software debería estar terminado pronto."

**SMART:** "El nuevo software de gestión de inventario debería estar completado para el 30 de junio de 2026, cumplir con todos los requisitos definidos en el documento de requisitos, y lograr al menos un 99.5% de disponibilidad en la operación de prueba."

#### Ejemplo 2: Objetivo de Formación
**No SMART:** "Quiero ser mejor programando."

**SMART:** "Para el examen AP1 del 5 de mayo de 2026, quiero ser capaz de convertir pseudocódigo en código Python funcional sin errores y resolver al menos tres ejercicios de práctica sin ayuda."

## Términos Clave

| Término | Explicación |
|---------|-------------|
| **SMART** | Acrónimo para formulación de objetivos: Específico, Medible, Alcanzable, Realista, Temporal |
| **Acuerdo de Objetivos** | Establecimiento de objetivos entre cliente y contratista |
| **Hito** | Fecha límite intermedia fija en el proyecto |
| **Indicador Clave de Rendimiento (KPI)** | Métrica para verificar el logro de objetivos |

## Consejos para el Examen

### Pregunta Típica del Examen

> "Un objetivo de proyecto debe desarrollarse según la regla SMART. ¿Cuáles de las siguientes características debe tener el objetivo?"
>
> [ ] específico [ ] espontáneo [ ] métrico
> [ ] especial [ ] medible [ ] temporal
> [ ] mediocre [ ] redundante [ ] aceptado
> [ ] realista [ ] con tiempo definido [ ] tradicional

**Solución:** específico, medible, aceptado (o alcanzable), realista, con tiempo definido

### Consejos Adicionales
- Conocer las cinco letras de memoria
- Conocer designaciones alternativas (A = Alcanzable O Aceptado O Atractivo)
- Ser capaz de distinguir ejemplos de objetivos SMART y no SMART

## Ejercicios Prácticos

### Ejercicio 1
¿Qué significa el acrónimo SMART?

**Solución:**
- S = Específico
- M = Medible
- A = Alcanzable (o Aceptado)
- R = Realista
- T = Temporal

### Ejercicio 2
Reformule el siguiente objetivo para que sea SMART:
"El sitio web debería ser más rápido."

**Posible Solución:**
"El tiempo de carga de la página de inicio debería reducirse de los 4 segundos actuales a menos de 2 segundos para el 31 de diciembre de 2025."

### Ejercicio 3
¿Qué criterios SMART faltan en el siguiente objetivo?
"La satisfacción del cliente debería mejorar."

**Solución:**
- **M falta:** No es medible (¿en cuánto?)
- **T falta:** No tiene tiempo definido (¿para cuándo?)
- Posiblemente también **S falta:** No es lo suficientemente específico (¿qué aspecto de la satisfacción?)
