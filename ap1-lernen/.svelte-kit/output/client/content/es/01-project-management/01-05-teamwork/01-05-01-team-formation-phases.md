# Fases de Formación del Equipo (Modelo de Tuckman)

## Objetivos de Aprendizaje

Después de este capítulo, usted debería ser capaz de:
- Conocer las cuatro fases de formación del equipo según Tuckman
- Describir las características de cada fase
- Comprender cómo se desarrollan los equipos
- Comprender el rol del liderazgo en las diferentes fases

---

## Contenido Principal

### El Modelo de Tuckman

El **Modelo de Tuckman** describe el desarrollo de los equipos en cuatro fases. Fue desarrollado en 1965 por Bruce Tuckman y es el modelo más conocido para el desarrollo de equipos.

---

### Las Cuatro Fases

```
+-------------+    +-------------+    +-------------+    +-------------+
|   FORMING   | -> |  STORMING   | -> |   NORMING   | -> | PERFORMING  |
|             |    |             |    |             |    |             |
| Conocerse   |    | Conflictos  |    |   Reglas    |    | Rendimiento |
|             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+
```

---

### Fase 1: Forming (Orientación)

**Característica:** Conocerse y orientación inicial

| Aspecto | Descripción |
|---------|-------------|
| **Comportamiento** | Cortés, cauteloso, expectante |
| **Enfoque** | Conocerse, comprender objetivos |
| **Sentimientos** | Incertidumbre, curiosidad, expectativa |
| **Productividad** | Baja |
| **Conflictos** | Se evitan |

**Tareas de Liderazgo:**
- Comunicar objetivos claros
- Proporcionar estructura
- Promover el conocimiento mutuo
- Ofrecer orientación

---

### Fase 2: Storming (Confrontación)

**Característica:** Primeros conflictos y disputas

| Aspecto | Descripción |
|---------|-------------|
| **Comportamiento** | Luchas de poder, resistencia, crítica |
| **Enfoque** | Aclarar roles, dinámicas de poder |
| **Sentimientos** | Frustración, decepción, competencia |
| **Productividad** | Baja a media |
| **Conflictos** | Ocurren abiertamente |

**Tareas de Liderazgo:**
- Moderar conflictos
- Introducir reglas claras
- Definir roles
- Tener paciencia

**Importante:** ¡Esta fase es normal y necesaria para el desarrollo del equipo!

---

### Fase 3: Norming (Organización)

**Característica:** Desarrollo de reglas y normas comunes

| Aspecto | Descripción |
|---------|-------------|
| **Comportamiento** | Cooperativo, dispuesto a comprometerse |
| **Enfoque** | Estándares comunes, métodos de trabajo |
| **Sentimientos** | Pertenencia, confianza |
| **Productividad** | Aumenta significativamente |
| **Conflictos** | Se resuelven constructivamente |

**Tareas de Liderazgo:**
- Apoyar las reglas del equipo
- Fomentar la delegación
- Fortalecer la responsabilidad personal
- Dar retroalimentación

---

### Fase 4: Performing (Rendimiento)

**Característica:** Colaboración efectiva y orientada a objetivos

| Aspecto | Descripción |
|---------|-------------|
| **Comportamiento** | Independiente, flexible, solidario |
| **Enfoque** | Completar tareas, mejora continua |
| **Sentimientos** | Orgullo, satisfacción, conexión |
| **Productividad** | Alta |
| **Conflictos** | Raros, rápidamente resueltos |

**Tareas de Liderazgo:**
- Delegar
- Asegurar las condiciones marco
- Dejar que el equipo celebre
- Apoyar el desarrollo

---

### Resumen de las Fases

| Fase | Indicadores | Enfoque del Equipo | Estilo de Liderazgo |
|------|-------------|-------------------|---------------------|
| **Forming** | Incertidumbre | Orientación | Directivo |
| **Storming** | Conflictos | Clarificación | Moderador |
| **Norming** | Reglas | Organización | Participativo |
| **Performing** | Rendimiento | Resultados | Delegativo |

---

### Dinámica en el Modelo

**Notas Importantes:**
- Las fases pueden durar tiempos diferentes
- Los equipos pueden retroceder a fases anteriores (ej., con nuevos miembros)
- No todos los equipos alcanzan la fase Performing
- Con Scrum: Cada Sprint puede contener elementos de todas las fases

```
                    ¿Nuevo miembro?
                    ¿Nueva tarea?
                          |
                          v
Forming -> Storming -> Norming -> Performing
   ^          |          |            |
   +----------+----------+------------+
              Regresión posible
```

## Términos Clave

| Término | Explicación |
|---------|-------------|
| **Forming** | Fase de orientación, conocerse |
| **Storming** | Fase de conflicto, luchas de poder |
| **Norming** | Fase de organización, desarrollo de reglas |
| **Performing** | Fase de rendimiento, colaboración efectiva |
| **Modelo de Tuckman** | Modelo de desarrollo de equipos en cuatro fases |

- Nombrar las cuatro fases de formación del equipo en el orden correcto
- Describir la fase Storming
- ¿En qué fase es más alta la productividad?

## Ejercicios Prácticos

### Ejercicio 1
Nombre las cuatro fases de formación del equipo según Tuckman en el orden correcto.

**Solución:**
1. Forming (Orientación)
2. Storming (Confrontación)
3. Norming (Organización)
4. Performing (Rendimiento)

### Ejercicio 2
¿En qué fase ocurren típicamente más conflictos?

**Solución:** En la fase Storming (Fase 2)

### Ejercicio 3
Un nuevo miembro del equipo se une. ¿Qué efecto puede tener esto en la dinámica del equipo?

**Solución:** El equipo puede retroceder a una fase anterior (generalmente Forming o Storming), ya que las dinámicas del grupo necesitan ser renegociadas.
