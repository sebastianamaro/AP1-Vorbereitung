## Contenido Principal

### ¿Qué es la Gestión de Proyectos?

La **gestión de proyectos** abarca la planificación, dirección y control de proyectos. Implica la aplicación de métodos, herramientas, técnicas y competencias en un proyecto (según DIN ISO 21500).

### Áreas de Gestión en la Gestión de Proyectos

La gestión de proyectos abarca o utiliza las siguientes áreas:

| Área | Descripción |
|------|-------------|
| **Gestión del Cronograma** | Planificación de tiempo y secuencia |
| **Gestión de Costos** | Planificación y control del presupuesto |
| **Gestión de Calidad** | Asegurar la calidad del resultado |
| **Gestión de Riesgos** | Identificación y manejo de riesgos |
| **Gestión de Comunicación** | Flujo de información en el proyecto |
| **Gestión de Recursos Humanos** | Composición y liderazgo del equipo |
| **Gestión de Adquisiciones** | Compra de recursos y servicios |
| **Gestión del Alcance** | Definición del alcance del proyecto |
| **Gestión de Integración** | Coordinación de todas las áreas |

---

### El Triángulo Mágico de la Gestión de Proyectos

Los tres factores centrales de un proyecto están en tensión entre sí:

```
           CALIDAD
              /\
             /  \
            /    \
           /      \
          /________\
     COSTO       TIEMPO
```

**Relación:** Si un factor cambia, afecta a los otros dos. Ejemplo: Menos tiempo a menudo conduce a mayores costos o menor calidad.
