# Entorno Empresarial

## Objetivos de Aprendizaje

Despues de esta seccion, podras:
- Distinguir los tres sectores economicos
- Nombrar las partes interesadas de una empresa
- Analizar el entorno empresarial
- Clasificar el rol de las empresas de TI en el sistema economico

### Empresas de TI en el Sistema Economico

| Sector | Ejemplos de TI |
|--------|----------------|
| **Primario** | TI en agricultura (Smart Farming) |
| **Secundario** | Fabricantes de hardware, produccion de chips |
| **Terciario** | Integradores de sistemas, proveedores de nube, consultoria TI |

La mayoria de las empresas de TI pertenecen al **sector terciario** (servicios).

### Partes Interesadas (Grupos de Interes)

**Las partes interesadas** son todas las personas y grupos que tienen interes en la empresa o se ven afectados por sus acciones.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PARTES INTERESADAS                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                         ┌───────────────┐                           │
│           Propietarios  │               │    Clientes               │
│         (Accionistas)   │   EMPRESA     │   (Compradores)           │
│                 ↘       │               │       ↙                   │
│                         │   ConSystem   │                           │
│    Empleados ──────────►│     GmbH      │◄──────── Proveedores      │
│                         │               │                           │
│                 ↗       │               │       ↖                   │
│            Gobierno     │               │    Bancos                 │
│         (Autoridades)   └───────────────┘  (Proveedores de          │
│                                 ↑           capital)                │
│                           Sociedad                                  │
│                          (Publico)                                  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Intereses de las Partes Interesadas

| Parte Interesada | Intereses |
|------------------|-----------|
| **Propietarios** | Beneficio, aumento de valor, exito empresarial |
| **Empleados** | Seguridad laboral, buen salario, desarrollo |
| **Clientes** | Calidad, precios justos, buen servicio |
| **Proveedores** | Seguridad de pago, relacion a largo plazo |
| **Bancos** | Intereses, reembolso, solvencia |
| **Gobierno** | Impuestos, empleos, cumplimiento de leyes |
| **Sociedad** | Proteccion ambiental, responsabilidad social |

## Terminos Clave

| Termino | Explicacion |
|---------|-------------|
| **Sector primario** | Extraccion de materias primas (agricultura, mineria) |
| **Sector secundario** | Industria manufacturera |
| **Sector terciario** | Servicios |
| **Parte interesada** | Todos los grupos de interes de una empresa |
| **Accionista** | Titular de capital/propietario |
| **Macroentorno** | Factores externos, no influenciables |
| **Microentorno** | Entorno cercano, parcialmente influenciable |

## Consejos para el Examen

1. **Clasificacion de sectores:**
   - Proveedor de servicios TI = Terciario (servicio)
   - Fabricante de hardware = Secundario (manufactura)

2. **Recordar partes interesadas:**
   - **P**ropietarios, **E**mpleados, **C**lientes, **P**roveedores, **B**ancos, **G**obierno
   - Mnemotecnico: "**P**ara **E**mpresas **C**omo **P**ymes **B**uscan **G**anancias"

3. **Preguntas tipicas de examen:**
   - Que partes interesadas tiene una empresa de TI?
   - Diferencia entre accionista/parte interesada?
   - Asignacion a sectores economicos

## Ejercicios Practicos

### Ejercicio 1: Clasificacion de Sectores

Asigna las siguientes empresas al sector correcto:

a) Un proveedor de servicios en la nube
b) Un fabricante de chips (Intel, AMD)
c) Un centro de datos que ofrece alojamiento de servidores
d) Una mina de cobre para produccion de cables

<details>
<summary>Mostrar Solucion</summary>

a) **Sector terciario** - La nube es un servicio
b) **Sector secundario** - Produccion/fabricacion de hardware
c) **Sector terciario** - El alojamiento es un servicio
d) **Sector primario** - Extraccion de materias primas

</details>

### Ejercicio 2: Analisis de Partes Interesadas

ConSystem GmbH esta planeando introducir un nuevo sistema ERP. Que partes interesadas se ven afectadas y cuales son sus intereses?

<details>
<summary>Mostrar Solucion</summary>

| Parte Interesada | Interes en el Sistema ERP |
|------------------|---------------------------|
| **Direccion** | Mayor eficiencia, reduccion de costos, mejores datos |
| **Empleados** | Facilidad de uso, formacion, sin trabajo extra |
| **Departamento TI** | Mantenibilidad, integracion, soporte |
| **Clientes** | Procesamiento de pedidos mas rapido |
| **Proveedores** | Mejores procesos de pedido |
| **Bancos** | ROI de la inversion |

</details>

### Ejercicio 3: Caso de Estudio

Se aprueba una nueva ley de proteccion de datos (modificacion del RGPD). A que parte del entorno empresarial pertenece esto y por que la empresa no puede influir directamente?

<details>
<summary>Mostrar Solucion</summary>

**Clasificacion:** Macroentorno (Politica/Ley)

**Justificacion:**
- Las leyes son aprobadas por organismos gubernamentales
- Las empresas individuales no pueden influir directamente en las leyes
- La empresa debe adaptarse a la ley (cumplimiento)

**Posible influencia indirecta:**
- A traves de asociaciones industriales (por ejemplo, BITKOM)
- Lobby
- Comentarios sobre propuestas legislativas

</details>
