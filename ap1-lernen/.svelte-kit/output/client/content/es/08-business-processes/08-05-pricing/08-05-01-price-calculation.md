# Cálculo de Precios

## Objetivos de Aprendizaje

Después de esta sección, serás capaz de:
- Realizar cálculos con recargos
- Calcular tarifas horarias de TI
- Comprender el cálculo comercial
- Explicar los componentes de costo de un precio

---

## Contenido Principal

### Principio Básico del Cálculo

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CÁLCULO DE PRECIOS                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  OBJETIVO: Determinar un precio de venta que cubra todos los costos│
│            y permita beneficio                                       │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │   Precio de costo                                            │    │
│  │   + Recargo de beneficio                                     │    │
│  │   ──────────────────                                         │    │
│  │   = Precio de venta al contado (neto)                        │    │
│  │   + IVA (19%)                                                │    │
│  │   ──────────────────                                         │    │
│  │   = Precio de venta bruto                                    │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Cálculo con Recargos (Cálculo Comercial)

```
┌─────────────────────────────────────────────────────────────────────┐
│           CÁLCULO PROGRESIVO (de compra a venta)                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│      Precio de lista de compra (neto)                               │
│    - Descuento del proveedor                                        │
│    ─────────────────────────────                                    │
│    = Precio objetivo de compra                                      │
│    - Descuento pronto pago del proveedor                            │
│    ─────────────────────────────                                    │
│    = Precio de compra al contado                                    │
│    + Costos de aprovisionamiento                                    │
│    ─────────────────────────────                                    │
│    = COSTO DE ADQUISICIÓN (precio de entrada)                       │
│    + Costos operativos (gastos generales)                           │
│    ─────────────────────────────                                    │
│    = PRECIO DE COSTO                                                │
│    + Recargo de beneficio                                           │
│    ─────────────────────────────                                    │
│    = Precio de venta al contado                                     │
│    + Descuento pronto pago del cliente                              │
│    ─────────────────────────────                                    │
│    = Precio objetivo de venta                                       │
│    + Descuento del cliente                                          │
│    ─────────────────────────────                                    │
│    = PRECIO DE LISTA DE VENTA (neto)                                │
│    + IVA (19%)                                                      │
│    ─────────────────────────────                                    │
│    = PRECIO DE VENTA BRUTO                                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Ejemplo de Cálculo: Precio de un Portátil

```

│  COMPRA:                                                            │
│  Precio de lista de compra (neto):         €800.00                  │

│  - Descuento pronto pago proveedor (2%):   -€14.40                  │
│  ─────────────────────────────────────────────────                  │
│  = Precio de compra al contado:            €705.60                  │
│  + Costos de aprovisionamiento (envío):    +€15.00                  │
│  ─────────────────────────────────────────────────                  │
│  = COSTO DE ADQUISICIÓN:                   €720.60                  │
│                                                                      │
│  RECARGO:                                                           │
│  + Costos operativos (25%):                +€180.15                 │
│  ─────────────────────────────────────────────────                  │
│  = PRECIO DE COSTO:                        €900.75                  │
│  + Beneficio (15%):                        +€135.11                 │
│  ─────────────────────────────────────────────────                  │
│  = Precio de venta al contado:             €1,035.86                │
│                                                                      │
│  VENTA:                                                             │
│  + Descuento pronto pago cliente (3%):     +€32.05                  │
│  ─────────────────────────────────────────────────                  │
│  = Precio objetivo de venta:               €1,067.91                │
│  + Descuento cliente (5%):                 +€56.21                  │
│  ─────────────────────────────────────────────────                  │
│  = PRECIO DE LISTA DE VENTA (neto):        €1,124.12                │
│  + IVA (19%):                              +€213.58                 │
│  ─────────────────────────────────────────────────                  │
│  = PRECIO DE VENTA BRUTO:                  €1,337.70                │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

│                   Horas productivas         1,220 h                 │
│                                                                      │
│     + Recargo gastos generales (80%):       +€39.40                 │
│     ─────────────────────────────────────────────                   │
│     = Precio de costo/hora:                 €88.65                  │

│     ─────────────────────────────────────────────                   │

│     + IVA (19%):                            +€20.21                 │
│     ─────────────────────────────────────────────                   │
│     = TARIFA HORARIA (bruto):               €126.59                 │
│                                                                      │

### Recargos Importantes

| Recargo | Descripción | Valor Típico |
|--------|-------------|---------------|
| **Costos operativos** | Gastos generales (alquiler, personal, etc.) | 20-40% |
| **Recargo de beneficio** | Beneficio objetivo | 10-20% |
| **Descuento pronto pago cliente** | Descuento por pago rápido | 2-3% |
| **Descuento cliente** | Descuento sobre precio de lista | 5-20% |
| **IVA** | Impuesto sobre el valor añadido | 19% (7% reducido) |

---

## Términos Clave

| Término | Explicación |
|------|-------------|
| **Costo de adquisición** | Precio de compra incluyendo costos de aprovisionamiento |
| **Precio de costo** | Todos los costos para un producto |
| **Costos operativos** | Costos generales del negocio |
| **Recargo de beneficio** | Recargo para el beneficio de la empresa |
| **Cálculo con recargos** | Cálculo mediante recargos porcentuales |
| **Horas productivas** | Horas facturadas al cliente |

---

## Consejos para el Examen

1. **Recuerda el esquema de cálculo:**
   - Costo de adquisición + Costos operativos = Precio de costo
   - Precio de costo + Beneficio = Precio de venta al contado
   - ... + Descuento + Pronto pago + IVA = Precio bruto

2. **Calcular descuento/pronto pago "hacia atrás":**
   - Para descuento/pronto pago: Calcular "en el cien"
   - 3% de descuento pronto pago significa: Precio venta contado × 100/97

3. **Tarifa horaria:**
   - ¡Las horas productivas son MENOS que las horas de trabajo!
   - No olvides el recargo de gastos generales

## Ejercicios Prácticos

### Ejercicio 1: Calcular Precio de Venta

ConSystem compra un monitor por €200 neto. Calcula el precio de venta bruto con:
- Costos de aprovisionamiento: €10
- Recargo de costos operativos: 30%
- Recargo de beneficio: 15%
- IVA: 19%

<details>
<summary>Mostrar solución</summary>

**Cálculo:**

Precio de compra neto: €200.00
+ Costos de aprovisionamiento: +€10.00
────────────────────────────
= Costo de adquisición: €210.00
+ Costos operativos (30%): +€63.00
────────────────────────────
= Precio de costo: €273.00
+ Beneficio (15%): +€40.95
────────────────────────────
= Precio de venta neto: €313.95
+ IVA (19%): +€59.65
────────────────────────────
= **Precio de venta bruto: €373.60**

</details>

### Ejercicio 2: Calcular Tarifa Horaria

Un consultor de TI tiene los siguientes datos:
- Salario bruto anual: €55,000
- Contribución seguridad social empleador: 21%
- Otros costos de personal: €3,000
- Días de trabajo/año: 220
- Horas productivas: 65% del tiempo de trabajo
- Recargo de gastos generales: 70%
- Recargo de beneficio: 25%

Calcula la tarifa horaria neta.

<details>
<summary>Mostrar solución</summary>

**1. Costos de personal:**
- Salario: €55,000
- Contrib. SS empleador: €55,000 × 21% = €11,550
- Otros: €3,000
- **Total: €69,550**

**2. Horas productivas:**
- Horas de trabajo: 220 días × 8h = 1,760 horas
- Productivas (65%): 1,760 × 0.65 = **1,144 horas**

**3. Tarifa horaria:**
- Costos personal/hora: €69,550 / 1,144 h = €60.79
- Gastos generales (70%): €60.79 × 0.70 = €42.55
- Precio de costo/hora: €60.79 + €42.55 = €103.34
- Beneficio (25%): €103.34 × 0.25 = €25.84
- **Tarifa horaria neta: €129.18**

</details>

### Ejercicio 3: Cálculo Regresivo

El precio de venta bruto de un servidor debe ser €2,380. ¿Cuál es el precio de compra máximo, si:
- Costos de aprovisionamiento: €50
- Recargo de costos operativos: 25%
- Recargo de beneficio: 20%
- IVA: 19%

<details>
<summary>Mostrar solución</summary>

**Cálculo regresivo:**

Precio de venta bruto: €2,380.00
÷ 1.19 (quitar IVA)
────────────────────────────
= Precio de venta neto: €2,000.00
÷ 1.20 (quitar beneficio)
────────────────────────────
= Precio de costo: €1,666.67
÷ 1.25 (quitar costos operativos)
────────────────────────────
= Costo de adquisición: €1,333.33
- Costos de aprovisionamiento: -€50.00
────────────────────────────
= **Precio de compra máximo: €1,283.33**

</details>
