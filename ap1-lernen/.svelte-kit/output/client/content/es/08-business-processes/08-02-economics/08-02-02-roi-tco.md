# ROI y TCO

## Objetivos de Aprendizaje

Despues de esta seccion, podras:
- Calcular e interpretar el Retorno sobre la Inversion (ROI)
- Determinar el Costo Total de Propiedad (TCO)
- Aplicar ambas metricas a inversiones de TI
- Evaluar la viabilidad economica de proyectos de TI

---

## Contenido Principal

### Retorno sobre la Inversion (ROI)

**ROI** mide el retorno de una inversion - cuanto beneficio se genera en relacion con el capital invertido.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ROI - RETORNO SOBRE LA INVERSION                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                         Beneficio                                    │
│         ROI  =  ─────────────────────  x  100%                       │
│                    Inversion                                         │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  O alternativamente:                                                 │
│                                                                      │
│               Retorno  -  Inversion                                  │
│         ROI  =  ─────────────────────────  x  100%                   │
│                      Inversion                                       │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  EJEMPLO:                                                           │
│                                                                      │
│  Inversion en nuevos servidores: $50,000                            │
│  Ahorros por eficiencia: $15,000/ano                                │
│  Periodo de observacion: 3 anos                                     │
│                                                                      │
│  Beneficio = ($15,000 x 3) - $50,000 = $45,000 - $50,000 = -$5,000  │
│                                                                      │
│  O: Retorno total = $45,000                                         │
│                                                                      │
│         $45,000 - $50,000     -$5,000                               │
│  ROI = ─────────────────────  = ────────── = -10%                   │
│              $50,000            $50,000                              │
│                                                                      │
│  → La inversion NO se amortiza en 3 anos todavia                    │
│                                                                      │
│  A 4 anos: ($15,000 x 4) - $50,000 = $10,000                        │
│  ROI = $10,000 / $50,000 = +20%                                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Interpretacion del ROI

| Valor ROI | Significado |
|-----------|-------------|
| **> 0%** | La inversion genera beneficio |
| **= 0%** | La inversion es neutral en costos |
| **< 0%** | La inversion genera perdida |
| **> 20%** | Buen retorno (depende de la industria) |

### Periodo de Amortizacion (Payback)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PERIODO DE AMORTIZACION                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                           Inversion                                  │
│  Periodo Amortizacion  =  ─────────────────────────                  │
│                          Ahorros Anuales                             │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  EJEMPLO:                                                           │
│                                                                      │
│  Inversion: $50,000                                                 │
│  Ahorros anuales: $15,000                                           │
│                                                                      │
│                           $50,000                                    │
│  Periodo Amortizacion = ───────── = 3.33 anos                       │
│                           $15,000                                    │
│                                                                      │
│  → Despues de 3 anos y 4 meses, la inversion se ha amortizado       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Costo Total de Propiedad (TCO)

**TCO** captura TODOS los costos que surgen durante toda la vida util de un producto o sistema.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TCO - COSTO TOTAL DE PROPIEDAD                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  TCO = Costos de Adquisicion + Costos Operativos + Costos Disposic. │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │     ADQUISICION          OPERACION           FIN             │    │
│  │     ───────────          ─────────           ───             │    │
│  │                                                              │    │
│  │     • Precio compra      • Mantenimiento     • Eliminacion   │    │
│  │     • Instalacion        • Actualizaciones   • Borrado datos │    │
│  │     • Formacion          • Soporte           • Desmontaje    │    │
│  │     • Configuracion      • Energia           • Migracion     │    │
│  │     • Migracion datos    • Consumibles                       │    │
│  │                          • Reparaciones                      │    │
│  │                          • Licencias                         │    │
│  │                                                              │    │
│  │  ←────────────────── Vida Util ─────────────────────→        │    │
│  │         (tipicamente 3-5 anos para hardware TI)              │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  IMPORTANTE: El precio de compra suele ser solo el 20-30% del TCO!  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Ejemplo TCO: Estacion de Trabajo PC

```
┌─────────────────────────────────────────────────────────────────────┐
│              EJEMPLO TCO: ESTACION DE TRABAJO PC (5 anos)           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  COSTOS DE ADQUISICION (unicos):                                    │
│  ├── Hardware (PC, monitor, perifericos)        $1,200              │
│  ├── Licencias de software                        $300              │
│  ├── Instalacion y configuracion                  $150              │
│  └── Formacion del empleado                       $200              │
│                                          ──────────                 │
│                               Total:           $1,850               │
│                                                                      │
│  COSTOS OPERATIVOS (por ano x 5 anos):                              │
│  ├── Mantenimiento y soporte                      $300              │
│  ├── Actualizaciones de software                  $100              │
│  ├── Costos de energia                             $80              │
│  ├── Consumibles                                   $50              │
│  └── Costos de personal TI asignados              $400              │
│                                          ──────────                 │
│                        Por ano:                $930                 │
│                        x 5 anos:             $4,650                 │
│                                                                      │
│  COSTOS DE DISPOSICION:                                             │
│  ├── Eliminacion de datos                          $50              │
│  └── Eliminacion/reciclaje                         $30              │
│                                          ──────────                 │
│                               Total:             $80                │
│                                                                      │
│  ════════════════════════════════════════════════════════════════   │
│                                                                      │
│  COSTO TOTAL DE PROPIEDAD:                                          │
│  Adquisicion:          $1,850                                       │
│  Operacion (5 anos):   $4,650                                       │
│  Disposicion:             $80                                       │
│                   ──────────                                        │
│  TCO TOTAL:            $6,580                                       │
│                                                                      │
│  → La adquisicion es solo el 28% del TCO!                           │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### ROI vs. TCO

| Metrica | Pregunta | Calcula |
|---------|----------|---------|
| **ROI** | ¿Vale la pena la inversion? | Retorno / Beneficio |
| **TCO** | ¿Cuanto cuesta en total? | Todos los costos |

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ROI vs. TCO                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ROI responde:                      TCO responde:                   │
│  "¿Que tan rentable es              "¿Cuanto cuesta                 │
│   la inversion?"                      en total?"                    │
│                                                                      │
│  Usado para:                         Usado para:                    │
│  → Decision de inversion             → Planificacion presupuestaria │
│  → Evaluacion de proyectos           → Comparacion de alternativas  │
│  → Justificacion economica           → Prevision de costos          │
│                                                                      │
│  TCO puede fluir HACIA el ROI:                                      │
│                                                                      │
│         Ahorros de nueva solucion - TCO de nueva solucion           │
│  ROI  = ──────────────────────────────────────────────── x 100%     │
│                      TCO de nueva solucion                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

---

## Terminos Clave

| Termino | Explicacion |
|---------|-------------|
| **ROI** | Retorno sobre la Inversion - Retorno de una inversion |
| **TCO** | Costo Total de Propiedad - Costos operativos totales |
| **Amortizacion** | Periodo hasta la recuperacion de costos |
| **Payback** | Igual que periodo de amortizacion |
| **Break-Even** | Punto en el que se logra beneficio |
| **Vida util** | Periodo de uso planificado (ej., 5 anos) |

---

## Consejos para el Examen

1. **Recordar formula ROI:**
   - ROI = Beneficio / Inversion x 100%
   - ROI positivo = La inversion vale la pena

2. **No olvidar el TCO:**
   - El precio de compra NO es todo!
   - Los costos operativos suelen ser mayores que la adquisicion

3. **Tareas tipicas de examen:**
   - Calcular ROI
   - Nombrar componentes del TCO
   - Determinar periodo de amortizacion

4. **Error comun:**
   - Para TCO, multiplicar costos operativos x vida util!

## Ejercicios Practicos

### Ejercicio 1: Calculo de ROI

ConSystem GmbH invierte $30,000 en un nuevo sistema de tickets. A traves de procesos mas eficientes, se ahorran $8,000 por ano.

a) Calcula el ROI despues de 3 anos.
b) Calcula el periodo de amortizacion.
c) ¿Vale la pena la inversion?

<details>
<summary>Mostrar Solucion</summary>

a) **ROI despues de 3 anos:**
- Ahorros: $8,000 x 3 = $24,000
- Beneficio: $24,000 - $30,000 = -$6,000
- ROI = -$6,000 / $30,000 x 100% = **-20%**

b) **Periodo de amortizacion:**
- $30,000 / $8,000 = **3.75 anos**

c) **Evaluacion:**
- Despues de 3 anos: ROI negativo → todavia no vale la pena
- Despues de 4 anos: ($8,000 x 4 - $30,000) / $30,000 = $2,000 / $30,000 = **+6.7%**
- **A partir del 4to ano, la inversion vale la pena.**

</details>

### Ejercicio 2: Comparacion de TCO

Compara dos alternativas para un sistema ERP:

| Elemento | Solucion A (On-Premise) | Solucion B (Nube) |
|----------|-------------------------|-------------------|
| Adquisicion | $50,000 | $0 |
| Instalacion | $10,000 | $5,000 |
| Licencia anual | $5,000 | $15,000 |
| Mantenimiento/ano | $8,000 | $0 (incluido) |
| Hardware/ano | $2,000 | $0 |

Vida util: 5 anos

¿Cual solucion tiene menor TCO?

<details>
<summary>Mostrar Solucion</summary>

**Solucion A (On-Premise):**
- Adquisicion: $50,000
- Instalacion: $10,000
- Licencias (5 anos): $5,000 x 5 = $25,000
- Mantenimiento (5 anos): $8,000 x 5 = $40,000
- Hardware (5 anos): $2,000 x 5 = $10,000
- **TCO Solucion A: $135,000**

**Solucion B (Nube):**
- Adquisicion: $0
- Instalacion: $5,000
- Licencias (5 anos): $15,000 x 5 = $75,000
- Mantenimiento: $0 (incluido)
- Hardware: $0
- **TCO Solucion B: $80,000**

**Resultado: La Solucion B (Nube) es $55,000 mas economica en 5 anos.**

</details>

### Ejercicio 3: Calculo Combinado

Una empresa esta considerando reemplazar su antigua infraestructura de servidores (TCO: $80,000/ano) con una solucion en la nube (TCO: $50,000/ano). La migracion cuesta $60,000 una sola vez.

a) ¿Cual es el ahorro anual?
b) ¿Despues de cuantos anos se amortiza la migracion?
c) ¿Cual es el ROI despues de 5 anos?

<details>
<summary>Mostrar Solucion</summary>

a) **Ahorro anual:**
$80,000 - $50,000 = **$30,000/ano**

b) **Periodo de amortizacion:**
$60,000 / $30,000 = **2 anos**

c) **ROI despues de 5 anos:**
- Ahorro total: $30,000 x 5 = $150,000
- Beneficio: $150,000 - $60,000 = $90,000
- ROI = $90,000 / $60,000 x 100% = **150%**

**Conclusion:** Muy buena inversion - alto ROI, amortizacion rapida.

</details>
