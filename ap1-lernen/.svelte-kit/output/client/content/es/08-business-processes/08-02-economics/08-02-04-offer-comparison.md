# Comparación de Ofertas

## Objetivos de Aprendizaje

Después de esta sección, serás capaz de:
- Aplicar métodos de comparación cuantitativos y cualitativos
- Calcular el precio de adquisición (costo total)
- Evaluar ofertas sistemáticamente
- Determinar la mejor alternativa de aprovisionamiento

### Cálculo del Precio de Adquisición

El **precio de adquisición** (también llamado costo total o costo de compra) es el precio real que una empresa paga por los bienes - incluyendo todas las deducciones y adiciones.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FÓRMULA DEL PRECIO DE ADQUISICIÓN                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│      Precio de lista de compra (neto)                               │
│    - Descuento del proveedor                                        │
│    ─────────────────────────────                                    │
│    = Precio objetivo de compra                                      │
│    - Descuento por pronto pago del proveedor                        │
│    ─────────────────────────────                                    │
│    = Precio neto de compra                                          │
│    + Costos de aprovisionamiento                                    │
│      (Embalaje, transporte, seguro, aduanas)                        │
│    ─────────────────────────────                                    │
│    = PRECIO DE ADQUISICIÓN (Costo total)                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Ejemplo de Cálculo

```
┌─────────────────────────────────────────────────────────────────────┐
│          CÁLCULO DEL PRECIO DE ADQUISICIÓN - EJEMPLO                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ConSystem GmbH ordena 10 portátiles:                               │
│                                                                      │
│  Precio de lista de compra (neto): $1,000.00 x 10 = $10,000.00     │
│  - Descuento por volumen 15%:                     -  $1,500.00      │
│  ─────────────────────────────────────────────────────────────      │
│  = Precio objetivo de compra:                        $8,500.00      │
│  - Descuento pronto pago 2% (pago en 10 días):    -   $170.00       │
│  ─────────────────────────────────────────────────────────────      │
│  = Precio neto de compra:                           $8,330.00       │
│  + Costos de envío:                               +   $150.00       │
│  + Embalaje:                                      +    $50.00       │
│  ─────────────────────────────────────────────────────────────      │
│  = PRECIO DE ADQUISICIÓN:                           $8,530.00       │
│                                                                      │
│  Precio de adquisición por portátil: $8,530 / 10 = $853.00         │
│                                                                      │
│  Comparación:                                                       │
│  Precio de lista:     $1,000 por portátil                          │
│  Precio adquisición:    $853 por portátil                          │
│  Ahorro:                $147 por portátil (14.7%)                  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

```
│  ──────────────────────┼────────────┼────────────┼────────────      │
│  = PRECIO ADQUISICIÓN  │   $8,530   │ $8,443.50  │   $8,400         │
│  ──────────────────────┼────────────┼────────────┼────────────      │
│  Precio adquis./unidad │     $853   │   $844.35  │     $840         │
│                                                                      │
│  RESULTADO CUANTITATIVO: El Proveedor C es el más económico        │
│                                                                      │

```

### Terminología de Condiciones de Pago

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CONDICIONES DE PAGO                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  DESCUENTO                                                          │
│  → Reducción de precio en la compra                                 │
│  → Ej., descuento por volumen, descuento por fidelidad, especial   │
│  → Se deduce del precio de lista                                    │
│                                                                      │
│  DESCUENTO POR PRONTO PAGO (SKONTO)                                 │
│  → Reducción de precio por pago rápido                              │
│  → Ej., "2% descuento por pago en 10 días"                         │
│  → Se deduce del precio objetivo de compra                          │
│                                                                      │
│  PLAZO DE PAGO                                                      │
│  → Período hasta que el pago vence                                  │
│  → Ej., "Neto 30" = pago dentro de 30 días sin deducción           │
│                                                                      │
│  Redacción típica:                                                  │
│  "2% descuento por pago en 10 días, neto 30 días"                  │
│  → Dentro de 10 días: 2% descuento por pronto pago                 │
│  → Dentro de 30 días: Sin descuento, pero sin recargo              │
│  → Después de 30 días: El pago está vencido                        │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Comparación Cualitativa

| Criterio | Proveedor A | Proveedor B | Proveedor C |
|-----------|------------|------------|------------|
| Precio de adquisición | $8,530 | $8,443.50 | $8,400 |
| Tiempo de entrega | 3 días | 7 días | 14 días |
| Garantía | 3 años | 2 años | 5 años |
| Soporte | Teléfono + In situ | Solo teléfono | Teléfono + Chat |
| Política de devolución | 14 días | 30 días | 14 días |

**Conclusión:** El proveedor más barato (C) tiene el tiempo de entrega más largo. ¡La decisión depende de las prioridades!

## Términos Clave

| Término | Explicación |
|------|-------------|
| **Precio de lista de compra** | Precio original sin deducciones |
| **Descuento** | Reducción de precio (ej., descuento por volumen) |
| **Precio objetivo de compra** | Precio de lista después del descuento |
| **Descuento por pronto pago** | Reducción de precio por pago rápido |
| **Precio neto de compra** | Precio después del descuento por pronto pago |
| **Costos de aprovisionamiento** | Transporte, embalaje, seguro |
| **Precio de adquisición** | Costo real de adquisición (costo total) |
| **Plazo de pago** | Período hasta que el pago vence |

---

## Consejos para el Examen

1. **Aprende la fórmula de memoria:**
   - Precio de lista - Descuento = Precio objetivo de compra
   - Precio objetivo de compra - Descuento pronto pago = Precio neto de compra
   - Precio neto de compra + Costos de aprovisionamiento = **Precio de adquisición**

2. **Nota el orden:**
   - ¡Primero descuento, luego descuento por pronto pago!
   - ¡El descuento por pronto pago se calcula sobre el PRECIO OBJETIVO DE COMPRA (no sobre el precio de lista)!

3. **Errores típicos del examen:**
   - Calcular el descuento por pronto pago sobre el precio de lista (¡INCORRECTO!)
   - Olvidar los costos de aprovisionamiento
   - Cálculo de porcentaje erróneo (15% de 1000 = 150, ¡no 850!)

4. **Formato del examen:**
   - A menudo: Completar tabla con espacios en blanco
   - O: Determinar el proveedor más económico

## Ejercicios Prácticos

### Ejercicio 1: Cálculo del Precio de Adquisición

ConSystem GmbH recibe la siguiente oferta por 20 monitores:

- Precio de lista: $300 por unidad
- Descuento por volumen: 10%
- Descuento por pronto pago: 3% por pago en 14 días
- Costos de envío: $80 fijo
- Embalaje: $2 por unidad

Calcula el precio de adquisición y el precio unitario.

<details>
<summary>Mostrar Solución</summary>

**Cálculo:**

Precio de lista de compra: $300 x 20 = **$6,000.00**
- Descuento 10%: **- $600.00**
───────────────────────────────────────
= Precio objetivo de compra: **$5,400.00**
- Descuento pronto pago 3%: **- $162.00**
───────────────────────────────────────
= Precio neto de compra: **$5,238.00**
+ Costos de envío: **+ $80.00**
+ Embalaje (20 x $2): **+ $40.00**
───────────────────────────────────────
= **Precio de adquisición: $5,358.00**

**Precio unitario:** $5,358 / 20 = **$267.90**

</details>

### Ejercicio 2: Comparación de Ofertas

Compara dos ofertas por 50 teclados:

| Elemento | Proveedor X | Proveedor Y |
|------|------------|------------|
| Precio lista/unidad | $45 | $42 |
| Descuento | 20% | 15% |
| Descuento pronto pago | 2% | 3% |
| Costos de envío | $50 | gratis |

¿Cuál proveedor es más económico?

<details>
<summary>Mostrar Solución</summary>

**Proveedor X:**
- Precio de lista: $45 x 50 = $2,250
- Descuento 20%: - $450
- Precio objetivo de compra: $1,800
- Descuento pronto pago 2%: - $36
- Precio neto de compra: $1,764
- Envío: + $50
- **Precio de adquisición X: $1,814**

**Proveedor Y:**
- Precio de lista: $42 x 50 = $2,100
- Descuento 15%: - $315
- Precio objetivo de compra: $1,785
- Descuento pronto pago 3%: - $53.55
- Precio neto de compra: $1,731.45
- Envío: $0
- **Precio de adquisición Y: $1,731.45**

**Resultado: El Proveedor Y es $82.55 más económico.**

(Aunque el descuento es menor, el mayor descuento por pronto pago y el envío gratuito marcan la diferencia.)

</details>

### Ejercicio 3: Evaluación Combinada

ConSystem GmbH ha recibido las siguientes ofertas para servidores:

| Criterio | Proveedor 1 | Proveedor 2 |
|-----------|------------|------------|
| Precio de adquisición | $12,500 | $11,800 |
| Tiempo de entrega | 5 días | 21 días |
| Garantía | 5 años | 3 años |
| Soporte | 24/7 In situ | Teléfono días laborables |

El servidor se necesita urgentemente (el proyecto comienza en 10 días). En caso de falla, surgen altos costos.

a) ¿Cuál proveedor es cuantitativamente más económico?
b) ¿Qué factores cualitativos apoyan al Proveedor 1?
c) ¿Cuál es tu recomendación?

<details>
<summary>Mostrar Solución</summary>

a) **Cuantitativamente:** El Proveedor 2 es $700 más económico ($11,800 vs. $12,500)

b) **Ventajas cualitativas del Proveedor 1:**
- Tiempo de entrega 5 días (¡el proyecto puede comenzar!)
- Garantía de 5 años (2 años más)
- Soporte 24/7 in situ (crítico para falla del servidor)

c) **Recomendación: Proveedor 1**

Razonamiento:
- El Proveedor 2 no puede entregar a tiempo (21 días > 10 días hasta inicio del proyecto)
- El costo adicional de $700 es pequeño comparado con los costos del proyecto por retraso
- Mejor soporte reduce riesgos de tiempo de inactividad
- La garantía más larga ahorra costos a largo plazo

El precio más bajo del Proveedor 2 no puede compensar las desventajas.

</details>
