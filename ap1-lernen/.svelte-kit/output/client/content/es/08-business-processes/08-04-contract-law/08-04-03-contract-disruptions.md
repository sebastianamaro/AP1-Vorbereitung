# Perturbaciones Contractuales y Derechos de Garantía

## Objetivos de Aprendizaje

Después de esta sección, serás capaz de:
- Distinguir tipos de perturbaciones contractuales
- Explicar los derechos de garantía del comprador
- Nombrar la secuencia de los derechos de garantía
- Distinguir entre garantía legal y garantía comercial

---

## Contenido Principal

### Tipos de Perturbaciones Contractuales

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PERTURBACIONES CONTRACTUALES                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. ENTREGA DEFECTUOSA (Prestación no conforme)            │    │
│  │     → Los bienes tienen defectos                            │    │
│  │     → Ejemplo: El portátil no arranca                       │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  2. ENTREGA INCORRECTA                                      │    │
│  │     → Se entregan bienes incorrectos                        │    │
│  │     → Ejemplo: Servidor en lugar de portátil                │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  3. ENTREGA INCOMPLETA                                      │    │
│  │     → Se entrega menos de lo acordado                       │    │
│  │     → Ejemplo: 8 en lugar de 10 portátiles                  │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  4. RETRASO EN LA ENTREGA                                   │    │
│  │     → Entrega tardía                                        │    │
│  │     → Ejemplo: Entrega 2 semanas después del plazo          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  5. RETRASO EN EL PAGO                                      │    │
│  │     → El comprador no paga a tiempo                         │    │
│  │     → Ejemplo: Factura no pagada después del vencimiento    │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Tipos de Defectos

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TIPOS DE DEFECTOS                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  DEFECTO MATERIAL (§ 434 BGB)                                       │
│  → El artículo no cumple con la especificación acordada            │
│                                                                      │
│  Existe un defecto material cuando:                                 │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. Falta la especificación acordada                        │    │
│  │     → Ej. "16 GB RAM" acordado, solo 8 GB instalados        │    │
│  │                                                              │    │
│  │  2. Uso normal no es posible                                 │    │
│  │     → Ej. El PC no arranca                                   │    │
│  │                                                              │    │
│  │  3. La especificación difiere de la publicidad               │    │
│  │     → Ej. "10 horas de batería" prometidas, solo 5 horas    │    │
│  │                                                              │    │
│  │  4. Montaje defectuoso (cuando el montaje fue acordado)      │    │
│  │     → Ej. Servidor instalado incorrectamente                 │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  DEFECTO JURÍDICO (§ 435 BGB)                                       │
│  → Terceros pueden reclamar derechos sobre el artículo             │
│  → Ej. software sin licencia válida, bienes robados               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Derechos de Garantía del Comprador (§ 437 BGB)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DERECHOS DE GARANTÍA                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  En caso de defecto, el comprador tiene los siguientes derechos:   │
│                                                                      │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  1. CUMPLIMIENTO POSTERIOR (§ 439 BGB)  ← ¡PRIORIDAD!        │   │
│  │     a) Reparación (subsanación)                               │   │
│  │        O                                                       │   │
│  │     b) Entrega de sustitución (bienes nuevos)                 │   │
│  │                                                               │   │
│  │     → El comprador tiene la ELECCIÓN                          │   │
│  │     → El vendedor solo puede rechazar si es desproporcionado  │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│                      │ Solo si el cumplimiento posterior             │
│                      │ FALLA (2 intentos)                            │
│                      │ o es RECHAZADO                                │
│                      ▼                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  2. DESISTIMIENTO (§ 323 BGB)                                 │   │
│  │     → Revertir el contrato                                    │   │
│  │     → Devolver bienes, devolver dinero                        │   │
│  │     → ¡Solo para defectos significativos!                     │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  3. REDUCCIÓN DE PRECIO (§ 441 BGB)                           │   │
│  │     → Reducir precio de compra                                │   │
│  │     → Quedarse con bienes, pagar menos                        │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                      │                                               │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │  4. INDEMNIZACIÓN (§ 280 BGB)                                 │   │
│  │     → Si el vendedor tiene culpa                              │   │
│  │     → Compensación por daños consecuentes                     │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Secuencia de los Derechos de Garantía

```
┌─────────────────────────────────────────────────────────────────────┐
│          JERARQUÍA DE DERECHOS DE GARANTÍA                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  NIVEL 1: CUMPLIMIENTO POSTERIOR                                    │
│  ─────────────────────────────────────────────────────────────────  │
│  → ¡Siempre primero!                                                │
│  → El vendedor tiene derecho a subsanar                             │
│  → El comprador debe establecer plazo (usualmente 2 intentos)       │
│                                                                      │
│           │                                                         │
│           │ ¿Falló el cumplimiento posterior?                       │
│           │ ¿El vendedor rechaza?                                   │
│           │ ¿Irrazonable?                                           │
│           ▼                                                         │
│                                                                      │
│  NIVEL 2: DESISTIMIENTO o REDUCCIÓN DE PRECIO                       │
│  ─────────────────────────────────────────────────────────────────  │
│  → El comprador puede elegir                                        │
│  → Desistimiento solo para defecto significativo                    │
│  → Reducción de precio también para defecto menor                   │
│                                                                      │
│  ADICIONALMENTE: INDEMNIZACIÓN                                      │
│  ─────────────────────────────────────────────────────────────────  │
│  → Puede reclamarse adicionalmente                                  │
│  → Solo si el vendedor tiene culpa                                  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Garantía Legal vs. Garantía Comercial

```
┌─────────────────────────────────────────────────────────────────────┐
│           GARANTÍA LEGAL vs. GARANTÍA COMERCIAL                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  GARANTÍA LEGAL (Estatutaria)     GARANTÍA COMERCIAL (Voluntaria)   │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  Regulada legalmente              Voluntaria del fabricante         │
│  (§§ 434 y ss. BGB)                                                 │
│                                                                      │
│  24 meses desde la entrega        Establecida por el fabricante     │
│  (bienes nuevos, B2C)             (ej. 3 años, 5 años)              │
│                                                                      │
│  El vendedor es responsable       El fabricante es responsable      │
│                                   (o vendedor, según la garantía)   │
│                                                                      │
│  El defecto debe haber existido   El defecto ocurrió durante        │
│  en la entrega                    el período de garantía            │
│                                                                      │
│  La carga de la prueba cambia     A menudo gestión más simple       │
│  después de 12 meses al comprador (buena voluntad)                  │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  RECUERDA: ¡La garantía comercial NO reemplaza la garantía legal!  │
│            La garantía legal es un derecho, la comercial es extra  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Plazos

| Límite | Duración | Nota |
|-------|----------|------|
| **Período de garantía legal** | 24 meses | Para bienes nuevos B2C |
| **Período de garantía legal** | 12 meses | Para bienes usados B2C |
| **Período de garantía legal B2B** | 12 meses | Para clientes comerciales (puede acortarse) |
| **Inversión de carga de prueba** | 12 meses | Después de esto, el comprador debe probar |
| **Plazo de reclamación B2B** | Inmediatamente | ¡El comerciante debe reclamar inmediatamente! |

### Mora

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MORA                                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  MORA EN LA ENTREGA (El vendedor no entrega a tiempo)               │
│  ──────────────────────────────────────────────────────────────     │
│  Requisitos:                                                        │
│  1. Vencimiento (fecha de entrega superada)                         │
│  2. Recordatorio (o la fecha era específica del calendario)         │
│  3. Culpa del vendedor                                              │
│                                                                      │
│  Derechos del comprador:                                            │
│  → Indemnización además del cumplimiento                            │
│  → Después del plazo: Indemnización en lugar del cumplimiento       │
│  → Desistimiento del contrato                                       │
│                                                                      │
│  MORA EN EL PAGO (El comprador no paga a tiempo)                    │
│  ──────────────────────────────────────────────────────────────     │
│  Requisitos:                                                        │
│  1. Vencimiento (plazo de pago superado)                            │
│  2. Recordatorio (o 30 días después de recibir la factura)          │
│                                                                      │
│  Derechos del vendedor:                                             │
│  → Intereses de mora (5% sobre tipo base, B2B: 9%)                  │
│  → Costos de recordatorio                                           │
│  → Indemnización                                                    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Términos Clave

| Término | Explicación |
|------|-------------|
| **Defecto material** | Los bienes no cumplen con el acuerdo |
| **Defecto jurídico** | Terceros tienen derechos sobre el artículo |
| **Cumplimiento posterior** | Reparación o entrega de sustitución |
| **Desistimiento** | Revertir el contrato |
| **Reducción de precio** | Reducir el precio de compra |
| **Garantía legal** | Responsabilidad estatutaria por defectos |
| **Garantía comercial** | Servicio adicional voluntario |
| **Mora** | Retraso en el cumplimiento/pago |

---

## Consejos para el Examen

1. **Recuerda la secuencia:**
   - **C**umplimiento posterior primero → luego **D**esistimiento/**R**educción/**I**ndemnización

2. **Garantía legal ≠ Garantía comercial:**
   - Garantía legal = estatutaria
   - Garantía comercial = voluntaria

3. **Preguntas típicas del examen:**
   - ¿Qué derechos tiene el comprador en caso de defecto?
   - ¿Diferencia garantía legal/comercial?
   - ¿Qué es un defecto material?

## Ejercicios Prácticos

### Ejercicio 1: Aplicar Derechos de Garantía

ConSystem compró 10 portátiles a un distribuidor. El teclado no funciona en 3 portátiles.

a) ¿Hay un defecto?
b) ¿Qué derecho debe ejercer ConSystem primero?
c) ¿Qué pasa si la reparación falla dos veces?

<details>
<summary>Mostrar solución</summary>

a) **Sí, hay un defecto material.**
- Los portátiles no cumplen con el uso normal
- Un teclado debe funcionar

b) **Cumplimiento posterior (reparación o entrega de sustitución)**
- ConSystem debe solicitar al distribuidor reparar O entregar portátiles nuevos
- Establecer plazo (ej. 14 días)

c) **Después de 2 intentos fallidos:**
- ConSystem puede desistir del contrato de compra (para los 3 portátiles)
- O exigir reducción de precio
- O reclamar indemnización (ej. costo de dispositivos de reemplazo)

</details>

### Ejercicio 2: Garantía Legal vs. Comercial

Un cliente compra un PC de ConSystem. Después de 14 meses la fuente de alimentación falla.

a) ¿Puede el cliente reclamar garantía legal?
b) ¿Quién debe probar que el defecto existía en la entrega?
c) ¿Qué sería diferente si el fabricante da 3 años de garantía comercial?

<details>
<summary>Mostrar solución</summary>

a) **Sí, el período de garantía legal (24 meses) aún no ha expirado.**

b) **El CLIENTE debe probar.**
- La inversión de la carga de la prueba solo aplica por 12 meses
- Después de 14 meses, el cliente debe probar que la fuente de alimentación ya estaba defectuosa en la entrega (ej. por dictamen pericial)
- Esto es difícil en la práctica

c) **Con 3 años de garantía comercial del fabricante:**
- El cliente puede contactar al fabricante
- La garantía comercial es más fácil de gestionar (a menudo sin prueba)
- El fabricante intercambia/repara según sus términos de garantía
- Ventaja: No se requiere prueba complicada

</details>

### Ejercicio 3: Mora en la Entrega

ConSystem ordenó servidores para el 15 de enero. Hoy es 25 de enero y los servidores no han llegado. El distribuidor no responde a las consultas.

¿Qué pasos debe tomar ConSystem?

<details>
<summary>Mostrar solución</summary>

**Paso 1: Establecer la mora**
- La fecha de entrega era específica del calendario (15 de enero)
- Por lo tanto no es necesario recordatorio (§ 286 párr. 2 BGB)
- El distribuidor está en mora desde el 16 de enero

**Paso 2: Establecer plazo**
- Recordatorio escrito con plazo
- "Le solicitamos entregar los servidores antes del 1 de febrero. Tras expirar este plazo, desistiremos del contrato."

**Paso 3: Después de expirar el plazo**
- Desistir del contrato O
- Reclamar indemnización en lugar del cumplimiento O
- Compra sustitutiva de otro distribuidor + indemnización

**Paso 4: Documentar**
- Guardar toda la correspondencia
- Documentar costos adicionales por compra sustitutiva
- Calcular daños por retraso si aplica (el proyecto no puede comenzar)

</details>
