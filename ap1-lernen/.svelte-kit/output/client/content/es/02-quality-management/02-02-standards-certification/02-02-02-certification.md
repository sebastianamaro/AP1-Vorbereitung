# Certificación

## Objetivos de Aprendizaje

Después de este capítulo usted debería:
- Comprender el propósito de la certificación

- Saber qué es una auditoría

---

## Contenido Principal

### ¿Qué es la Certificación?

**Certificación** es la prueba formal de que una organización, un producto o una persona cumple con las normas establecidas. Es realizada por organismos de prueba independientes y externos.

**Beneficios de la certificación:**
- Prueba de capacidad de calidad ante los clientes
- Ventaja competitiva en el mercado
- Mejora sistemática de los procesos internos
- Cumplimiento de requisitos del cliente (ej., para licitaciones)
- Construcción de confianza con socios comerciales

---

### La Auditoría

Una **auditoría** es un procedimiento de examen sistemático y documentado para evaluar el cumplimiento de las normas.

#### Tipos de Auditorías

| Tipo de Auditoría | Descripción | Realizada por |
|-------------------|-------------|---------------|
| **Auditoría Interna** | Autoexamen por empleados propios | Interno a la organización |
| **Auditoría Externa** | Examen por organismo independiente | Organismo de certificación |
| **Auditoría de Proveedor** | Examen de proveedores | El cliente examina al proveedor |

#### El Auditor

Un **auditor** es una persona especialmente capacitada que realiza auditorías:

- Independiente y objetivo
- Técnicamente calificado
- Para auditorías externas: de un organismo acreditado

### Proceso de Certificación

```
┌─────────────────┐
│ 1. PREPARACIÓN  │
│ • Autoevaluación
│ • Documentación
│ • Auditorías internas
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 2. SOLICITUD    │
│ • Elegir organismo
│   de certificación
│ • Presentar solicitud
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 3. AUDITORÍA    │
│ • Revisión de
│   documentos
│ • Auditoría in situ
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 4. CERTIFICADO  │
│ • Si es exitoso:
│   certificado
│ • Validez:
│   normalmente 3 años
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 5. VIGILANCIA   │
│ • Auditorías de
│   vigilancia
│   anuales
└─────────────────┘
```

#### Las Fases en Detalle

| Fase | Actividades |
|------|-------------|
| **1. Preparación** | Construir SGC, crear documentación, realizar auditorías internas |
| **2. Solicitud** | Seleccionar organismo de certificación (ej., TÜV, DEKRA), presentar solicitud |
| **3. Auditoría** | Etapa 1: Revisión de documentos; Etapa 2: Auditoría in situ |
| **4. Certificado** | Si el resultado es positivo: certificado (típicamente válido por 3 años) |
| **5. Vigilancia** | Auditorías de vigilancia regulares (anualmente), recertificación después de 3 años |

### Certificaciones Importantes en TI

| Certificación | Área | Descripción |
|---------------|------|-------------|
| **ISO 9001** | Gestión de Calidad | Prueba de un SGC funcional |
| **ISO 27001** | Seguridad de la Información | Prueba de un SGSI |
| **Marcado CE** | Seguridad del Producto | Conformidad Europea |
| **ITIL** | Gestión de Servicios de TI | Mejores prácticas para servicios de TI |

---

### Acreditación vs. Certificación

| Término | Significado |
|---------|-------------|
| **Certificación** | Confirmación de que una organización/producto cumple las normas |
| **Acreditación** | Confirmación de que un organismo de certificación es competente para realizar certificaciones |

**Recuerde:** Un organismo acreditado certifica organizaciones.

---

## Términos Clave

| Término | Explicación |
|---------|-------------|
| **Certificación** | Prueba formal del cumplimiento de normas |
| **Auditoría** | Procedimiento de examen sistemático |
| **Auditor** | Examinador especialmente capacitado |
| **Organismo de Certificación** | Organización que realiza certificaciones |
| **Acreditación** | Reconocimiento de un organismo de certificación |
| **Auditoría de Vigilancia** | Examen regular para mantener el certificado |
| **Recertificación** | Nueva certificación completa después del vencimiento |

## Consejos para el Examen

### Preguntas Comunes del Examen
- ¿Cuál es el propósito de la certificación?
- Describa el proceso de certificación
- ¿Qué es una auditoría?
- ¿Qué tipos de auditorías existen?

### Importante Recordar
- Las certificaciones tienen **duración limitada** (normalmente 3 años)
- Las **auditorías de vigilancia** se realizan anualmente
- Solo los organismos **acreditados** pueden certificar

## Ejercicios Prácticos

### Ejercicio 1
¿Qué es la certificación y qué beneficio tiene para una organización?

**Solución:** La certificación es la prueba formal de que una organización cumple con las normas establecidas. Beneficios:
- Prueba de capacidad de calidad
- Ventaja competitiva
- Construcción de confianza con clientes y socios
- Requisito previo para ciertos contratos/licitaciones

### Ejercicio 2
Describa el proceso de certificación en cinco pasos.

**Solución:**
1. **Preparación:** Autoevaluación, documentación, auditorías internas
2. **Solicitud:** Elegir organismo de certificación, presentar solicitud
3. **Auditoría:** Revisión de documentos y auditoría in situ por auditores externos
4. **Certificado:** Emisión del certificado con resultado positivo
5. **Vigilancia:** Auditorías de vigilancia regulares para mantener la certificación

### Ejercicio 3
Distinga entre auditorías internas y externas.

**Solución:**
- **Auditoría interna:** Autoexamen por empleados propios capacitados para preparación y mejora
- **Auditoría externa:** Examen por organismo de certificación independiente para la certificación oficial
