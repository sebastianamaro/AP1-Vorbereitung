### CRM (Customer Relationship Management)

**CRM** es software para gestionar las relaciones con los clientes y optimizar los procesos de ventas, marketing y servicio.

### Funciones de CRM

| Area | Funciones |
|------|-----------|
| **Ventas** | Gestion de leads, oportunidades, creacion de ofertas |
| **Marketing** | Gestion de campanas, email marketing, segmentacion |
| **Servicio** | Sistema de tickets, FAQ, chat en vivo, gestion de quejas |
| **Analitica** | Dashboards, informes, pronosticos de ingresos |

| Ventaja | Descripcion |

### SCM (Supply Chain Management)

**SCM** se refiere a la planificacion y control de toda la cadena de suministro desde la materia prima hasta el cliente final.

```
┌────────────────────────────────────────────────────────────────┐
│                    CADENA DE SUMINISTRO                        │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   PROVEEDOR ──► COMPRAS ──► PRODUCCION ──► ALMACEN ──► CLIENTE│
│                                                                │
│   ┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐  ┌─────┐ │
│   │Proveedor│  │Compras │   │Fabrica-│   │Distri- │  │Cliente││
│   │materia │   │        │   │cion    │   │bucion  │  │final │ │
│   │prima   │   │        │   │        │   │        │  │      │ │
│   └────────┘   └────────┘   └────────┘   └────────┘  └─────┘ │
│       │            │            │            │           │    │
│       └────────────┴────────────┴────────────┴───────────┘    │
│                                │                              │
│                     SISTEMA SCM                               │
│                                                                │
│   Objetivo: Flujo optimo de bienes, informacion y dinero     │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

---

### Funciones de SCM

| Funcion | Descripcion |
|---------|-------------|
| **Planificacion de demanda** | Pronosticar demanda del cliente |
| **Compras** | Optimizacion de compras, gestion de proveedores |
| **Produccion** | Planificacion y control de produccion |
| **Almacenamiento** | Optimizacion de inventario |
| **Distribucion** | Logistica, planificacion de rutas |
| **Seguimiento** | Rastreo de envios |

---

### Ventajas de SCM

| Ventaja | Descripcion |
|---------|-------------|
| **Reduccion de costos** | Inventario optimizado, menos almacenamiento |
| **Entrega mas rapida** | Logistica eficiente |
| **Transparencia** | Vision general de toda la cadena |
| **Flexibilidad** | Respuesta rapida a cambios |
| **Calidad** | Trazabilidad |

```
┌────────────────────────────────────────────────────────────────┐
│              COMPARACION ERP, CRM, SCM                         │

│                                                                │
│                        EMPRESA                                 │
│                             │                                  │
│   ┌─────────────────────────┼─────────────────────────┐       │
│   │                         │                         │       │

│  ┌──────────┐          ┌──────────┐          ┌──────────┐    │
│  │Cadena de │          │Interno   │          │Clientes  │    │
│  │Suministro│          │          │          │          │    │
│  │Proveedores│         │Finanzas  │          │Ventas    │    │
```

| Sistema | Enfoque | Objetivo Principal |
|---------|---------|-------------------|
| **ERP** | Recursos internos | Eficiencia de todos los procesos de negocio |
| **CRM** | Clientes (aguas abajo) | Optimizar relaciones con clientes |
| **SCM** | Cadena de suministro (aguas arriba) | Optimizar cadena de suministro |

---

### Integracion de Sistemas

Las empresas modernas integran ERP, CRM y SCM para un flujo de informacion sin interrupciones.

| Interfaz | Ejemplo |
|----------|---------|
| **CRM ↔ ERP** | Pedido de cliente desde CRM dispara produccion en ERP |
| **SCM ↔ ERP** | Sugerencia de pedido en SCM se registra en ERP |
| **CRM ↔ SCM** | Estado de entrega desde SCM en CRM para consultas de clientes |

## Ejercicios Practicos

### Ejercicio 1
Que es CRM y cuales tres areas principales cubre?

**Solucion:** **CRM (Customer Relationship Management)** es software para gestionar las relaciones con los clientes.

Tres areas principales:
1. **Ventas:** Gestion de leads, ofertas, oportunidades de venta
2. **Marketing:** Campanas, email marketing, segmentacion de clientes
3. **Servicio:** Sistema de tickets, gestion de quejas, FAQ

### Ejercicio 2
Cual es la diferencia entre ERP y CRM?

**Solucion:**
| Aspecto | ERP | CRM |
|---------|-----|-----|
| **Enfoque** | Procesos internos de negocio | Relaciones con clientes |
| **Areas** | Finanzas, RRHH, produccion | Ventas, marketing, servicio |
| **Objetivo** | Eficiencia de planificacion de recursos | Satisfaccion del cliente e ingresos |

ERP gestiona operaciones internas, CRM se enfoca en el cliente.

### Ejercicio 3
Que significa SCM y cual es el objetivo?

**Solucion:** **SCM (Supply Chain Management)** se refiere a la planificacion y control de toda la cadena de suministro desde el proveedor de materia prima hasta el cliente final.

**Objetivo:** Optimizacion del flujo de bienes, informacion y dinero a lo largo de toda la cadena de suministro para:
- Menores costos (menos inventario)
- Tiempos de entrega mas rapidos
- Mejor disponibilidad
- Mayor transparencia
