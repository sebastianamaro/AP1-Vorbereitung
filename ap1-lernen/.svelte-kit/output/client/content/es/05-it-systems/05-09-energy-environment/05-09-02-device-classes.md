# Clases de Dispositivos y Etiquetas Energeticas

## Objetivos de Aprendizaje

Despues de este capitulo, deberias ser capaz de:
- Entender la etiqueta energetica de la UE
- Conocer las clases de eficiencia energetica
- Considerar la etiqueta en la adquisicion de dispositivos
- Conocer la eliminacion de equipos TI

---

## Contenido Principal

### La Etiqueta Energetica de la UE

La **etiqueta energetica de la UE** muestra la eficiencia energetica de los aparatos electricos de un vistazo.

```
┌─────────────────────────────────────────────────────────────────┐
│                    ETIQUETA ENERGETICA UE                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌───────────────────────────────────────┐                    │
│   │  ████████████████████  A  (verde osc.)│  ← Muy eficiente   │
│   │  █████████████████  B  (verde claro)  │                    │
│   │  ██████████████  C  (amarillo-verde)  │                    │
│   │  ███████████  D  (amarillo)           │                    │
│   │  ████████  E  (naranja)               │                    │
│   │  █████  F  (naranja oscuro)           │                    │
│   │  ██  G  (rojo)                        │  ← Menos eficiente │
│   └───────────────────────────────────────┘                    │
│                                                                 │
│   Informacion adicional:                                       │
│   • Consumo de energia en kWh/ano o kWh/1000h                  │
│   • Datos especificos del producto                             │
│   • Codigo QR a base de datos de productos                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

### Clases de Eficiencia Energetica (desde 2021)

Desde marzo 2021, se aplica la nueva etiqueta simplificada de **A a G** (sin clases plus).

| Clase | Calificacion |
|-------|--------------|
| **A** | Mejor eficiencia |
| **B** | Muy buena |
| **C** | Buena |
| **D** | Promedio |
| **E** | Por debajo del promedio |
| **F** | Pobre |
| **G** | Peor eficiencia |

**Nota:** A a menudo esta vacia inicialmente - espacio para mejoras futuras!

### Escala Antigua vs. Nueva

```
┌────────────────────────────────────────────────────────────────┐
│            CLASES ENERGETICAS ANTIGUA vs. NUEVA                │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   ANTIGUA (hasta 2021)          NUEVA (desde 2021)            │
│                                                                │
│   A+++  ─────────────────►   B o C                            │
│   A++   ─────────────────►   C o D                            │
│   A+    ─────────────────►   D o E                            │
│   A     ─────────────────►   E o F                            │
│   B     ─────────────────►   F o G                            │
│   C-G   ─────────────────►   G (o ya no permitido)            │
│                                                                │
│   La antigua clase A+++ YA NO ES la mejor!                    │
│   La clase A esta reservada para futuras innovaciones         │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Dispositivos con Etiqueta Energetica

| Categoria de Dispositivo | Datos Relevantes |
|--------------------------|------------------|
| **Monitores** | Consumo de energia en kWh/1000h, diagonal de pantalla |
| **Computadoras** | (Sin etiqueta UE, pero certificacion TCO) |
| **Refrigeradores** | kWh/ano, volumen |
| **Lavadoras** | kWh/100 ciclos, consumo de agua |
| **Televisores** | kWh/1000h |
| **Fuentes de luz** | kWh/1000h |

### Certificacion TCO para TI

Como las computadoras no tienen etiqueta energetica de la UE, la **certificacion TCO** es importante.

```
┌────────────────────────────────────────────────────────────────┐
│                 TCO CERTIFIED                                  │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   TCO Development (Suecia) certifica productos TI:            │
│                                                                │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  • Eficiencia energetica                               │  │
│   │  • Ergonomia (monitores, teclados)                     │  │
│   │  • Emisiones (electromagneticas, quimicas)             │  │
│   │  • Durabilidad y reparabilidad                         │  │
│   │  • Responsabilidad social en produccion                │  │
│   │  • Reciclabilidad                                      │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                │
│   Productos: Monitores, laptops, desktops, impresoras, servidores│
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

---

### Energy Star

**Energy Star** es un programa de eficiencia energetica, originalmente de EE.UU.

| Aspecto | Descripcion |
|---------|-------------|
| **Enfoque** | Consumo de energia en operacion y standby |
| **Productos** | Computadoras, monitores, impresoras, servidores |
| **Requisito** | Cumplir umbrales de energia |
| **Standby** | Max. 1-2 Watts |

### Eliminacion de Equipos TI (WEEE)

Los equipos TI son **residuos electronicos** y deben eliminarse correctamente.

```
┌────────────────────────────────────────────────────────────────┐
│              ELIMINACION DE EQUIPOS TI                         │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  [X]  NO en basura domestica!                          │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                │
│   Eliminacion correcta:                                       │
│                                                                │
│   ┌──────────────────┐  ┌──────────────────┐                 │
│   │ Punto de reciclaje│  │ Devolucion en   │                 │
│   │ (municipal)       │  │ comercio        │                 │
│   └──────────────────┘  └──────────────────┘                 │
│                                                                │
│   ┌──────────────────┐  ┌──────────────────┐                 │
│   │ Devolucion al    │  │ Donacion/        │                 │
│   │ fabricante       │  │ Reutilizacion    │                 │
│   └──────────────────┘  └──────────────────┘                 │
│                                                                │
│   Importante:                                                  │
│   • Borrar datos primero!                                      │
│   • Eliminar baterias por separado                             │
│   • Para hardware empresarial: Comprobante de eliminacion     │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

| Regulacion | Descripcion |
|------------|-------------|
| **WEEE** | Waste Electrical and Electronic Equipment (UE) |
| **Leyes nacionales** | Leyes de residuos electronicos en cada pais |

### Adquisicion de TI Eficiente Energeticamente

| Criterio | Buscar |
|----------|--------|
| **Etiqueta energetica** | Preferir clase A-C |
| **Certificacion TCO** | Para monitores y computadoras |
| **Energy Star** | Eficiencia en standby |
| **80 PLUS** | Para fuentes de alimentacion |
| **TDP** | Para procesadores (menor = mas eficiente) |
| **SSD en lugar de HDD** | Menor consumo de energia |

## Consejos para el Examen

### Preguntas Comunes del Examen

- Que muestra la etiqueta energetica de la UE?
- Que clases de eficiencia existen?
- Como se eliminan correctamente los dispositivos TI?
- Que es la certificacion TCO?

### Importante Recordar

- **Etiqueta UE:** A a G (A = mejor eficiencia)
- **Clase A** a menudo aun esta vacia (espacio para innovacion)
- **TCO** = certificacion para dispositivos TI
- **Residuos electronicos** = NO en la basura domestica!

## Ejercicios Practicos

### Ejercicio 1
Que clases de eficiencia energetica existen en la nueva etiqueta de la UE (desde 2021)?

**Solucion:** La nueva etiqueta energetica de la UE tiene clases **A a G**:
- A = Mejor eficiencia (a menudo aun vacia)
- B = Muy buena
- C = Buena
- D = Promedio
- E = Por debajo del promedio
- F = Pobre
- G = Peor eficiencia

Las antiguas clases A+++, A++, A+ fueron abolidas.

### Ejercicio 2
Una empresa quiere comprar nuevos monitores. Que certificaciones deben considerarse?

**Solucion:**
- **Etiqueta energetica UE:** Preferir clase A-C
- **Certificacion TCO:** Para eficiencia energetica, ergonomia y medio ambiente
- **Energy Star:** Para consumo eficiente en standby
- Adicionalmente: Datos sobre consumo de energia en kWh/1000h

### Ejercicio 3
Como deben eliminarse las computadoras y monitores viejos?

**Solucion:** Los equipos TI son **residuos electronicos** y NO deben ir a la basura domestica!

Opciones correctas de eliminacion:
1. **Punto de reciclaje:** Los puntos de recoleccion municipales aceptan residuos electronicos gratis
2. **Devolucion en comercio:** Los grandes comerciantes deben aceptar dispositivos
3. **Devolucion al fabricante:** Muchos fabricantes ofrecen programas de devolucion

**Importante:** Borrar datos de forma segura antes de la eliminacion!
