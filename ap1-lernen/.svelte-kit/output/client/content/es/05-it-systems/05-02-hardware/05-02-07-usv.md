## Contenido Principal

### ¿Qué es un SAI?

Un **SAI (Sistema de Alimentación Ininterrumpida)** protege los sistemas de TI contra fallos de energía y fluctuaciones de voltaje mediante baterías que toman el control cuando es necesario.

```
┌─────────────────────────────────────────────────────────────────┐
│                    FUNCIÓN DEL SAI                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Operación normal:                                             │
│                                                                 │
│   ┌────────┐      ┌──────────┐      ┌──────────┐              │
│   │  Red   │ ───► │   SAI    │ ───► │ Servidor │              │
│   │Eléctrica│      │(cargando)│      │          │              │
│   └────────┘      └──────────┘      └──────────┘              │
│                                                                 │
│   Durante fallo de energía:                                     │
│                                                                 │
│   ┌────────┐      ┌──────────┐      ┌──────────┐              │
│   │  Red   │  ✗   │   SAI    │ ───► │ Servidor │              │
│   │Eléctrica│      │(batería) │      │(continúa │              │
│   └────────┘      └──────────┘      │funcionando)│              │
│                                      └──────────┘              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

### Tareas de un SAI

| Tarea | Descripción |
|-------|-------------|
| **Puenteo** | Suministro de energía durante fallo de red |
| **Suavizado de Voltaje** | Compensación de fluctuaciones |
| **Protección contra Sobretensiones** | Protección contra picos de voltaje |
| **Apagado Seguro** | Tiempo para apagado ordenado |

### Tipos de SAI

```
┌────────────────────────────────────────────────────────────────┐
│                    TIPOS DE SAI                                │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  1. OFFLINE/STANDBY                                            │
│     ┌───────┐                                                  │
│     │  CA   │──────────────────────►  Carga                   │
│     └───────┘        ↑                                         │
│                      │ (solo durante fallo)                    │
│                 ┌────┴────┐                                    │
│                 │ Batería │                                    │
│                 └─────────┘                                    │
│     → Económico, tiempo de transferencia corto (5-12 ms)      │
│                                                                │
│  2. LINE-INTERACTIVE                                           │
│     ┌───────┐    ┌──────────┐                                 │
│     │  CA   │───►│ Regulador│────────►  Carga                 │
│     └───────┘    │ de Voltaje│                                 │
│                  └────┬─────┘                                  │
│                       │                                        │
│                  ┌────┴────┐                                   │
│                  │ Batería │                                   │
│                  └─────────┘                                   │
│     → Regulación de voltaje, transferencia rápida (2-4 ms)    │
│                                                                │
│  3. ONLINE/DOBLE CONVERSIÓN                                    │
│     ┌───────┐    ┌────────┐    ┌────────┐                     │
│     │  CA   │───►│Rectific.│───►│Inversor│───► Carga          │
│     └───────┘    └────┬────┘    └────────┘                     │
│                       │                                        │
│                  ┌────┴────┐                                   │
│                  │ Batería │ (siempre en línea)                │
│                  └─────────┘                                   │
│     → Mejor calidad, sin tiempo de transferencia (0 ms)       │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Comparación de Tipos de SAI

| Tipo | Tiempo de Transferencia | Protección | Eficiencia | Precio | Uso |
|------|-------------------------|------------|------------|--------|-----|
| **Offline/Standby** | 5-12 ms | Básica | Alta | Económico | PC doméstico |
| **Line-Interactive** | 2-4 ms | Media | Media | Medio | Oficina, Servidor |
| **Online/Doble Conversión** | 0 ms | Máxima | Menor | Alto | Centro de datos |

---

### Métricas Importantes

| Métrica | Descripción | Ejemplo |
|---------|-------------|---------|
| **VA (Volt-Ampere)** | Potencia aparente | 1500 VA |
| **Watt** | Potencia real (~60-70% de VA) | 900 W |
| **Tiempo de Autonomía** | Duración durante fallo de energía | 5-30 minutos |
| **Factor de Potencia** | Relación Watt/VA | 0,6 - 0,9 |

**Regla general:** Watt ≈ VA × 0,6 a 0,7

### Dimensionamiento

```
┌────────────────────────────────────────────────────────────────┐
│                DIMENSIONAMIENTO DE SAI                         │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  Paso 1: Determinar requisito de potencia                      │
│                                                                │
│    Servidor:          300 W                                    │
│    Monitor:            50 W                                    │
│    Switch:             20 W                                    │
│    ─────────────────────────                                   │
│    Total:             370 W                                    │
│                                                                │
│  Paso 2: Margen de seguridad (+20-30%)                        │
│                                                                │
│    370 W × 1,3 = 481 W                                        │
│                                                                │
│  Paso 3: Calcular valor VA                                     │
│                                                                │
│    481 W ÷ 0,6 = 802 VA → SAI con al menos 1000 VA           │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Perturbaciones de la Red Eléctrica

| Perturbación | Descripción | Protección SAI |
|--------------|-------------|----------------|
| **Fallo de Energía** | Fallo completo | Todos los tipos |
| **Caída de Voltaje** | Voltaje muy bajo | Line-Interactive, Online |
| **Sobrevoltaje** | Voltaje muy alto | Line-Interactive, Online |
| **Picos de Voltaje** | Picos cortos | Todos los tipos |
| **Fluctuación de Frecuencia** | Frecuencia de red inestable | Solo Online |
| **Ruido** | Interferencia eléctrica | Online |

---

### Casos de Uso

| Escenario | Tipo de SAI | Justificación |
|-----------|-------------|---------------|
| **PC Doméstico** | Offline/Standby | Económico, protección simple |
| **Estación de Trabajo de Oficina** | Line-Interactive | Buen compromiso |
| **Servidor Individual** | Line-Interactive | Transferencia rápida |
| **Sala de Servidores** | Online | Máxima protección |
| **Centro de Datos** | Online + Diésel | Máxima disponibilidad |

## Ejercicios Prácticos

### Ejercicio 1
Nombra los tres tipos de SAI y sus tiempos de transferencia.

**Solución:**
1. **Offline/Standby:** 5-12 ms tiempo de transferencia
2. **Line-Interactive:** 2-4 ms tiempo de transferencia
3. **Online/Doble Conversión:** 0 ms tiempo de transferencia

### Ejercicio 2
Un servidor requiere 400 Watts. ¿Qué capacidad de SAI (VA) se recomienda?

**Solución:**
1. Margen de seguridad: 400 W × 1,3 = 520 W
2. Cálculo VA: 520 W ÷ 0,6 = 867 VA
3. **Recomendación:** SAI con al menos **1000 VA**

### Ejercicio 3
¿Por qué un SAI Online es más adecuado para un centro de datos que un SAI Offline?

**Solución:**
- **Sin tiempo de transferencia:** SAI Online entrega sin interrupción (0 ms), SAI Offline tiene 5-12 ms de tiempo de transferencia
- **Voltaje constante:** El convertidor doble siempre produce energía limpia
- **Protección completa:** Protege contra todos los tipos de perturbaciones (incluyendo fluctuaciones de frecuencia, ruido)
- **Sistemas críticos:** Los centros de datos requieren la máxima disponibilidad
