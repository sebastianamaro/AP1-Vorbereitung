# USV (Unterbrechungsfreie Stromversorgung)

## Lernziele

Nach diesem Kapitel solltest du:
- Den Zweck einer USV erklären können
- Die verschiedenen USV-Typen unterscheiden können
- Wichtige Kennzahlen verstehen
- Einsatzszenarien beschreiben können

---

## Kerninhalt

### Was ist eine USV?

Eine **USV (Unterbrechungsfreie Stromversorgung)** schützt IT-Systeme vor Stromausfällen und Spannungsschwankungen durch Batterien, die bei Bedarf einspringen.

```
┌─────────────────────────────────────────────────────────────────┐
│                    USV-FUNKTION                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Normalbetrieb:                                                │
│                                                                 │
│   ┌────────┐      ┌──────────┐      ┌──────────┐              │
│   │ Strom- │ ───► │   USV    │ ───► │  Server  │              │
│   │  netz  │      │ (lädt)   │      │          │              │
│   └────────┘      └──────────┘      └──────────┘              │
│                                                                 │
│   Bei Stromausfall:                                             │
│                                                                 │
│   ┌────────┐      ┌──────────┐      ┌──────────┐              │
│   │ Strom- │  ✗   │   USV    │ ───► │  Server  │              │
│   │  netz  │      │(Batterie)│      │(läuft    │              │
│   └────────┘      └──────────┘      │ weiter)  │              │
│                                      └──────────┘              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

### Aufgaben einer USV

| Aufgabe | Beschreibung |
|---------|--------------|
| **Überbrückung** | Stromversorgung bei Netzausfall |
| **Spannungsglättung** | Ausgleich von Schwankungen |
| **Überspannungsschutz** | Schutz vor Spannungsspitzen |
| **Sicheres Herunterfahren** | Zeit für geordnetes Abschalten |

---

### USV-Typen

```
┌────────────────────────────────────────────────────────────────┐
│                    USV-TYPEN                                   │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  1. OFFLINE/STANDBY                                            │
│     ┌───────┐                                                  │
│     │  AC   │──────────────────────►  Verbraucher             │
│     └───────┘        ↑                                         │
│                      │ (nur bei Ausfall)                       │
│                 ┌────┴────┐                                    │
│                 │ Batterie│                                    │
│                 └─────────┘                                    │
│     → Günstig, kurze Umschaltzeit (5-12 ms)                   │
│                                                                │
│  2. LINE-INTERACTIVE                                           │
│     ┌───────┐    ┌──────────┐                                 │
│     │  AC   │───►│Spannungs-│────────►  Verbraucher           │
│     └───────┘    │ regler   │                                 │
│                  └────┬─────┘                                  │
│                       │                                        │
│                  ┌────┴────┐                                   │
│                  │ Batterie│                                   │
│                  └─────────┘                                   │
│     → Spannungsregelung, schnelle Umschaltung (2-4 ms)        │
│                                                                │
│  3. ONLINE/DOPPELWANDLER                                       │
│     ┌───────┐    ┌────────┐    ┌────────┐                     │
│     │  AC   │───►│Gleich-  │───►│Wechsel-│───► Verbraucher    │
│     └───────┘    │richter  │    │richter │                     │
│                  └────┬────┘    └────────┘                     │
│                       │                                        │
│                  ┌────┴────┐                                   │
│                  │ Batterie│ (immer in Reihe)                  │
│                  └─────────┘                                   │
│     → Beste Qualität, keine Umschaltzeit (0 ms)               │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Vergleich der USV-Typen

| Typ | Umschaltzeit | Schutz | Effizienz | Preis | Einsatz |
|-----|--------------|--------|-----------|-------|---------|
| **Offline/Standby** | 5-12 ms | Basis | Hoch | Günstig | Heim-PC |
| **Line-Interactive** | 2-4 ms | Mittel | Mittel | Mittel | Büro, Server |
| **Online/Doppelwandler** | 0 ms | Maximal | Niedriger | Hoch | Rechenzentrum |

---

### Wichtige Kennzahlen

| Kennzahl | Beschreibung | Beispiel |
|----------|--------------|----------|
| **VA (Voltampere)** | Scheinleistung | 1500 VA |
| **Watt** | Wirkleistung (~60-70% von VA) | 900 W |
| **Autonomiezeit** | Laufzeit bei Stromausfall | 5-30 Minuten |
| **Leistungsfaktor** | Verhältnis Watt/VA | 0,6 - 0,9 |

**Faustformel:** Watt ≈ VA × 0,6 bis 0,7

---

### Dimensionierung

```
┌────────────────────────────────────────────────────────────────┐
│                USV-DIMENSIONIERUNG                             │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  Schritt 1: Leistungsbedarf ermitteln                         │
│                                                                │
│    Server:           300 W                                     │
│    Monitor:           50 W                                     │
│    Switch:            20 W                                     │
│    ─────────────────────────                                   │
│    Summe:            370 W                                     │
│                                                                │
│  Schritt 2: Sicherheitszuschlag (+20-30%)                     │
│                                                                │
│    370 W × 1,3 = 481 W                                        │
│                                                                │
│  Schritt 3: VA-Wert berechnen                                 │
│                                                                │
│    481 W ÷ 0,6 = 802 VA → USV mit mindestens 1000 VA         │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

---

### Störungen im Stromnetz

| Störung | Beschreibung | USV-Schutz |
|---------|--------------|------------|
| **Stromausfall** | Kompletter Ausfall | Alle Typen |
| **Spannungsabfall** | Spannung zu niedrig | Line-Interactive, Online |
| **Überspannung** | Spannung zu hoch | Line-Interactive, Online |
| **Spannungsspitzen** | Kurze Spitzen | Alle Typen |
| **Frequenzschwankung** | Netzfrequenz instabil | Nur Online |
| **Rauschen** | Elektr. Störungen | Online |

---

### Einsatzszenarien

| Szenario | USV-Typ | Begründung |
|----------|---------|------------|
| **Heim-PC** | Offline/Standby | Kostengünstig, einfacher Schutz |
| **Büro-Arbeitsplatz** | Line-Interactive | Guter Kompromiss |
| **Einzelner Server** | Line-Interactive | Schnelle Umschaltung |
| **Serverraum** | Online | Maximaler Schutz |
| **Rechenzentrum** | Online + Diesel | Höchste Verfügbarkeit |

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **USV** | Unterbrechungsfreie Stromversorgung |
| **VA** | Voltampere (Scheinleistung) |
| **Autonomiezeit** | Überbrückungszeit bei Stromausfall |
| **Online-USV** | Doppelwandler, höchste Qualität |
| **Offline-USV** | Standby, schaltet bei Ausfall um |
| **Line-Interactive** | Mittlere Variante mit Spannungsregler |

## Prüfungstipps

### Häufige Prüfungsfragen

- Wozu dient eine USV?
- Welche USV-Typen gibt es?
- Wie dimensioniert man eine USV?

### Wichtig zu merken

- **USV** = Schutz vor Stromausfall und Spannungsschwankungen
- **Online-USV** = beste Qualität, 0 ms Umschaltzeit
- **VA > Watt** (Faktor ca. 0,6-0,7)
- USV ermöglicht **sicheres Herunterfahren**

## Übungsaufgaben

### Aufgabe 1
Nennen Sie die drei USV-Typen und ihre Umschaltzeiten.

**Lösung:**
1. **Offline/Standby:** 5-12 ms Umschaltzeit
2. **Line-Interactive:** 2-4 ms Umschaltzeit
3. **Online/Doppelwandler:** 0 ms Umschaltzeit

### Aufgabe 2
Ein Server benötigt 400 Watt. Welche USV-Kapazität (VA) wird empfohlen?

**Lösung:**
1. Sicherheitszuschlag: 400 W × 1,3 = 520 W
2. VA-Berechnung: 520 W ÷ 0,6 = 867 VA
3. **Empfehlung:** USV mit mindestens **1000 VA**

### Aufgabe 3
Warum ist eine Online-USV für ein Rechenzentrum besser geeignet als eine Offline-USV?

**Lösung:**
- **Keine Umschaltzeit:** Online-USV liefert ohne Unterbrechung (0 ms), Offline-USV hat 5-12 ms Umschaltzeit
- **Konstante Spannung:** Doppelwandler erzeugt immer sauberen Strom
- **Vollständiger Schutz:** Schützt vor allen Störungsarten (auch Frequenzschwankungen, Rauschen)
- **Kritische Systeme:** Rechenzentren erfordern höchste Verfügbarkeit
