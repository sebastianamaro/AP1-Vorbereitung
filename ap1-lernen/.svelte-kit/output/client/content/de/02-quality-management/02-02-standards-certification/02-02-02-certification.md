# Zertifizierung

## Lernziele

Nach diesem Kapitel solltest du:
- Den Zweck einer Zertifizierung verstehen
- Den Ablauf einer Zertifizierung beschreiben können
- Wissen, was ein Audit ist
- Verschiedene Zertifizierungsbeispiele kennen

---

## Kerninhalt

### Was ist eine Zertifizierung?

Eine **Zertifizierung** ist ein formaler Nachweis, dass ein Unternehmen, ein Produkt oder eine Person festgelegte Standards erfüllt. Sie wird durch unabhängige, externe Prüfstellen durchgeführt.

**Vorteile einer Zertifizierung:**
- Nachweis der Qualitätsfähigkeit gegenüber Kunden
- Wettbewerbsvorteil am Markt
- Systematische Verbesserung interner Prozesse
- Erfüllung von Kundenanforderungen (z.B. bei Ausschreibungen)
- Vertrauensbildung bei Geschäftspartnern

---

### Das Audit

Ein **Audit** ist ein systematisches, dokumentiertes Untersuchungsverfahren zur Bewertung der Einhaltung von Standards.

#### Arten von Audits

| Audit-Art | Beschreibung | Durchführung |
|-----------|--------------|--------------|
| **Internes Audit** | Selbstprüfung durch eigene Mitarbeiter | Unternehmensintern |
| **Externes Audit** | Prüfung durch unabhängige Stelle | Zertifizierungsstelle |
| **Lieferantenaudit** | Prüfung von Zulieferern | Kunde prüft Lieferanten |

#### Der Auditor

Ein **Auditor** ist eine speziell ausgebildete Person, die Audits durchführt:

- Unabhängig und objektiv
- Fachlich qualifiziert
- Bei externen Audits: von akkreditierter Stelle

### Ablauf einer Zertifizierung

```
┌─────────────────┐
│ 1. VORBEREITUNG │
│ • Selbstevaluierung
│ • Dokumentation
│ • Interne Audits
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 2. ANTRAG       │
│ • Zertifizierungs-
│   stelle wählen
│ • Antrag stellen
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 3. AUDIT        │
│ • Dokumenten-
│   prüfung
│ • Vor-Ort-Audit
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 4. ZERTIFIKAT   │
│ • Bei Erfolg:
│   Zertifikat
│ • Gültigkeit:
│   meist 3 Jahre
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 5. ÜBERWACHUNG  │
│ • Jährliche
│   Überwachungs-
│   audits
└─────────────────┘
```

#### Die Phasen im Detail

| Phase | Aktivitäten |
|-------|-------------|
| **1. Vorbereitung** | QMS aufbauen, Dokumentation erstellen, interne Audits durchführen |
| **2. Antrag** | Zertifizierungsstelle auswählen (z.B. TÜV, DEKRA), Antrag einreichen |
| **3. Audit** | Stufe 1: Dokumentenprüfung; Stufe 2: Vor-Ort-Audit |
| **4. Zertifikat** | Bei positivem Ergebnis: Zertifikat (typisch 3 Jahre gültig) |
| **5. Überwachung** | Regelmäßige Überwachungsaudits (jährlich), Re-Zertifizierung nach 3 Jahren |

### Bekannte Zertifizierungsstellen

| Stelle | Bereich |
|--------|---------|
| **TÜV** (Technischer Überwachungsverein) | Vielfältige Zertifizierungen |
| **DEKRA** | Prüfungen und Zertifizierungen |
| **DQS** (Deutsche Gesellschaft zur Zertifizierung von Managementsystemen) | Managementsysteme |

---

### Wichtige Zertifizierungen in der IT

| Zertifizierung | Bereich | Beschreibung |
|----------------|---------|--------------|
| **ISO 9001** | Qualitätsmanagement | Nachweis eines funktionierenden QMS |
| **ISO 27001** | Informationssicherheit | Nachweis eines ISMS |
| **CE-Kennzeichnung** | Produktsicherheit | Europäische Konformität |
| **ITIL** | IT-Service-Management | Best Practices für IT-Services |

---

### Akkreditierung vs. Zertifizierung

| Begriff | Bedeutung |
|---------|-----------|
| **Zertifizierung** | Bestätigung, dass ein Unternehmen/Produkt Standards erfüllt |
| **Akkreditierung** | Bestätigung, dass eine Zertifizierungsstelle kompetent ist, Zertifizierungen durchzuführen |

**Merke:** Eine akkreditierte Stelle zertifiziert Unternehmen.

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Zertifizierung** | Formaler Nachweis der Erfüllung von Standards |
| **Audit** | Systematisches Untersuchungsverfahren |
| **Auditor** | Speziell ausgebildeter Prüfer |
| **Zertifizierungsstelle** | Organisation, die Zertifizierungen durchführt |
| **Akkreditierung** | Anerkennung einer Zertifizierungsstelle |
| **Überwachungsaudit** | Regelmäßige Prüfung zur Aufrechterhaltung des Zertifikats |
| **Re-Zertifizierung** | Erneute vollständige Zertifizierung nach Ablauf |

## Prüfungstipps

### Häufige Prüfungsfragen
- Was ist der Zweck einer Zertifizierung?
- Beschreiben Sie den Ablauf einer Zertifizierung
- Was ist ein Audit?
- Welche Arten von Audits gibt es?

### Wichtig zu merken
- Zertifizierungen sind **zeitlich begrenzt** (meist 3 Jahre)
- **Überwachungsaudits** finden jährlich statt
- Nur **akkreditierte** Stellen dürfen zertifizieren

## Übungsaufgaben

### Aufgabe 1
Was ist eine Zertifizierung und welchen Nutzen hat sie für ein Unternehmen?

**Lösung:** Eine Zertifizierung ist ein formaler Nachweis, dass ein Unternehmen festgelegte Standards erfüllt. Nutzen:
- Nachweis der Qualitätsfähigkeit
- Wettbewerbsvorteil
- Vertrauensbildung bei Kunden und Partnern
- Voraussetzung für bestimmte Aufträge/Ausschreibungen

### Aufgabe 2
Beschreiben Sie den Ablauf einer Zertifizierung in fünf Schritten.

**Lösung:**
1. **Vorbereitung:** Selbstevaluierung, Dokumentation, interne Audits
2. **Antrag:** Zertifizierungsstelle wählen, Antrag stellen
3. **Audit:** Dokumentenprüfung und Vor-Ort-Audit durch externe Auditoren
4. **Zertifikat:** Bei positivem Ergebnis Ausstellung des Zertifikats
5. **Überwachung:** Regelmäßige Überwachungsaudits zur Aufrechterhaltung

### Aufgabe 3
Unterscheiden Sie internes und externes Audit.

**Lösung:**
- **Internes Audit:** Selbstprüfung durch eigene, geschulte Mitarbeiter zur Vorbereitung und Verbesserung
- **Externes Audit:** Prüfung durch unabhängige Zertifizierungsstelle zur offiziellen Zertifizierung
