# Beschaffungsprozess

## Lernziele

Nach diesem Abschnitt kannst du:
- Die Phasen des Beschaffungsprozesses benennen
- Bedarfsarten unterscheiden
- Den Ablauf von der Bedarfsermittlung bis zur Bestellung beschreiben
- Relevante Dokumente im Beschaffungsprozess zuordnen

---

## Kerninhalt

### Der Beschaffungsprozess im Überblick

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BESCHAFFUNGSPROZESS                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌───────────────┐                                                  │
│  │ 1. BEDARF     │  "Was brauchen wir?"                            │
│  │    ERMITTELN  │  → Bedarfsart bestimmen                         │
│  └───────┬───────┘  → Menge und Zeitpunkt festlegen                │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 2. LIEFERANTEN│  "Wer kann liefern?"                            │
│  │    SUCHEN     │  → Potenzielle Anbieter ermitteln               │
│  └───────┬───────┘  → Anfragen versenden                           │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 3. ANGEBOTE   │  "Was kostet es?"                               │
│  │    EINHOLEN   │  → Angebote anfordern                           │
│  └───────┬───────┘  → Konditionen vergleichen                      │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 4. ANGEBOTE   │  "Welches Angebot ist am besten?"               │
│  │    VERGLEICHEN│  → Quantitativ (Bezugspreis)                    │
│  └───────┬───────┘  → Qualitativ (Nutzwertanalyse)                 │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 5. ENTSCHEIDUNG│ "Wen wählen wir?"                              │
│  │    TREFFEN    │  → Lieferantenauswahl                           │
│  └───────┬───────┘  → Ggf. Verhandlung                             │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 6. BESTELLUNG │  "Wir kaufen!"                                  │
│  │    AUFGEBEN   │  → Bestellung versenden                         │
│  └───────┬───────┘  → Auftragsbestätigung erhalten                 │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 7. WARENEIN-  │  "Ist alles da?"                                │
│  │    GANG       │  → Lieferung prüfen                             │
│  └───────┬───────┘  → Mängel reklamieren                           │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 8. RECHNUNGS- │  "Stimmt die Rechnung?"                         │
│  │    PRÜFUNG    │  → Rechnung mit Bestellung vergleichen          │
│  └───────┬───────┘  → Zahlung veranlassen                          │
│          │                                                          │
│          ▼                                                          │
│  ┌───────────────┐                                                  │
│  │ 9. ZAHLUNG    │  "Wir bezahlen!"                                │
│  │               │  → Skonto nutzen                                 │
│  └───────────────┘  → Buchung                                       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Bedarfsarten

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BEDARFSARTEN                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  PRIMÄRBEDARF                                                       │
│  → Bedarf an Endprodukten                                           │
│  → Z.B. fertige PCs, die an Kunden verkauft werden                  │
│                                                                      │
│  SEKUNDÄRBEDARF                                                     │
│  → Bedarf an Bauteilen/Komponenten für die Produktion               │
│  → Z.B. Prozessoren, RAM, Festplatten für PC-Montage                │
│                                                                      │
│  TERTIÄRBEDARF                                                      │
│  → Bedarf an Hilfsstoffen und Betriebsmitteln                       │
│  → Z.B. Schrauben, Kabel, Verpackung, Strom                         │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  Für IT-Dienstleister (ConSystem GmbH):                             │
│  → Primärbedarf: Fertige Lösungen für Kunden                        │
│  → Sekundärbedarf: Hardware, Software, Lizenzen                     │
│  → Tertiärbedarf: Büromaterial, Kabel, Werkzeug                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Bedarfsermittlungsmethoden

| Methode | Beschreibung | Anwendung |
|---------|--------------|-----------|
| **Programmorientiert** | Aus Produktionsprogramm ableiten | Bekannter Bedarf |
| **Verbrauchsorientiert** | Aus Vergangenheitswerten | Regelmäßiger Bedarf |
| **Heuristisch** | Schätzung durch Experten | Neuer/unbekannter Bedarf |

### Dokumente im Beschaffungsprozess

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DOKUMENTE IM BESCHAFFUNGSPROZESS                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Phase              │ Dokument               │ Von → An              │
│  ───────────────────┼────────────────────────┼─────────────────────  │
│  Bedarfsmeldung     │ Bestellanforderung     │ Abteilung → Einkauf   │
│  Lieferantensuche   │ Anfrage                │ Einkauf → Lieferant   │
│  Angebote einholen  │ Angebot                │ Lieferant → Einkauf   │
│  Bestellung         │ Bestellung             │ Einkauf → Lieferant   │
│  Bestätigung        │ Auftragsbestätigung    │ Lieferant → Einkauf   │
│  Lieferung          │ Lieferschein           │ Lieferant → Einkauf   │
│  Abrechnung         │ Rechnung               │ Lieferant → Einkauf   │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  ANFRAGE  ≠  ANGEBOT  ≠  BESTELLUNG                                 │
│                                                                      │
│  Anfrage:    "Was kostet X?" (unverbindlich)                        │
│  Angebot:    "X kostet Y €" (bindet den Anbieter)                   │
│  Bestellung: "Ich kaufe X!" (bindet den Käufer)                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Bestellung vs. Auftragsbestätigung

```
┌─────────────────────────────────────────────────────────────────────┐
│               BESTELLUNG UND AUFTRAGSBESTÄTIGUNG                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  KÄUFER                                       VERKÄUFER             │
│  (ConSystem GmbH)                             (Lieferant)           │
│                                                                      │
│       ┌─────────────────┐                                           │
│       │   BESTELLUNG    │ ─────────────────────────────►           │
│       │ • Artikel       │                                           │
│       │ • Menge         │                                           │
│       │ • Preis         │                                           │
│       │ • Liefertermin  │                                           │
│       └─────────────────┘                                           │
│                                                                      │
│                         ◄─────────────────────────────               │
│       ┌─────────────────┐                                           │
│       │AUFTRAGSBESTÄTI- │                                           │
│       │GUNG (AB)        │                                           │
│       │ • Bestätigt     │                                           │
│       │   Bestellung    │                                           │
│       │ • Oder zeigt    │                                           │
│       │   Abweichungen  │                                           │
│       └─────────────────┘                                           │
│                                                                      │
│  WICHTIG: Auftragsbestätigung IMMER prüfen!                         │
│  → Stimmen Artikel, Menge, Preis, Termin?                           │
│  → Bei Abweichung sofort reklamieren!                               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Wareneingangsprüfung

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WARENEINGANGSPRÜFUNG                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  1. MENGENKONTROLLE (sofort bei Anlieferung)                        │
│     → Anzahl der Pakete stimmt?                                     │
│     → Alle bestellten Artikel vorhanden?                            │
│     → Auf Lieferschein vermerken!                                   │
│                                                                      │
│  2. SICHTKONTROLLE (sofort bei Anlieferung)                         │
│     → Verpackung unbeschädigt?                                      │
│     → Sichtbare Transportschäden?                                   │
│     → Bei Schäden: SOFORT dokumentieren und melden!                 │
│                                                                      │
│  3. QUALITÄTSKONTROLLE (nach Auspacken)                             │
│     → Ware entspricht Bestellung?                                   │
│     → Funktioniert die Ware?                                        │
│     → Versteckte Mängel?                                            │
│                                                                      │
│  Bei Mängeln:                                                       │
│  → Innerhalb der Rügefrist reklamieren (i.d.R. unverzüglich)        │
│  → Mängelrechte geltend machen                                      │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Bedarfsermittlung** | Feststellung, was in welcher Menge benötigt wird |
| **Anfrage** | Unverbindliche Bitte um ein Angebot |
| **Angebot** | Verbindliche Erklärung über Preis und Konditionen |
| **Bestellung** | Verbindliche Kaufabsicht des Kunden |
| **Auftragsbestätigung** | Bestätigung des Auftrags durch den Lieferanten |
| **Lieferschein** | Begleitet die Ware bei Lieferung |
| **Rügefrist** | Zeitraum für die Meldung von Mängeln |

---

## Prüfungstipps

1. **Reihenfolge der Dokumente:**
   - Anfrage → Angebot → Bestellung → AB → Lieferschein → Rechnung

2. **Unterschied merken:**
   - Anfrage = unverbindlich
   - Angebot = bindet VERKÄUFER
   - Bestellung = bindet KÄUFER

3. **Wareneingangsprüfung:**
   - Mängel SOFORT melden (Rügepflicht!)
   - Auf Lieferschein dokumentieren

- [ ] Rechnung bezahlen

- [ ] Bestellung aufgeben
- [ ] Bedarf ermitteln
- [ ] Wareneingang prüfen
- [ ] Angebote einholen
- [ ] Lieferanten suchen

### Aufgabe 2: Bedarfsarten zuordnen

Die ConSystem GmbH plant die Auslieferung von 10 PC-Arbeitsplätzen an einen Kunden. Ordnen Sie zu:

a) Die 10 fertigen PCs für den Kunden
b) Prozessoren und RAM für die PC-Montage
c) Kabelbinder für die Verkabelung
d) Kartons für den Versand

<details>
<summary>Lösung anzeigen</summary>

a) **Primärbedarf** - Endprodukte für den Kunden
b) **Sekundärbedarf** - Komponenten für die Produktion
c) **Tertiärbedarf** - Hilfsstoffe
d) **Tertiärbedarf** - Betriebsmittel/Verpackung

</details>

### Aufgabe 3: Wareneingangskontrolle

Bei der Lieferung von 20 Monitoren stellen Sie fest:
- 2 Kartons sind eingedrückt
- Laut Lieferschein sind nur 18 Monitore enthalten

Welche Schritte müssen Sie enterprise?

<details>
<summary>Lösung anzeigen</summary>

**Sofortmaßnahmen:**
1. **Mengendifferenz dokumentieren:**
   - Auf dem Lieferschein vermerken: "Nur 18 statt 20 Monitore erhalten"
   - Vom Fahrer unterschreiben lassen

2. **Beschädigte Kartons dokumentieren:**
   - Fotos machen
   - Auf Lieferschein vermerken: "2 Kartons beschädigt"

3. **Lieferant kontaktieren:**
   - Sofort telefonisch/per E-Mail reklamieren
   - Fehlmenge und Beschädigung melden
   - Schriftliche Mängelrüge senden

4. **Ware prüfen:**
   - Beschädigte Kartons öffnen
   - Monitore auf Funktionstüchtigkeit prüfen
   - Ergebnis dokumentieren

5. **Interne Dokumentation:**
   - Wareneingangsbericht erstellen
   - Für Rechnungsprüfung: Nur 18 Monitore bezahlen!

</details>
