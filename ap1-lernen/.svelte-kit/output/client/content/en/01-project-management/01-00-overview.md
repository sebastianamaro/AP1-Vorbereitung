# Project Management - Overview

## Learning Objectives

After this chapter, you should be able to:
- Understand the fundamentals of project management
- Distinguish projects from routine tasks
- Apply important planning tools (Gantt, network diagram)
- Know and compare process models (Waterfall, Scrum)
- Apply the SMART principle for goal formulation

---

## Core Content

### What is Project Management?

**Project management** encompasses the planning, steering, and controlling of projects. It involves the application of methods, tools, techniques, and competencies in a project (according to DIN ISO 21500).

### Management Areas in Project Management

Project management encompasses or utilizes the following areas:

| Area | Description |
|------|-------------|
| **Schedule Management** | Time and sequence planning |
| **Cost Management** | Budget planning and control |
| **Quality Management** | Ensuring result quality |
| **Risk Management** | Identification and handling of risks |
| **Communication Management** | Information flow in the project |
| **Human Resource Management** | Team composition and leadership |
| **Procurement Management** | Purchasing of resources and services |
| **Scope Management** | Definition of project scope |
| **Integration Management** | Coordination of all areas |

---

### The Magic Triangle of Project Management

The three central factors of a project are in tension with each other:

```
           QUALITY
              /\
             /  \
            /    \
           /      \
          /________\
     COST        TIME
```

**Relationship:** If one factor changes, it affects the other two. Example: Less time often leads to higher costs or lower quality.
