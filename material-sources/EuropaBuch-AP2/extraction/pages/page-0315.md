Ihre Erläuterung:

DMZ ist eine Abkürzung für Demilitarized Zone (Demilitarisierte Zone). Eine DMZ kann in einem
Netzwerk mithilfe von Firewalls aufgebaut werden. Die Rechner (meistens Server) in dieser Zone
werden durch Firewalls isoliert. Ein klassisches Beispiel dazu ist eine Firma mit einem Intranet und
einem Web-Server. Der Zugriff auf den Web-Server soll aus dem Internet möglich sein, aber es
muss verhindert werden, dass Zugriffe aus dem Internet über den Web-Server in das Intranet
erfolgen. Der Web-Server wird dazu durch eine DMZ abgesichert.
