2.7.6 SQL-Skript

Ausgangsszenario:

Eine Kunde der IT-Firma ConSystem GmbH hat seine Datenbank neu konzipiert und die entsprechenden
Tabellen aufgestellt. Für die Umsetzung der Tabellen in die relationale Datenbank braucht er Unter-
stützung.

Aufgabenstellung:

Als Auszubildende/-r der Firma erhalten Sie den Auftrag, das entsprechende SQL-Skript zu schreiben,
das die untenstehenden Tabellen erzeugt und mit den Daten füllt.

Ihr SQL-Skript ConSEystem

GmbH

CREATE TABLE Kategorie

(
Kategorie_ID INTEGER PRIMARY KEY,
Bezeichnung VARCHAR(100)

)

CREATE TABLE Ressource

(
Ressourcen_ID INTEGER PRIMARY KEY,
Bezeichnung VARCHAR(100),
Hersteller VARCHAR(100),
Kategorie_ID INTEGER NOT NULL,
CONSTRAINT FK_Kat FOREIGN KEY (Kategorie_ID) REFERENCES Kategorie(Kategorie_ID)

)

CREATE TABLE Projekt

(
Projekt_ID INTEGER PRIMARY KEY,
Bezeichnung VARCHAR(100),
Startdatum DATE,
Enddatum DATE

)

CREATE TABLE Ressource_Projekt

(
Res_Pro_ID INTEGER PRIMARY KEY,
Ressourcen_ID INTEGER,
Projekt_ID INTEGER,
CONSTRAINT FK_Res FOREIGN KEY (Ressourcen_ID) REFERENCES Ressource(Ressourcen_ID),
CONSTRAINT FK_Pro FOREIGN KEY (Projekt_ID) REFERENCES Projekt(Projekt_ID)
