Der Kunde möchte den Kenntnisstand seiner Entwickler/-innen verbessern und hat GmbH

a) Übersetzen Sie den englischen Text zu Kernaussagen der OOP ins Deutsche [4 Punkte]:

Ihre Übersetzung:

Die objektorientierte Programmierung ist ein formaler Programmieransatz, der Daten und
zugehörige Aktionen (Methoden) in logischen Strukturen (Objekten) zusammenführt. Dieser
Ansatz verbessert die Möglichkeit, mit Softwarekomplexität umzugehen — das ist insbeson-
dere wichtig beim Entwickeln und Warten von großen Anwendungen und Datenstrukturen.

0 Aus einer Klasse kann immer nur ein Objekt instanziiert werden.

X Eine Methode kann auch privat sein.

X Kapselung bedeutet, dass nur über Methoden auf Attribute zugegriffen werden kann.
0 Bei der Vererbung werden nur die Methoden weitervererbt.

X Public-Attribute verhalten sich innerhalb der Klasse wie private Attribute.

0 Die Begriffe abstrakte Klasse und Interface haben die gleiche Bedeutung.

0 Interfaces werden wie Klassen vererbt.

XI Konstruktoren werden bei der Instanziierung automatisch aufgerufen.

0 Methoden dürfen nicht innerhalb von Methoden aufgerufen werden.
