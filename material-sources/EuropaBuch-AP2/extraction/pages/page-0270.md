Erläuterung des Grundkonzepts einer VDI-Umgebung:

Virtual Desktop Infrastructure (VDI) ist eine Technologie, die es ermöglicht, Desktop-Umgebun-
gen auf einem zentralen Server zu hosten und diese über ein Netzwerk an Endgeräte zu liefern.
Benutzer können von verschiedenen Geräten aus auf ihre Desktop-Umgebung zugreifen, als ob
sie lokal auf ihrem eigenen Gerät laufen würde. Da beim vorliegenden Fall bereits Endgeräte
vorhanden sind, könnte bei Einführung einer VDI-Lösung auf die Beschaffung leistungsstarker
Endgeräte verzichtet werden.

Es soll sowohl eine selbst gehostete Variante als auch eine Cloud-Lösung für VDI-Umgebungen be-
leuchtet werden. Für diesen Zweck haben Sie bereits einen Text vorbereitet.
