Informationen zum Thema GUI-Design (Forsetzung)

Information 3: Bei der Entwicklung von grafischen Benutzeroberflächen (GUI)
kommt es darauf an, dass die Kommunikation zwischen Schnittelle (Interface) und
Mensch reibungslos abläuft. Die Normenreihe DIN EN ISO 9241 bietet dazu sieben allgemeine Inter-
aktionskriterien für die Gestaltung von Benutzeroberflächen:

Erlernbarkeit Selbstbeschreibungsfähigkeit Erwartungskonformität
Steuerbarkeit Aufgabenangemessenheit Robustheit Benutzerbindung
Ordnen Sie die Interaktionskriterien der entsprechenden Beschreibung zu:

Die Schnittstelle sollte so gestaltet sein, dass die zu erledigende
Aufgabe vom Benutzer reibungslos bearbeitet werden kann.

Die Schnittstelle informiert den Benutzer über Vorgänge oder
Selbstbeschreibungsfähigkeit Fehler und beschreibt mögliche Alternativen, so dass Arbeits-
abläufe nicht an fehlenden Informationen scheitern.

Die Schnittstelle sollte die Interaktionsmöglichkeiten so an-
bieten, wie es der Benutzer erwarten würde.

Die Schnittstelle sollte Möglichkeiten zum Verstehen der

Erlernbarkeit Schnittstelle bieten.

Der Benutzer sollte die Möglichkeiten erhalten, die Nutzung der
Schnittstelle zu steuern.

Auf Fehleingaben oder auch das versehentliche Drücken eines
Robustheit Buttons sollte die Schnittstelle geeignet reagieren und Möglich-
keiten der Korrektur anbieten.

Schnittstellen sollten so gestaltet sein, dass sie die Benutzer bin-
den — die Benutzer sollten gerne mit der Schnittstelle arbeiten.
