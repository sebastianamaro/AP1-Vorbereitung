Wiederholungsfrage 23: Netzwerke

Welche der folgenden Aussagen zu DHCP sind korrekt?

O0 DHCP steht für Dynamic Host Control Protocol.

DO DHCP wird verwendet, um Namen in IP-Adressen aufzulösen und umgekehrt.

XI DHCP steht für Dynamic Host Configuration Protocol.

0 DHCP wird nur bei Verwendung von IPv6 eingesetzt.

X DHCP wird verwendet, um die IP-Konfiguration automatisiert an Geräte zuzuweisen.

X DHCP kann auch Optionen wie den Domänennamen oder den PXE Boot-Server bereitstellen.
