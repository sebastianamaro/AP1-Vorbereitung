2.7.1 Software-Schnittstellen

Ausgangsszenario:
Ein Kunde der IT-Firma ConSystem GmbH möchte eine Beratung im Bereich der Software-Schnittstellen.
Er hat dazu einige Aspekte aufgeschrieben, die ihm unklar sind.

Aufgabenstellung:
Als erfahrene/-r Auszubildende/-r der Firma werden Sie beauftragt, die unten stehenden Informationen
zu ergänzen.

Informationen zu Software-Schnittstellen

Information 1: Im Zusammenhang mit Software-Schnittstellen ist dem Kunden der
Begriff API aufgefallen. Was bedeutet API?

Ihre Definition:

API steht für „Application Programming Interface“ und bezeichnet eine Programmierschnittstelle.
Eine API bietet die möglichen Interaktionen mit einer Software-Komponente. Eine bekannte API
ist die Windows-API zur Erstellung von Anwendungsprogrammen für Windows-Betriebssysteme.

mit Software-Schnittstellen zusammengestellt. Vervollständigen Sie die Zusammenstellung durch die
korrekte Einordung der folgenden Begriffe (Begriffe können auch mehrfach eingeordnet werden):
