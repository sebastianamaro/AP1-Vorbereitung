Ihre Begründung:

Bei diesen Leitungen handelt es sich um Trunk-Leitungen. Auf Trunk-Leitungen werden die
Daten mehrerer VLANs transportiert. Anhand von VLAN-Tags kann eine Zuordnung zu den
VLANS erfolgen. Das VLAN-Tag wird hinter der MAC-Adresse des Senders im Ethernet Header
eingefügt.

Aufgabe 5: Der Kunde beschäftigt auch Außendienstmitarbeiter/-innen, die im Firmengebäude
keinen festen Arbeitsplatz haben. Sie werden aufgefordert, ein mögliches technisches Konzept dar-
zustellen, das es erlaubt, die Außendienstmitarbeiter/-innen unabhängig von Ihrem Arbeitsplatz in
das für sie richtige VLAN einzuordnen.

Ihre Darstellung:

Oft werden VLANS statisch konfiguriert. Hierzu wird einem bestimmten Switchport ein VLAN
zugeordnet. Es ist allerdings auch möglich eine dynamische Zuordnung zu VLANs vorzunehmen.
Eine Option ist es, das Endgerät anhand der MAC-Adresse einem bestimmten VLAN zuzuord-
nen. Eine weitere Option ist die Zuweisung des jeweiligen VLANs über einen RADIUS-Server.

In diesem Fall ist das VLAN nicht an die MAC-Adresse des Geräts, sondern an das Benutzerkonto
gekoppelt. Dies ist sicherheitstechnisch günstiger.
