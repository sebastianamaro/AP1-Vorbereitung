# Page 89 - ERD Chen-Notation Symbole (Fortsetzung)

## Symbole (Fortsetzung)

| Symbol | Name | Bedeutung |
|--------|------|-----------|
| Ellipse | Attribut | Merkmal oder Eigenschaft einer Entität |
| Ellipse mit unterstrichenem Text | Schlüsselattribut | Eindeutiger Identifikator (Primärschlüssel) |
| Ellipse mit gestricheltem Unterstrich | Teilschlüssel | Schlüssel einer schwachen Entität |
| Doppelte Ellipse | Mehrwertiges Attribut | Attribut mit mehreren möglichen Werten |
| Gestrichelte Ellipse | Abgeleitetes Attribut | Berechnetes Attribut aus anderen Attributen |
| Raute | Beziehung | Verbindung zwischen Entitäten |
| Doppelte Raute | Identifizierende Beziehung | Beziehung bei schwachen Entitäten |

## Teilnahmegrad

| Symbol | Bedeutung |
|--------|-----------|
| ═ (Doppellinie) | Total (verpflichtend) |
| ─ (Einfachlinie) | Partiell (optional) |
| --- (Gestrichelt) | Optional (gestrichelt) |

## Kardinalitäten

- **1:1** (Eins-zu-Eins)
- **1:N** (Eins-zu-Viele)
- **N:1** (Viele-zu-Eins)
- **N:M** (Viele-zu-Viele)
