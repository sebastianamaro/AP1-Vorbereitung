# Page 78 - 8.8 UML - Aktivitätsdiagramm

## 8.8 UML

### Aktivitätsdiagramm

Ein **Aktivitätsdiagramm** ist eine Art von UML-Diagramm, das den Ablauf von Aktivitäten und Aktionen innerhalb eines Systems veranschaulicht. Es wird verwendet, um den Workflow eines Systems zu modellieren und die Schritte zu identifizieren, die in einem Prozess involviert sind.

### Symbole des Aktivitätsdiagramms

| Symbol | Name |
|--------|------|
| ● (ausgefüllter Kreis) | Start/Anfangsknoten |
| Abgerundetes Rechteck mit "Activity" | Aktivität/Aktionsstatus |
| Abgerundetes Rechteck mit "Action" | Aktion |
| Durchgezogener Pfeil → | Kontrollfluss/Kante |
| Gestrichelter Pfeil - - → | Objektfluss/Steuerkante |
| ◉ (Kreis mit Punkt) | Aktivität Endknoten |
| ⊗ (Kreis mit X) | Fluss-Endknoten |
| ◇ (Raute) | Entscheidungs-Knotenpunkt |
