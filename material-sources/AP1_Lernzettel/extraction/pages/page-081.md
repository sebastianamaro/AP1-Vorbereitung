# Page 81 - Anwendungsfalldiagramm Symbole (Fortsetzung)

## Symbole des Anwendungsfalldiagramms (Fortsetzung)

| Symbol | Bedeutung |
|--------|-----------|
| Durchgezogene Linie | **Beziehung zwischen Akteur und Anwendungsfall** - Ein Akteur nutzt einen Anwendungsfall, wenn er ihn auslöst. |
| Gestrichelter Pfeil mit «extend» | **«extend»-Beziehung** - Ein Anwendungsfall erweitert einen anderen unter bestimmten Bedingungen. Die Pfeilspitze zeigt auf den erweiterten Anwendungsfall. |
| Gestrichelter Pfeil mit «include» | **«include»-Beziehung** - Ein Anwendungsfall ist fester Bestandteil eines anderen. Die Pfeilspitze zeigt auf den enthaltenen Anwendungsfall. |
| Pfeil mit leerer Dreiecksspitze | **Generalisierung** - Ein Akteur oder Anwendungsfall wird spezialisiert. Die Pfeilspitze zeigt auf das übergeordnete Element. |
| Rechteck mit gestrichelter Linie | **Notiz** - Zusatzinformationen zum Modell, verbunden mit einer gestrichelten Linie. |
