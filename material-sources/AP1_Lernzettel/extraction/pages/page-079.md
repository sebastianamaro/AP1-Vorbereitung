# Page 79 - UML Aktivitätsdiagramm Symbole (Fortsetzung)

## Symbole des Aktivitätsdiagramms (Fortsetzung)

| Symbol | Name |
|--------|------|
| ◇ (Raute) | Knoten verschmelzen |
| Vertikaler Balken mit mehreren Ausgängen | Gabel (Fork) - Parallele Aktivitäten starten |
| Vertikaler Balken mit mehreren Eingängen | Zusammenführen (Join) - Parallele Aktivitäten zusammenführen |
| Fünfeck mit Pfeilspitze rechts | Senden von Signalen |
| Fünfeck mit Pfeilspitze links | Signal-Empfang |
| Rechteck mit Eselsohr | Anmerkung/Kommentar |
