# Interfaces y ergonomía del software

## Objetivos de Aprendizaje

Después de este capítulo deberías ser capaz de:
- Distinguir los tipos de interfaces
- Definir la ergonomía del software
- Nombrar los requisitos de la ArbStättV para los puestos de trabajo con pantalla

---

## Contenido Principal

### Tipos de interfaces

| Tipo | Descripción | Ejemplos |
|------|-------------|----------|
| **Interfaz de hardware** | conexión física de dispositivos | USB, HDMI, Ethernet |
| **Interfaz de software** | comunicación entre programas | API, REST, SOAP |
| **Interfaz de usuario** | persona ↔ sistema | GUI, CLI, control por voz |

### Ergonomía del software

La **ergonomía del software** se ocupa del diseño del software de modo que se pueda usar de forma **eficaz, eficiente y satisfactoria** (usabilidad). Su base incluye, entre otras, la norma **ISO 9241** (principios del diseño del diálogo: adecuación a la tarea, autodescriptividad, conformidad con las expectativas, tolerancia a errores, controlabilidad, adecuación a la individualización, facilidad de aprendizaje).

### Ordenanza de centros de trabajo (ArbStättV, Arbeitsstättenverordnung / Ordenanza alemana de centros de trabajo) – puesto de trabajo con pantalla

Requisitos para un puesto de trabajo con pantalla (selección):

- **Pantalla:** sin parpadeo, sin reflejos, giratoria/inclinable, suficientemente grande
- **Iluminación:** suficiente, sin deslumbramiento; ventanas con protección antirreflejos
- **Mesa/silla:** suficientemente grandes, de altura regulable, ergonómicas
- **Espacio:** suficiente margen de movimiento, distancia visual adecuada
- **Ruido/clima:** baja exposición al ruido, clima ambiental agradable
- **Software:** diseñado ergonómicamente (véase ISO 9241)

## Términos Clave

| Término | Explicación |
|---------|-------------|
| **API** | Interfaz de programación entre componentes de software |
| **Ergonomía del software** | Usabilidad del software (ISO 9241) |
| **ArbStättV** | Arbeitsstättenverordnung (Ordenanza alemana de centros de trabajo; incl. puestos con pantalla) |

## Consejos para el Examen

- Asociar los tipos de interfaces (hardware/software/usuario) con ejemplos.
- Tener preparados **al menos 4 requisitos de la ArbStättV** para un puesto de trabajo con pantalla.
- Vincular la ergonomía del software con la **ISO 9241** (también es tema de UX, cap. 06).
