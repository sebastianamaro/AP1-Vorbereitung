# Patrones de Diseño (Entwurfsmuster)

Los **patrones de diseño (Entwurfsmuster)** son plantillas de solución reutilizables para problemas de diseño frecuentes – **no son código terminado, sino decisiones de arquitectura probadas**. En el catálogo actualizado son explícitamente relevantes para la AP2 como **patrones de arquitectura/diseño**.

## Las tres categorías

| Categoría | Propósito | Patrones (en este capítulo) |
|-----------|-------|----------------------------|
| **Creacionales (Erzeugungsmuster)** | Encapsular la creación de objetos | **Singleton**, **Factory** |
| **Estructurales (Strukturmuster)** | Componer clases/objetos | **Facade** |
| **De comportamiento (Verhaltensmuster)** | Interacción/comunicación | **Observer**, **MVC** (patrón de arquitectura) |
