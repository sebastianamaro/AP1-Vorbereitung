# GUI Design

## Learning Objectives

After this chapter you should:
- Know the basic principles of good GUI design
- Apply the dialogue principles of ISO 9241
- Take accessibility into account

---

## Core Content

**GUI design** (Graphical User Interface) is about designing graphical interfaces in a functional and user-friendly way.

### Design principles

- **Consistency** – uniform elements, colors, terms
- **Simplicity/clarity** – not overloaded, clear structure
- **Visible feedback** – response to actions
- **Error prevention & tolerance** – validate input, allow undo
- **Orientation** – clear navigation, "Where am I?"
- **Readability** – contrast, font size, whitespace

### ISO 9241-110 – Principles of dialogue design

| Principle | Meaning |
|-----------|-----------|
| **Suitability for the task** | supports the task efficiently |
| **Self-descriptiveness** | understandable without outside help |
| **Conformity with user expectations** | consistent, predictable |
| **Error tolerance** | usable despite faulty input |
| **Controllability** | user controls the flow/pace |
| **Suitability for individualization** | adaptable to the user |
| **Suitability for learning** | easy to learn |

### Accessibility

Sufficient **contrast**, **keyboard operability**, **alternative texts** for images, scalable fonts – **newly emphasized** in the updated catalogue.

## Key Terms

| Term | Explanation |
|---------|-----------|
| **ISO 9241** | Standard for the ergonomics of human-system interaction |
| **Consistency** | Uniform design |
| **Accessibility (Barrierefreiheit)** | Usability for everyone, including people with disabilities |
