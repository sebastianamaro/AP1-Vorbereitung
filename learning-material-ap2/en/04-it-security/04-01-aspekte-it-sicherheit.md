# Aspects of IT Security

## Learning Objectives

After this chapter, you should be able to:
- Know the protection goals of IT security (CIA + additions)
- Classify the BSI IT-Grundschutz (IT baseline protection) and its topic areas
- Distinguish IT security from data protection

---

## Core Content

### The protection goals (CIA triad)

| Protection goal | Meaning |
|-----------|-----------|
| **Confidentiality** (Vertraulichkeit) | Data is accessible only to authorized parties |
| **Integrity** (Integrität) | Data is correct and unchanged |
| **Availability** (Verfügbarkeit) | Data/systems are usable when needed |

Additional goals: **Authenticity** (Authentizität), **Non-Repudiation** (Nichtabstreitbarkeit/Verbindlichkeit).

> **Memory aid:** **VIV** – Vertraulichkeit, Integrität, Verfügbarkeit (Confidentiality, Integrity, Availability).

### BSI IT-Grundschutz

The **IT-Grundschutz-Kompendium** (IT baseline protection compendium) of the **BSI** (Bundesamt für Sicherheit in der Informationstechnik – German Federal Office for Information Security) is a comprehensive work with modules on all aspects of information security. Typical topic areas (abbreviations):

| Abbreviation | Area |
|--------|---------|
| **ISMS** | Security management |
| **NET** | Network security |
| **APP** | Application security |
| **IAM** | Identity and access management |
| **PHY** | Physical security |
| **CLOUD** | Cloud security |
| **OPS** | IT operations |

### IT security vs. data protection

- **IT security:** protects **systems and data** (of all kinds) against threats.
- **Data protection (Datenschutz):** protects **personal data** and the rights of individuals (see [Chapter 03](../03-data-protection/03-00-overview.md)).

| Term | Explanation |
|---------|-----------|
| **CIA triad** | Confidentiality, Integrity, Availability |
| **BSI** | Bundesamt für Sicherheit in der Informationstechnik (German Federal Office for Information Security) |
| **IT-Grundschutz** | BSI methodology/compendium for information security |
| **ISMS** | Information Security Management System |

## Exam Tips

- Name the **three protection goals** confidently and give one example for each.
- **IT security ≠ data protection** – the distinction is a popular question.
- Cite the BSI-Grundschutz as the **standard reference** for security measures.
