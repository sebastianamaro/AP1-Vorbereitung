# Changes in the Exam Catalog (and what this means for old practice exams)

## Learning Objectives

After this chapter you should:
- Know which topics were added, moved or removed in the updated IHK exam catalog
- Be able to correctly assess old practice exams (what is still relevant, what is not?)

---

## Core Content

The **3rd edition** of the book already takes the **updated IHK exam catalog (December 2024)** into account. So you don't need to worry about the book's content – **but you do when practicing with older exams**: there, topics appear that have since been moved or are no longer tested at all.

### Topics that have moved into the AP2 / advanced area

These topics were removed from the AP1 or classified as "advanced" and belong to the AP2/advanced area:

- **SQL queries**
- **Inheritance** in object-oriented programming
- **Non-relational databases** (NoSQL)
- **RAID/SAN storage** (in AP1 only NAS remains)

### Topics that have been dropped entirely

If such a topic appears in an **exam before 2025**, it is generally **no longer exam-relevant** today:

- **Struktogramm** (Nassi-Shneiderman structured flowchart) and **Programmablaufplan (PAP)** (program flowchart)
- **SWOT analysis**
- **LTE/5G** mobile communication standards
- **ISO-27001** series of standards (in detail)
- Extensive **software quality criteria lists**

### Topics that are newly emphasized

The 2025 catalog strengthens, among others:

- **AI applications** and support
- **SMART** goals in project planning
- **Scrum** vs. **waterfall model**
- **IPv4/IPv6 configuration**
- **Two-factor authentication (2FA)**
- **DSGVO** (General Data Protection Regulation / GDPR) data subject rights
- **Hashing methods** and **anonymization**
- **UML activity diagrams**
- **Debugging code** and **Schreibtischtest** (desk check)
- **Hardening** of operating systems
- **Accessibility**

---

## Key Terms

| Term | Explanation |
|------|-------------|
| **Prüfungskatalog (exam catalog)** | Official IHK list of testable content; decisive for the exam scope |
| **Desk-Check / Schreibtischtest** | Manually stepping through code on paper to find errors |
| **SMART** | Criteria for good goals: Specific, Measurable, Attractive, Realistic, Time-bound |

---

## Exam Tips

- **When practicing with old exams:** compare each task against the lists above. A topic that was removed (e.g. Struktogramm, SWOT) no longer needs to be learned – a moved one (e.g. SQL, inheritance) now belongs to your AP2 material.
- Old exams remain valuable for practicing **task formats and time management** – only the selection of topics should be treated with caution.
- The official exam catalog (U-Form-Verlag) is the binding reference if you are unsure.
