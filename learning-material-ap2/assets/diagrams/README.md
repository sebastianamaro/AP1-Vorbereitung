# Diagram images — source & license

**Source:** ap2.online (`https://ap2.online/diagrams/…`).

**License:** Used **with the explicit permission of the site owner** (granted to the
repository owner). OK to ship in this project.

These PNGs are **language-neutral** and shared across de/en/es. They are stored
here once and referenced from the topic files via relative paths
(`../../../assets/diagrams/<name>.png`); the build copies them to
`static/content/ap2/assets/diagrams/` and the app rewrites the `src` at render time.
