# Geräteklassen und Energielabel

## Lernziele

Nach diesem Kapitel solltest du:
- Das EU-Energielabel verstehen
- Energieeffizienzklassen kennen
- Das Label bei der Geräteprocurement berücksichtigen können
- Entsorgung von IT-Geräten kennen

---

## Kerninhalt

### Das EU-Energielabel

Das **EU-Energielabel** zeigt die Energieeffizienz von Elektrogeräten auf einen Blick.

```
┌─────────────────────────────────────────────────────────────────┐
│                    EU-ENERGIELABEL                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌───────────────────────────────────────┐                    │
│   │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  A  (dunkelgrün)│  ← Sehr effizient │
│   │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  B  (hellgrün)     │                   │
│   │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓  C  (gelbgrün)        │                   │
│   │  ▓▓▓▓▓▓▓▓▓▓▓  D  (gelb)               │                   │
│   │  ▓▓▓▓▓▓▓▓  E  (orange)                │                   │
│   │  ▓▓▓▓▓  F  (dunkelorange)             │                   │
│   │  ▓▓  G  (rot)                         │  ← Weniger eff.   │
│   └───────────────────────────────────────┘                    │
│                                                                 │
│   Zusätzliche Informationen:                                   │
│   • Stromverbrauch in kWh/Jahr oder kWh/1000h                 │
│   • Produktspezifische Angaben                                 │
│   • QR-Code zur Produktdatenbank                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

### Energieeffizienzklassen (seit 2021)

Seit März 2021 gilt das neue, vereinfachte Label von **A bis G** (ohne Plus-Klassen).

| Klasse | Bewertung |
|--------|-----------|
| **A** | Beste Effizienz |
| **B** | Sehr gut |
| **C** | Gut |
| **D** | Durchschnitt |
| **E** | Unterdurchschnittlich |
| **F** | Schlecht |
| **G** | Schlechteste Effizienz |

**Hinweis:** A ist zunächst oft leer - Raum für zukünftige Verbesserungen!

### Alte vs. Neue Skala

```
┌────────────────────────────────────────────────────────────────┐
│            ALTE vs. NEUE ENERGIEKLASSEN                        │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   ALT (bis 2021)              NEU (ab 2021)                   │
│                                                                │
│   A+++  ─────────────────►   B oder C                         │
│   A++   ─────────────────►   C oder D                         │
│   A+    ─────────────────►   D oder E                         │
│   A     ─────────────────►   E oder F                         │
│   B     ─────────────────►   F oder G                         │
│   C-G   ─────────────────►   G (oder nicht mehr erlaubt)      │
│                                                                │
│   Die alte A+++-Klasse ist NICHT mehr die beste!              │
│   Klasse A ist für zukünftige Innovationen reserviert         │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Geräte mit Energielabel

| Gerätekategorie | Relevante Angaben |
|-----------------|-------------------|
| **Monitore** | Stromverbrauch in kWh/1000h, Bildschirmdiagonale |
| **Computer** | (Kein EU-Label, aber TCO-Zertifizierung) |
| **Kühlschränke** | kWh/Jahr, Volumen |
| **Waschmaschinen** | kWh/100 Zyklen, Wasserverbrauch |
| **TV-Geräte** | kWh/1000h |
| **Lichtquellen** | kWh/1000h |

### TCO-Zertifizierung für IT

Da Computer kein EU-Energielabel haben, ist die **TCO-Zertifizierung** wichtig.

```
┌────────────────────────────────────────────────────────────────┐
│                 TCO CERTIFIED                                  │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   TCO Development (Schweden) zertifiziert IT-Produkte:        │
│                                                                │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  • Energieeffizienz                                     │  │
│   │  • Ergonomie (Monitore, Tastaturen)                     │  │
│   │  • Emissionen (elektromagnetisch, chemisch)             │  │
│   │  • Langlebigkeit und Reparierbarkeit                    │  │
│   │  • Soziale Verantwortung in der Produktion             │  │
│   │  • Recyclingfähigkeit                                   │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                │
│   Produkte: Monitore, Laptops, Desktops, Drucker, Server     │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Energy Star

**Energy Star** ist ein Energieeffizienz-Programm, ursprünglich aus den USA.

| Aspekt | Beschreibung |
|--------|--------------|
| **Fokus** | Energieverbrauch im Betrieb und Standby |
| **Produkte** | Computer, Monitore, Drucker, Server |
| **Anforderung** | Unterschreiten von Energiegrenzwerten |
| **Standby** | Max. 1-2 Watt |

### Entsorgung von IT-Geräten (WEEE/ElektroG)

IT-Geräte sind **Elektroschrott** und müssen ordnungsgemäß entsorgt werden.

```
┌────────────────────────────────────────────────────────────────┐
│              ENTSORGUNG VON IT-GERÄTEN                         │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  [X]  NICHT in den Hausmüll!                           │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                │
│   Korrekte Entsorgung:                                        │
│                                                                │
│   ┌──────────────────┐  ┌──────────────────┐                 │
│   │ Wertstoffhof     │  │ Händlerrücknahme │                 │
│   │ (kommunal)       │  │ (ab 25cm²)       │                 │
│   └──────────────────┘  └──────────────────┘                 │
│                                                                │
│   ┌──────────────────┐  ┌──────────────────┐                 │
│   │ Hersteller-      │  │ Spende/Wieder-   │                 │
│   │ rücknahme        │  │ verwendung       │                 │
│   └──────────────────┘  └──────────────────┘                 │
│                                                                │
│   Wichtig:                                                    │
│   • Vorher Daten löschen!                                     │
│   • Akkus separat entsorgen                                   │
│   • Bei Firmenhardware: Nachweis der Entsorgung               │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

| Gesetz | Beschreibung |
|--------|--------------|
| **ElektroG** | Elektro- und Elektronikgerätegesetz (Deutschland) |
| **WEEE** | Waste Electrical and Electronic Equipment (EU) |

### Beschaffung energieeffizienter IT

| Kriterium | Achten auf |
|-----------|------------|
| **Energielabel** | Klasse A-C bevorzugen |
| **TCO-Zertifizierung** | Bei Monitoren und Computern |
| **Energy Star** | Effizienz im Standby |
| **80 PLUS** | Bei Netzteilen |
| **TDP** | Bei Prozessoren (niedriger = effizienter) |
| **SSD statt HDD** | Geringerer Stromverbrauch |

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Energielabel** | EU-Kennzeichnung der Energieeffizienz |
| **TCO** | Zertifizierung für IT-Produkte |
| **Energy Star** | Energieeffizienz-Programm |
| **80 PLUS** | Effizienzstandard für Netzteile |
| **ElektroG** | Gesetz zur Entsorgung von E-Schrott |
| **WEEE** | EU-Richtlinie für Elektroschrott |

## Prüfungstipps

### Häufige Prüfungsfragen

- Was zeigt das EU-Energielabel?
- Welche Effizienzklassen gibt es?
- Wie werden IT-Geräte korrekt entsorgt?
- Was ist TCO-Zertifizierung?

### Wichtig zu merken

- **EU-Label:** A bis G (A = beste Effizienz)
- **Klasse A** ist oft noch leer (Platz für Innovation)
- **TCO** = Zertifizierung für IT-Geräte
- **Elektroschrott** = NICHT in den Hausmüll!

## Übungsaufgaben

### Aufgabe 1
Welche Energieeffizienzklassen gibt es beim neuen EU-Label (seit 2021)?

**Lösung:** Das neue EU-Energielabel kennt die Klassen **A bis G**:
- A = Beste Effizienz (oft noch leer)
- B = Sehr gut
- C = Gut
- D = Durchschnitt
- E = Unterdurchschnittlich
- F = Schlecht
- G = Schlechteste Effizienz

Die alten Klassen A+++, A++, A+ wurden abgeschafft.

### Aufgabe 2
Ein Unternehmen will neue Monitore beschaffen. Auf welche Kennzeichnungen sollte geachtet werden?

**Lösung:**
- **EU-Energielabel:** Klasse A-C bevorzugen
- **TCO-Zertifizierung:** Für Energieeffizienz, Ergonomie und Umwelt
- **Energy Star:** Für effizienten Standby-Verbrauch
- Zusätzlich: Angaben zum Stromverbrauch in kWh/1000h

### Aufgabe 3
Wie müssen alte Computer und Monitore entsorgt werden?

**Lösung:** IT-Geräte sind **Elektroschrott** und dürfen **nicht in den Hausmüll**!

Korrekte Entsorgungswege:
1. **Wertstoffhof:** Kommunale Sammelstellen nehmen E-Schrott kostenlos an
2. **Händlerrücknahme:** Große Händler müssen Geräte zurücknehmen
3. **Herstellerrücknahme:** Viele Hersteller bieten Rücknahme an

**Wichtig:** Vor der Entsorgung Daten sicher löschen!
