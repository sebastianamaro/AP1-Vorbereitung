# Preiskalkulation

## Lernziele

Nach diesem Abschnitt kannst du:
- Die Zuschlagskalkulation durchführen
- IT-Stundensätze berechnen
- Handelskalkulation verstehen
- Kostenbestandteile eines Preises erklären

---

## Kerninhalt

### Grundprinzip der Kalkulation

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PREISKALKULATION                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ZIEL: Ermittlung eines Verkaufspreises, der alle Kosten deckt     │
│        und einen Gewinn ermöglicht                                  │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │   Selbstkosten                                               │    │
│  │   + Gewinnzuschlag                                           │    │
│  │   ──────────────────                                         │    │
│  │   = Barverkaufspreis (netto)                                 │    │
│  │   + Umsatzsteuer (19%)                                       │    │
│  │   ──────────────────                                         │    │
│  │   = Bruttoverkaufspreis                                      │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Zuschlagskalkulation (Handelskalkulation)

```
┌─────────────────────────────────────────────────────────────────────┐
│           VORWÄRTSKALKULATION (vom Einkauf zum Verkauf)             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│      Listeneinkaufspreis (netto)                                    │
│    - Lieferantenrabatt                                              │
│    ─────────────────────────────                                    │
│    = Zieleinkaufspreis                                              │
│    - Lieferantenskonto                                              │
│    ─────────────────────────────                                    │
│    = Bareinkaufspreis                                               │
│    + Bezugskosten                                                   │
│    ─────────────────────────────                                    │
│    = BEZUGSPREIS (Einstandspreis)                                   │
│    + Handlungskosten (Gemeinkosten)                                 │
│    ─────────────────────────────                                    │
│    = SELBSTKOSTEN                                                   │
│    + Gewinnzuschlag                                                 │
│    ─────────────────────────────                                    │
│    = Barverkaufspreis                                               │
│    + Kundenskonto                                                   │
│    ─────────────────────────────                                    │
│    = Zielverkaufspreis                                              │
│    + Kundenrabatt                                                   │
│    ─────────────────────────────                                    │
│    = LISTENVERKAUFSPREIS (netto)                                    │
│    + Umsatzsteuer (19%)                                             │
│    ─────────────────────────────                                    │
│    = BRUTTOVERKAUFSPREIS                                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

```
│                   Prod. Stunden       1.220 h                       │
│                                                                      │
│     + Gemeinkostenzuschlag (80%):        +39,40 €                   │
│     ─────────────────────────────────────────────                   │

│     + Umsatzsteuer (19%):                +20,21 €                   │
```


### Wichtige Zuschläge

| Zuschlag | Beschreibung | Typischer Wert |
|----------|--------------|----------------|
| **Handlungskosten** | Gemeinkosten (Miete, Personal, etc.) | 20-40% |
| **Gewinnzuschlag** | Angestrebter Gewinn | 10-20% |
| **Kundenskonto** | Skonto für schnelle Zahlung | 2-3% |
| **Kundenrabatt** | Rabatt auf Listenpreis | 5-20% |
| **Umsatzsteuer** | Mehrwertsteuer | 19% (7% ermäßigt) |

---

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Bezugspreis** | Einkaufspreis inkl. Bezugskosten |
| **Selbstkosten** | Alle Kosten für ein Produkt |
| **Handlungskosten** | Gemeinkosten des Betriebs |
| **Gewinnzuschlag** | Aufschlag für den Unternehmensgewinn |
| **Zuschlagskalkulation** | Kalkulation durch prozentuale Zuschläge |
| **Produktive Stunden** | Stunden, die dem Kunden berechnet werden |

---

## Prüfungstipps

1. **Kalkulationsschema merken:**
   - Bezugspreis + Handlungskosten = Selbstkosten
   - Selbstkosten + Gewinn = Barverkaufspreis
   - ... + Skonto + Rabatt + USt = Bruttopreis

2. **Skonto/Rabatt "rückwärts" rechnen:**
   - Bei Skonto/Rabatt: "Im Hundert" rechnen
   - 3% Skonto bedeutet: Barverkaufspreis × 100/97

3. **Stundensatz:**
   - Produktive Stunden sind WENIGER als Arbeitsstunden!
   - Gemeinkostenzuschlag nicht vergessen

## Übungsaufgaben

### Aufgabe 1: Verkaufspreis berechnen

ConSystem kauft einen Monitor für 200 € netto ein. Berechnen Sie den Bruttoverkaufspreis mit:
- Bezugskosten: 10 €
- Handlungskostenzuschlag: 30%
- Gewinnzuschlag: 15%
- Umsatzsteuer: 19%

<details>
<summary>Lösung anzeigen</summary>

**Berechnung:**

Einkaufspreis netto: 200,00 €
+ Bezugskosten: +10,00 €
────────────────────────────
= Bezugspreis: 210,00 €
+ Handlungskosten (30%): +63,00 €
────────────────────────────
= Selbstkosten: 273,00 €
+ Gewinn (15%): +40,95 €
────────────────────────────
= Nettoverkaufspreis: 313,95 €
+ Umsatzsteuer (19%): +59,65 €
────────────────────────────
= **Bruttoverkaufspreis: 373,60 €**

</details>

### Aufgabe 2: Stundensatz kalkulieren

Ein IT-Berater hat folgende Daten:
- Jahresgehalt brutto: 55.000 €
- Arbeitgeberanteil SV: 21%
- Sonstige Personalkosten: 3.000 €
- Arbeitstage/Jahr: 220
- Produktive Stunden: 65% der Arbeitszeit
- Gemeinkostenzuschlag: 70%
- Gewinnzuschlag: 25%

Berechnen Sie den Netto-Stundensatz.

<details>
<summary>Lösung anzeigen</summary>

**1. Personalkosten:**
- Gehalt: 55.000 €
- AG-Anteil SV: 55.000 × 21% = 11.550 €
- Sonstige: 3.000 €
- **Gesamt: 69.550 €**

**2. Produktive Stunden:**
- Arbeitsstunden: 220 Tage × 8h = 1.760 Stunden
- Produktiv (65%): 1.760 × 0,65 = **1.144 Stunden**

**3. Stundensatz:**
- Personalkosten/Stunde: 69.550 € / 1.144 h = 60,79 €
- Gemeinkosten (70%): 60,79 € × 0,70 = 42,55 €
- Selbstkosten/Stunde: 60,79 € + 42,55 € = 103,34 €
- Gewinn (25%): 103,34 € × 0,25 = 25,84 €
- **Stundensatz netto: 129,18 €**

</details>

### Aufgabe 3: Rückwärtskalkulation

Der Bruttoverkaufspreis eines Servers soll 2.380 € betragen. Wie hoch darf der Einkaufspreis maximal sein, wenn:
- Bezugskosten: 50 €
- Handlungskostenzuschlag: 25%
- Gewinnzuschlag: 20%
- Umsatzsteuer: 19%

<details>
<summary>Lösung anzeigen</summary>

**Rückwärtsrechnung:**

Bruttoverkaufspreis: 2.380,00 €
÷ 1,19 (USt raus)
────────────────────────────
= Nettoverkaufspreis: 2.000,00 €
÷ 1,20 (Gewinn raus)
────────────────────────────
= Selbstkosten: 1.666,67 €
÷ 1,25 (Handlungskosten raus)
────────────────────────────
= Bezugspreis: 1.333,33 €
- Bezugskosten: -50,00 €
────────────────────────────
= **Maximaler Einkaufspreis: 1.283,33 €**

</details>
