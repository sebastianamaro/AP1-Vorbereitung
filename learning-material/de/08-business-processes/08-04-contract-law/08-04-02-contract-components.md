# Vertragsbestandteile

## Lernziele

Nach diesem Abschnitt kannst du:
- Das Zustandekommen eines Vertrags erklären
- Willenserklärung, Angebot und Annahme unterscheiden
- Wesentliche Vertragsbestandteile benennen
- Die Bedeutung von AGB erläutern

---

## Kerninhalt

### Zustandekommen eines Vertrags

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VERTRAGSSCHLUSS                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Ein Vertrag kommt durch zwei übereinstimmende                       │
│  WILLENSERKLÄRUNGEN zustande:                                       │
│                                                                      │
│           ANGEBOT                    ANNAHME                        │
│        (Willenserklärung 1)      (Willenserklärung 2)               │
│                                                                      │
│        ┌─────────────┐           ┌─────────────┐                    │
│        │             │           │             │                    │
│        │  Verkäufer  │ ────────► │   Käufer    │                    │
│        │  "Ich biete │           │ "Ich nehme  │                    │
│        │   an..."    │           │  an!"       │                    │
│        │             │           │             │                    │
│        └─────────────┘           └─────────────┘                    │
│                                                                      │
│                        VERTRAG                                       │
│                     entsteht!                                        │
│                                                                      │
│  Voraussetzungen:                                                   │
│  → Rechtsfähigkeit beider Parteien                                  │
│  → Geschäftsfähigkeit                                               │
│  → Übereinstimmung von Angebot und Annahme                          │
│  → Keine Form vorgeschrieben (meist formfrei)                       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Anfrage vs. Angebot vs. Bestellung

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ABGRENZUNG                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ANFRAGE                                                            │
│  → "Was kostet ein Laptop bei Ihnen?"                               │
│  → KEINE Willenserklärung                                           │
│  → Bindet niemanden                                                 │
│                                                                      │
│  ANGEBOT (Antrag)                                                   │
│  → "Der Laptop kostet 899 €, Lieferung in 3 Tagen"                 │
│  → Willenserklärung des ANBIETERS                                   │
│  → Bindet den Anbieter (Bindung an das Angebot)                     │
│                                                                      │
│  BESTELLUNG (Annahme)                                               │
│  → "Ich bestelle den Laptop zu diesen Bedingungen"                  │
│  → Willenserklärung des KÄUFERS                                     │
│  → Vertrag kommt zustande!                                          │
│                                                                      │
│  SONDERFALL: Bestellung ohne vorheriges Angebot                     │
│  → Bestellung = Angebot des Käufers                                 │
│  → Auftragsbestätigung = Annahme des Verkäufers                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Wesentliche Vertragsbestandteile

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VERTRAGSBESTANDTEILE                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ESSENTIALIA NEGOTII (wesentliche Bestandteile)                     │
│  ──────────────────────────────────────────────────────────────     │
│  → Müssen enthalten sein, sonst kein Vertrag!                       │
│                                                                      │
│  Bei Kaufvertrag:                                                   │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. Vertragsparteien (Wer?)                                 │    │
│  │  2. Kaufgegenstand (Was?)                                   │    │
│  │  3. Kaufpreis (Wie viel?)                                   │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  NATURALIA NEGOTII (natürliche Bestandteile)                        │
│  ──────────────────────────────────────────────────────────────     │
│  → Gelten automatisch, wenn nichts anderes vereinbart               │
│  → Z.B. Gewährleistung, Eigentumsübergang                           │
│                                                                      │
│  ACCIDENTALIA NEGOTII (zufällige Bestandteile)                      │
│  ──────────────────────────────────────────────────────────────     │
│  → Individuelle Vereinbarungen                                      │
│  → Z.B. Liefertermin, besondere Zahlungsbedingungen                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Inhalte eines IT-Vertrags

| Bestandteil | Beschreibung | Beispiel |
|-------------|--------------|----------|
| **Vertragsparteien** | Wer schließt den Vertrag? | ConSystem GmbH und Kunde AG |
| **Leistungsgegenstand** | Was wird geliefert? | 10 Laptops Modell XY |
| **Preis** | Wie viel kostet es? | 8.990 € netto |
| **Liefertermin** | Wann wird geliefert? | Innerhalb von 5 Werktagen |
| **Zahlungsbedingungen** | Wie wird bezahlt? | 2% Skonto bei 10 Tagen |
| **Gewährleistung** | Haftung für Mängel | 24 Monate |
| **Haftung** | Haftungsbeschränkungen | Max. Auftragswert |
| **AGB** | Allgemeine Bedingungen | Anlage zum Vertrag |

### Allgemeine Geschäftsbedingungen (AGB)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    AGB - ALLGEMEINE GESCHÄFTSBEDINGUNGEN             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Was sind AGB?                                                      │
│  → Vorformulierte Vertragsbedingungen                               │
│  → Gelten für viele Verträge                                        │
│  → "Kleingedrucktes"                                                │
│                                                                      │
│  Einbeziehung von AGB (§ 305 BGB):                                  │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  1. Ausdrücklicher Hinweis                                  │    │
│  │     → "Es gelten unsere AGB"                                │    │
│  │                                                              │    │
│  │  2. Möglichkeit der Kenntnisnahme                           │    │
│  │     → AGB müssen zugänglich sein (Aushang, Website)         │    │
│  │                                                              │    │
│  │  3. Einverständnis des Kunden                               │    │
│  │     → Kunde muss zustimmen                                  │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  Unwirksame Klauseln (§§ 307-309 BGB):                              │
│  → Überraschende Klauseln                                           │
│  → Unangemessene Benachteiligung                                    │
│  → Verstoß gegen gesetzliche Verbote                                │
│                                                                      │
│  Beispiel unwirksame IT-Klausel:                                    │
│  "Für alle Schäden durch unsere Software wird jegliche              │
│   Haftung ausgeschlossen." → Unwirksam bei Vorsatz/grober Fahrl.    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Formvorschriften

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FORMVORSCHRIFTEN                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  GRUNDSATZ: Vertragsfreiheit = Formfreiheit                         │
│  → Die meisten Verträge können mündlich geschlossen werden          │
│                                                                      │
│  AUSNAHMEN (gesetzlich vorgeschrieben):                             │
│  ──────────────────────────────────────────────────────────────     │
│                                                                      │
│  Schriftform (§ 126 BGB):                                           │
│  → Unterschrift erforderlich                                        │
│  → Z.B. Bürgschaften, Arbeitszeugnisse                              │
│                                                                      │
│  Textform (§ 126b BGB):                                             │
│  → Lesbare Erklärung, Person erkennbar                              │
│  → Z.B. E-Mail                                                      │
│  → Z.B. Widerrufsbelehrung im Online-Handel                         │
│                                                                      │
│  Notarielle Beurkundung:                                            │
│  → Notar erforderlich                                               │
│  → Z.B. Grundstückskauf, GmbH-Gründung                              │
│                                                                      │
│  IT-VERTRÄGE: Meist formfrei!                                       │
│  → Aber: Schriftform zur Beweissicherung empfohlen                  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Willenserklärung** | Äußerung eines rechtlich relevanten Willens |
| **Angebot** | Verbindliche Willenserklärung des Anbieters |
| **Annahme** | Zustimmung zum Angebot |
| **Essentialia negotii** | Wesentliche Vertragsbestandteile |
| **AGB** | Allgemeine Geschäftsbedingungen |
| **Formfreiheit** | Vertrag kann in beliebiger Form geschlossen werden |
| **Textform** | Schriftliche Form ohne Unterschrift (z.B. E-Mail) |

---

## Prüfungstipps

1. **Vertragsschluss merken:**
   - Angebot + Annahme = Vertrag

2. **AGB-Einbeziehung:**
   - Hinweis + Kenntnismöglichkeit + Zustimmung

3. **Typische Prüfungsfrage:**
   - Wann kommt ein Vertrag zustande?
   - Was gehört in einen Kaufvertrag?

## Übungsaufgaben

### Aufgabe 1: Vertragsschluss

Die ConSystem GmbH sendet einem Kunden ein Angebot über 5 Server für 15.000 €. Der Kunde antwortet per E-Mail: "Wir bestellen die Server."

a) Ist ein Vertrag zustanden gekommen?
b) Welche Willenserklärungen liegen vor?

<details>
<summary>Lösung anzeigen</summary>

a) **Ja, ein Vertrag ist zustande gekommen.**

b) **Willenserklärungen:**
- Angebot von ConSystem: "5 Server für 15.000 €" (bindet ConSystem)
- Annahme durch Kunden: "Wir bestellen die Server" (bindet den Kunden)
- Beide Willenserklärungen stimmen überein → Vertrag

</details>

### Aufgabe 2: AGB-Einbeziehung

Ein Kunde bestellt online Software bei ConSystem. Auf der Website steht klein: "Es gelten unsere AGB". Die AGB sind nur über einen kleinen Link am Seitenende erreichbar.

Sind die AGB wirksam einbezogen?

<details>
<summary>Lösung anzeigen</summary>

**Prüfung der Einbeziehungsvoraussetzungen:**

1. **Hinweis:** Vorhanden ("Es gelten unsere AGB")
2. **Kenntnismöglichkeit:** Fraglich - Link sehr klein und versteckt
3. **Zustimmung:** Unklar - wurde der Kunde vor Bestellabschluss auf AGB hingewiesen?

**Ergebnis:** Die AGB könnten unwirksam einbezogen sein, wenn:
- Der Link nicht gut sichtbar ist
- Kein Häkchen für "AGB gelesen" erforderlich war
- Die AGB nicht vor Vertragsschluss einsehbar waren

**Empfehlung:** AGB-Link gut sichtbar platzieren und Checkbox zur Bestätigung einbauen.

</details>

### Aufgabe 3: Wesentliche Bestandteile

Prüfen Sie, ob folgende Angaben für einen gültigen Kaufvertrag über Server ausreichen:

"ConSystem liefert Server an Firma Müller"

<details>
<summary>Lösung anzeigen</summary>

**Analyse der wesentlichen Bestandteile (Essentialia):**

| Bestandteil | Vorhanden? | Bemerkung |
|-------------|------------|-----------|
| Vertragsparteien | ✓ | ConSystem und Firma Müller |
| Kaufgegenstand | ⚠️ | "Server" - aber welche? Menge? Typ? |
| Kaufpreis | ✗ | Fehlt komplett! |

**Ergebnis:** Der Vertrag ist **nicht gültig**, da:
- Der Kaufgegenstand nicht genau bestimmt ist
- Der Kaufpreis fehlt

**Korrekt wäre:**
"ConSystem liefert 3 Server vom Typ HP ProLiant DL380 Gen10 an Firma Müller für insgesamt 12.000 € netto."

</details>
