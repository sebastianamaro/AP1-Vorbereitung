# Vertriebsformen

## Lernziele

Nach diesem Abschnitt kannst du:
- Verschiedene Vertriebsformen unterscheiden
- Vor- und Nachteile der Vertriebswege erklären
- E-Commerce-Modelle benennen
- Geeignete Vertriebsformen für IT-Produkte identifizieren

---

## Kerninhalt

### Grundlegende Vertriebsformen

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VERTRIEBSWEGE                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  DIREKTVERTRIEB                   INDIREKTER VERTRIEB               │
│  (ohne Zwischenhändler)           (mit Zwischenhändler)             │
│                                                                      │
│  ┌──────────┐                     ┌──────────┐                      │
│  │Hersteller│                     │Hersteller│                      │
│  └────┬─────┘                     └────┬─────┘                      │
│       │                                │                            │
│       │                           ┌────┴─────┐                      │
│       │                           │Großhandel│                      │
│       │                           └────┬─────┘                      │
│       │                                │                            │
│       │                           ┌────┴─────┐                      │
│       │                           │Einzelhandel│                    │
│       │                           └────┬─────┘                      │
│       │                                │                            │
│       ▼                                ▼                            │
│  ┌──────────┐                     ┌──────────┐                      │
│  │  Kunde   │                     │  Kunde   │                      │
│  └──────────┘                     └──────────┘                      │
│                                                                      │
│  Beispiele:                       Beispiele:                        │
│  → Online-Shop des               → MediaMarkt, Saturn               │
│    Herstellers                   → PC-Händler vor Ort               │
│  → Außendienstmitarbeiter        → Systemhäuser                     │
│  → Werksverkauf                  → Online-Marktplätze               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Vergleich: Direkt vs. Indirekt

| Aspekt | Direktvertrieb | Indirekter Vertrieb |
|--------|----------------|---------------------|
| **Zwischenhändler** | Keine | Groß-/Einzelhandel |
| **Marge** | Höher | Niedriger (Händlermarge) |
| **Kundenkontakt** | Direkt | Über Händler |
| **Reichweite** | Begrenzt | Groß |
| **Investition** | Hoch (eigener Vertrieb) | Niedriger |
| **Kontrolle** | Hoch | Eingeschränkt |

### Vertriebsformen in der IT

| Vertriebsform | Beschreibung | IT-Beispiel |
|---------------|--------------|-------------|
| **Online-Shop** | Direkter Verkauf über Internet | Dell.de, apple.com |
| **Systemhaus** | Beratung + Verkauf + Service | Lokale IT-Dienstleister |
| **Fachhandel** | Spezialisierter Händler | Computergeschäft |
| **Marktplatz** | Plattform für viele Anbieter | Amazon, eBay |
| **Reseller** | Wiederverkäufer | Softwarelizenzhändler |
| **Großhandel** | Verkauf an Händler | Also, Ingram Micro |

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Direktvertrieb** | Verkauf ohne Zwischenhändler |
| **Indirekter Vertrieb** | Verkauf über Händler |
| **B2B** | Business-to-Business (Firmen an Firmen) |
| **B2C** | Business-to-Consumer (Firmen an Endkunden) |
| **C2C** | Consumer-to-Consumer (Privat an Privat) |
| **E-Commerce** | Elektronischer Handel über Internet |
| **Omnichannel** | Verknüpfung mehrerer Vertriebskanäle |
| **Reseller** | Wiederverkäufer |
| **SaaS** | Software as a Service (Abo-Modell) |

---

## Prüfungstipps

1. **Abkürzungen merken:**
   - B2B = **B**usiness to **B**usiness
   - B2C = **B**usiness to **C**onsumer
   - C2C = **C**onsumer to **C**onsumer

2. **Typische Prüfungsfragen:**
   - Welche Vertriebsform für IT-Dienstleistungen?
   - Vor-/Nachteile von Direktvertrieb
   - E-Commerce-Modelle zuordnen

3. **Wichtig für IT:**
   - Systemhaus = Indirekter Vertrieb mit Mehrwert (Beratung)
   - SaaS = Moderner Software-Vertrieb

## Übungsaufgaben

### Aufgabe 1: E-Commerce-Modelle

Ordnen Sie folgende Beispiele dem richtigen E-Commerce-Modell zu:

a) Ein Softwarehersteller verkauft Lizenzen an Unternehmen
b) Ein Privatverkäufer bietet seinen gebrauchten PC auf eBay an
c) Ein Online-Shop verkauft Elektronik an Endverbraucher
d) Eine Behörde beschafft neue Computer über eine Ausschreibung

<details>
<summary>Lösung anzeigen</summary>

a) **B2B** (Business-to-Business) - Unternehmen verkauft an Unternehmen
b) **C2C** (Consumer-to-Consumer) - Privat an Privat
c) **B2C** (Business-to-Consumer) - Unternehmen an Endverbraucher
d) **B2G** (Business-to-Government) - Unternehmen an öffentliche Hand

</details>

### Aufgabe 2: Vertriebswegentscheidung

Die ConSystem GmbH ist ein mittelständisches IT-Systemhaus und möchte seinen Vertrieb optimieren. Derzeit werden Kunden ausschließlich über den Außendienst betreut.

a) Nennen Sie zwei Vorteile des aktuellen Direktvertriebs.
b) Welche zusätzlichen Vertriebskanäle könnten sinnvoll sein?
c) Was bedeutet "Omnichannel" in diesem Zusammenhang?

<details>
<summary>Lösung anzeigen</summary>

a) **Vorteile Direktvertrieb:**
- Direkter Kundenkontakt und persönliche Beratung
- Höhere Marge (keine Händlerprovision)
- Kontrolle über das Kundenerlebnis
- Direkte Rückmeldung vom Kunden

b) **Zusätzliche Vertriebskanäle:**
- Online-Shop für Standardprodukte (Hardware, Lizenzen)
- Partnerprogramm mit anderen Dienstleistern
- Webinare und Online-Beratung
- Social Media für Leadgenerierung
- Kooperation mit Großhändlern

c) **Omnichannel:**
Integration aller Vertriebskanäle zu einem einheitlichen Kundenerlebnis. Der Kunde kann:
- Online recherchieren
- Telefonisch beraten werden
- Im Außendienst den Vertrag abschließen
- Support über alle Kanäle erhalten
- Alle Informationen sind verknüpft

</details>

### Aufgabe 3: Software-Vertriebsmodelle

Erklären Sie den Unterschied zwischen folgenden Software-Vertriebsmodellen:

a) Lizenzverkauf (klassisch)
b) SaaS (Software as a Service)
c) Freemium

<details>
<summary>Lösung anzeigen</summary>

a) **Lizenzverkauf (klassisch):**
- Einmalige Zahlung für dauerhafte Nutzung
- Software wird auf eigenen Systemen installiert
- Updates oft kostenpflichtig
- Beispiel: Microsoft Office 2021 (Kaufversion)

b) **SaaS (Software as a Service):**
- Regelmäßige Zahlung (monatlich/jährlich)
- Software läuft in der Cloud
- Updates automatisch inklusive
- Beispiel: Microsoft 365, Salesforce

c) **Freemium:**
- Basisversion kostenlos
- Premium-Funktionen kostenpflichtig
- Ziel: Nutzer zur kostenpflichtigen Version konvertieren
- Beispiel: Dropbox (2 GB kostenlos), Spotify

</details>
