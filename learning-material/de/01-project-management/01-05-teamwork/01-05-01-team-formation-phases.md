# Teambildungsphasen (Tuckman-Modell)

## Lernziele

Nach diesem Kapitel solltest du:
- Die vier Teambildungsphasen nach Tuckman kennen
- Die Charakteristika jeder Phase beschreiben können
- Verstehen, wie Teams sich entwickeln
- Die Rolle der Führung in den verschiedenen Phasen verstehen

---

## Kerninhalt

### Das Tuckman-Modell

Das **Tuckman-Modell** beschreibt die Entwicklung von Teams in vier Phasen. Es wurde 1965 von Bruce Tuckman entwickelt und ist das bekannteste Modell zur Teamentwicklung.

---

### Die vier Phasen

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  FORMING    │ ─► │  STORMING   │ ─► │  NORMING    │ ─► │ PERFORMING  │
│             │    │             │    │             │    │             │
│ Kennenlernen│    │  Konflikte  │    │   Regeln    │    │  Leistung   │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

---

### Phase 1: Forming (Orientierung)

**Charakteristik:** Kennenlernen und erste Orientierung

| Aspekt | Beschreibung |
|--------|-------------|
| **Verhalten** | Höflich, vorsichtig, abwartend |
| **Fokus** | Sich kennenlernen, Ziele verstehen |
| **Gefühle** | Unsicherheit, Neugier, Erwartung |
| **Produktivität** | Niedrig |
| **Konflikte** | Werden vermieden |

**Aufgaben der Führung:**
- Klare Ziele kommunizieren
- Struktur geben
- Kennenlernen fördern
- Orientierung bieten

---

### Phase 2: Storming (Konfrontation)

**Charakteristik:** Erste Konflikte und Auseinandersetzungen

| Aspekt | Beschreibung |
|--------|-------------|
| **Verhalten** | Positionskämpfe, Widerstand, Kritik |
| **Fokus** | Rollen klären, Machtverhältnisse |
| **Gefühle** | Frustration, Enttäuschung, Konkurrenz |
| **Produktivität** | Niedrig bis mittel |
| **Konflikte** | Treten offen auf |

**Aufgaben der Führung:**
- Konflikte moderieren
- Klare Regeln einführen
- Rollen definieren
- Geduld haben

**Wichtig:** Diese Phase ist normal und notwendig für die Teamentwicklung!

---

### Phase 3: Norming (Organisierung)

**Charakteristik:** Entwicklung gemeinsamer Regeln und Normen

| Aspekt | Beschreibung |
|--------|-------------|
| **Verhalten** | Kooperativ, kompromissbereit |
| **Fokus** | Gemeinsame Standards, Arbeitsweisen |
| **Gefühle** | Zugehörigkeit, Vertrauen |
| **Produktivität** | Steigt deutlich |
| **Konflikte** | Werden konstruktiv gelöst |

**Aufgaben der Führung:**
- Teamregeln unterstützen
- Delegation fördern
- Eigenverantwortung stärken
- Feedback geben

---

### Phase 4: Performing (Leistung)

**Charakteristik:** Effektive, zielgerichtete Zusammenarbeit

| Aspekt | Beschreibung |
|--------|-------------|
| **Verhalten** | Selbstständig, flexibel, unterstützend |
| **Fokus** | Aufgabenerfüllung, kontinuierliche Verbesserung |
| **Gefühle** | Stolz, Zufriedenheit, Verbundenheit |
| **Produktivität** | Hoch |
| **Konflikte** | Selten, schnell gelöst |

**Aufgaben der Führung:**
- Delegieren
- Rahmenbedingungen sichern
- Team feiern lassen
- Entwicklung unterstützen

---

### Übersicht der Phasen

| Phase | Kennzeichen | Team-Fokus | Führungsstil |
|-------|-------------|------------|--------------|
| **Forming** | Unsicherheit | Orientierung | Direktiv |
| **Storming** | Konflikte | Klärung | Moderierend |
| **Norming** | Regeln | Organisation | Partizipativ |
| **Performing** | Leistung | Ergebnisse | Delegativ |

---

### Dynamik im Modell

**Wichtige Hinweise:**
- Phasen können unterschiedlich lange dauern
- Teams können in frühere Phasen zurückfallen (z.B. bei neuen Mitgliedern)
- Nicht alle Teams erreichen die Performing-Phase
- Bei Scrum: Jeder Sprint kann Elemente aller Phasen enthalten

## Wichtige Begriffe

| Begriff | Erklärung |
|---------|-----------|
| **Forming** | Orientierungsphase, Kennenlernen |
| **Storming** | Konfliktphase, Positionskämpfe |
| **Norming** | Organisationsphase, Regeln entwickeln |
| **Performing** | Leistungsphase, effektive Zusammenarbeit |
| **Tuckman-Modell** | Modell der Teamentwicklung in vier Phasen |

- Beschreiben Sie die Storming-Phase
- In welcher Phase ist die Produktivität am höchsten?

---

## Übungsaufgaben

### Aufgabe 1
Nennen Sie die vier Teambildungsphasen nach Tuckman in der richtigen Reihenfolge.

**Lösung:**
1. Forming (Orientierung)
2. Storming (Konfrontation)
3. Norming (Organisierung)
4. Performing (Leistung)

### Aufgabe 2
In welcher Phase treten typischerweise die meisten Konflikte auf?

**Lösung:** In der Storming-Phase (Phase 2)

### Aufgabe 3
Ein neues Teammitglied kommt hinzu. Welche Auswirkung kann das auf die Teamdynamik haben?

**Lösung:** Das Team kann in eine frühere Phase (meist Forming oder Storming) zurückfallen, da die Gruppendynamik neu ausgehandelt werden muss.
