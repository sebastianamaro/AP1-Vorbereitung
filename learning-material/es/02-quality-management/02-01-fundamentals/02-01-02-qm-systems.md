# Sistemas de Gestión de Calidad (SGC)

## Objetivos de Aprendizaje

Después de este capítulo usted debería:
- Saber qué es un sistema de gestión de calidad
- Conocer los componentes de un SGC
- Ser capaz de describir los pasos para implementar un SGC
- Conocer la diferencia entre modelos de SGC

---

## Contenido Principal

### ¿Qué es un SGC?

Un **Sistema de Gestión de Calidad (SGC)** es un enfoque estructurado para asegurar y mejorar continuamente la calidad en las organizaciones. Abarca:

- **Procesos:** Flujos de trabajo definidos
- **Políticas:** Reglas y directrices
- **Responsabilidades:** Rendición de cuentas clara
- **Documentación:** Registros trazables

#### Los Cinco Bloques de Construcción en Detalle

| Bloque | Descripción | Ejemplo |
|--------|-------------|---------|
| **Política de Calidad** | Dirección e intenciones fundamentales | "Nos esforzamos por el 100% de satisfacción del cliente" |
| **Objetivos de Calidad** | Metas concretas y medibles | "Tasa de error inferior al 0.1%" |
| **Estructura Organizacional** | Estructuras y responsabilidades | Organigrama, representante de GC |
| **Procesos Operativos** | Procesos y flujos de trabajo definidos | Descripciones de procesos, instrucciones de trabajo |
| **Gestión de Recursos** | Provisión de los recursos necesarios | Personal, sistemas de TI, presupuesto |

### Implementación de un SGC

La implementación de un SGC se realiza en varios pasos:

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 1. ANÁLISIS │ ─► │ 2. ESTADO   │ ─► │ 3. MANUAL   │
│    DEL ESTADO│    │    OBJETIVO │    │    DE GC    │
│    ACTUAL   │    │             │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 6. CERTI-   │ ◄─ │ 5. AUDITORÍAS│ ◄─ │ 4. FORMACIÓN│
│    FICACIÓN │    │    INTERNAS │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
```

| Paso | Actividades |
|------|-------------|
| **1. Análisis del Estado Actual** | Capturar procesos y estructuras actuales |
| **2. Definir Estado Objetivo** | Establecer objetivos y normas de calidad |
| **3. Crear Manual de GC** | Documentar procesos, establecer responsabilidades |
| **4. Formar a los Empleados** | Familiarizar a todas las partes involucradas con el SGC |
| **5. Auditorías Internas** | Verificar el cumplimiento de los requisitos |
| **6. Certificación** | Examen externo por organismo de certificación |

---

### Comparación de Modelos de SGC

| Aspecto | DIN EN ISO 9001 | Modelo EFQM |
|---------|-----------------|-------------|
| **Origen** | Norma internacional | Fundación Europea para la Gestión de la Calidad |
| **Enfoque** | Orientado a requisitos | Autoevaluación y excelencia |
| **Certificación** | Sí, por auditores | No, autoevaluación |
| **Foco** | Procesos y documentación | Gestión empresarial integral |
| **Filosofía** | Cumplir requisitos mínimos | Esforzarse por la excelencia |

---

### El Manual de GC

El **Manual de GC** es el documento central de un SGC:

**Contenidos típicos:**
- Política y objetivos de calidad
- Estructura organizacional
- Descripciones de procesos
- Responsabilidades
- Instrucciones de procedimientos
- Formularios y listas de verificación

## Términos Clave

| Término | Explicación |
|---------|-------------|
| **SGC** | Sistema de Gestión de Calidad - enfoque estructurado para el aseguramiento de la calidad |
| **Política de Calidad** | Dirección fundamental de una organización respecto a la calidad |
| **Objetivos de Calidad** | Especificaciones concretas y medibles para la calidad |
| **Manual de GC** | Documento central con toda la información del SGC |
| **Auditoría** | Revisión sistemática del cumplimiento del SGC |
| **Estructura Organizacional** | Estructura jerárquica (quién es responsable de qué) |
| **Procesos Operativos** | Estructura de procesos (cómo se completan las tareas) |

## Ejercicios Prácticos

### Ejercicio 1
Ordene los siguientes pasos para implementar un SGC:
- Certificación
- Formación de empleados
- Análisis del estado actual
- Auditorías internas
- Redacción del manual de GC

**Solución:**
1. Análisis del estado actual
2. Redacción del manual de GC
3. Formación de empleados
4. Auditorías internas
5. Certificación

### Ejercicio 2
Defina los términos planificación de calidad y política de calidad.

**Solución:**
- **Planificación de calidad:** Establecimiento de objetivos de calidad y medidas para lograr estos objetivos
- **Política de calidad:** Las intenciones y dirección fundamentales de una organización respecto a la calidad, formuladas por la alta dirección

### Ejercicio 3
Nombre tres componentes de un SGC y explíquelos brevemente.

**Solución:**
1. **Política de calidad:** Dirección fundamental de calidad de la organización
2. **Estructura organizacional:** Estructuras, roles y responsabilidades
3. **Procesos operativos:** Procesos y flujos de trabajo definidos
