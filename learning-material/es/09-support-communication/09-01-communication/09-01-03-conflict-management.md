# Gestión de Conflictos en Soporte TI

## Objetivos de Aprendizaje

Después de esta sección, podrás:
- Reconocer situaciones de conflicto en soporte TI
- Aplicar técnicas de desescalada
- Tratar con usuarios difíciles
- Manejar quejas profesionalmente

### Técnicas de Desescalada

```
┌─────────────────────────────────────────────────────────────────────┐
│           DESESCALADA                                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  1. ESCUCHAR Y DEJAR TERMINAR                                       │
│     ───────────────────────────────────────────────────────────     │
│     → Dejar que el usuario "se desahogue" primero                   │
│     → No interrumpir                                                │
│     → Escuchar activamente (asentir, "Mm-hmm", "Entiendo")          │
│     → Tomar notas                                                   │
│                                                                      │
│  2. MOSTRAR COMPRENSIÓN                                             │
│     ───────────────────────────────────────────────────────────     │
│     → "Entiendo que estás frustrado"                                │
│     → "Eso es realmente molesto"                                    │
│     → "Yo también estaría molesto en tu posición"                   │
│     → IMPORTANTE: Comprensión ≠ Admitir culpa                       │
│                                                                      │
│  3. RESUMIR EL PROBLEMA                                             │
│     ───────────────────────────────────────────────────────────     │
│     → "Si te entiendo correctamente, el problema es..."             │
│     → Demuestra que escuchaste                                      │
│     → Asegura que entendiste correctamente                          │
│                                                                      │
│  4. OFRECER SOLUCIÓN                                                │
│     ───────────────────────────────────────────────────────────     │
│     → Nombrar próximos pasos concretos                              │
│     → Dar estimaciones de tiempo realistas                          │
│     → Asumir responsabilidad                                        │
│     → "Yo personalmente me encargaré de esto"                       │
│                                                                      │
│  5. CREAR COMPROMISO                                                │
│     ───────────────────────────────────────────────────────────     │
│     → Proporcionar número de ticket                                 │
│     → Programar llamada de retorno                                  │
│     → Dar nombre (principio SPOC)                                   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### El Principio CALM

```
┌─────────────────────────────────────────────────────────────────────┐
│           EL PRINCIPIO CALM                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  C - CONTROL (Mantener control)                                     │
│      → Mantener la calma, no tomarlo personalmente                  │
│      → Controlar tus propias emociones                              │
│      → Mantener voz calmada y amable                                │
│                                                                      │
│  A - ACKNOWLEDGE (Reconocer)                                        │
│      → Reconocer el problema                                        │
│      → Reconocer los sentimientos del usuario                       │
│      → "Entiendo tu situación"                                      │
│                                                                      │
│  L - LISTEN (Escuchar)                                              │
│      → Escuchar activamente                                         │
│      → No interrumpir                                               │
│      → Hacer preguntas de seguimiento                               │
│                                                                      │
│  M - MANAGE (Gestionar)                                             │
│      → Gestionar expectativas                                       │
│      → Ofrecer soluciones realistas                                 │
│      → Prometer seguimiento                                         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────────┐
│           TIPOS DE USUARIO                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  EL IMPACIENTE                                                      │
│  ──────────────────────────────────────────────────────────────     │
│  Comportamiento: Quiere solución inmediata, interrumpe, impaciente  │
│  Estrategia:                                                        │
│  → Ir al grano rápidamente                                          │
│  → Sin explicaciones largas                                         │
│  → Hacer compromisos de tiempo                                      │
│  → "Me encargaré de ello inmediatamente"                            │
│                                                                      │
│  EL ENOJADO                                                         │
│  ──────────────────────────────────────────────────────────────     │
│  Comportamiento: Agresivo, acusador, ruidoso                        │
│  Estrategia:                                                        │
│  → Mantener calma, no gritar de vuelta                              │
│  → Dejarlo terminar                                                 │
│  → Mostrar comprensión                                              │
│  → Volver al nivel factual                                          │
│                                                                      │
│  EL DESVALIDO                                                       │
│  ──────────────────────────────────────────────────────────────     │
│  Comportamiento: Abrumado, inseguro, necesita mucha guía            │
│  Estrategia:                                                        │
│  → Ser paciente                                                     │
│  → Explicar paso a paso                                             │
│  → Sin jerga técnica                                                │
│  → Animar y elogiar                                                 │
│                                                                      │
│  EL SABELOTODO                                                      │
│  ──────────────────────────────────────────────────────────────     │
│  Comportamiento: Cree saber todo mejor, argumenta                   │
│  Estrategia:                                                        │
│  → Tomar en serio                                                   │
│  → Mantenerse técnicamente correcto                                 │
│  → Reconocer experiencia donde sea apropiado                        │
│  → No entrar en luchas de poder                                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Gestión de Quejas

```
┌─────────────────────────────────────────────────────────────────────┐
│           MANEJO DE QUEJAS                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  PASO 1: RECIBIR                                                    │
│  ──────────────────────────────────────────────────────────────     │
│  → Tomar la queja en serio                                          │
│  → Documentar (ticket)                                              │
│  → Proporcionar confirmación de recepción                           │
│                                                                      │
│  PASO 2: INVESTIGAR                                                 │
│  ──────────────────────────────────────────────────────────────     │
│  → Aclarar los hechos                                               │
│  → Leer historial de tickets                                        │
│  → Consultar con colegas si es necesario                            │
│                                                                      │
│  PASO 3: RESPONDER                                                  │
│  ──────────────────────────────────────────────────────────────     │
│  → Responder prontamente                                            │
│  → Comunicar resultados de la investigación                         │
│  → Para queja justificada: Disculpa + Solución                      │
│  → Para queja injustificada: Explicar factualmente                  │
│                                                                      │
│  PASO 4: SEGUIMIENTO                                                │
│  ──────────────────────────────────────────────────────────────     │
│  → ¿El usuario quedó satisfecho?                                    │
│  → ¿Se resolvió el problema?                                        │
│  → ¿Qué podemos mejorar?                                            │
│                                                                      │
│  IMPORTANTE: ¡Las quejas son OPORTUNIDADES de mejora!               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Establecer Límites

```
┌─────────────────────────────────────────────────────────────────────┐
│           ESTABLECER LÍMITES                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  PARA ATAQUES PERSONALES:                                           │
│  ──────────────────────────────────────────────────────────────     │
│  → "Entiendo que estás molesto, pero te pido que                    │
│     permanezcas en lo factual."                                     │
│  → "Me gustaría ayudarte, pero por favor en un                      │
│     tono respetuoso."                                               │
│                                                                      │
│  PARA INSULTOS:                                                     │
│  ──────────────────────────────────────────────────────────────     │
│  → "No puedo continuar la conversación de esta manera."             │
│  → "Una vez que te hayas calmado, con gusto te llamo de nuevo."     │
│  → En casos extremos: Terminar conversación (después de anuncio)    │
│                                                                      │
│  PARA DEMANDAS POCO REALISTAS:                                      │
│  ──────────────────────────────────────────────────────────────     │
│  → "Desafortunadamente, eso no es posible porque..."                │
│  → "Lo que puedo ofrecerte es..."                                   │
│  → Mostrar alternativas                                             │
│                                                                      │
│  PARA ESCALACIÓN A SUPERVISOR:                                      │
│  ──────────────────────────────────────────────────────────────     │
│  → No verlo como fracaso                                            │
│  → A veces transferir es la mejor solución                          │
│  → Entregar documentación                                           │
│  → No "despachar", transferir activamente                           │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Ayuda para Formulación

| Situación | No Digas | Mejor Decir |
|-----------|----------|-------------|
| Usuario está enojado | "¡Cálmate!" | "Entiendo tu frustración" |
| Problema toma tiempo | "No sé cuánto tiempo" | "Te contactaré en 30 minutos" |
| No responsable | "Yo no soy responsable de eso" | "Te conectaré con mi colega" |
| Ocurrió error | "No puedo hacer nada al respecto" | "Lo siento, me encargaré de ello" |
| Demanda imposible | "Eso no es posible" | "Lo que puedo hacer es..." |

## Consejos para el Examen

1. **Recordar principio básico:**
   - Primero mostrar comprensión, luego ofrecer solución
   - Nunca tomarlo personalmente
   - Siempre mantenerse factual y amable

2. **Principio CALM:**
   - Control, Acknowledge, Listen, Manage
   - Buena ayuda de memoria para exámenes

3. **Preguntas típicas de examen:**
   - "¿Cómo respondes a un cliente molesto?"
   - "¿Qué técnicas de desescalada conoces?"

## Ejercicios Prácticos

### Ejercicio 1: Formular Respuesta

Un usuario llama enojado: "¡Esta es la tercera vez esta semana que llamo! ¡Tu TI es realmente incompetente! ¡Necesito trabajar y mi PC no funciona de nuevo!"

Formula una respuesta de desescalada.

<details>
<summary>Mostrar Solución</summary>

**Respuesta profesional:**

"Entiendo completamente que estés molesto cuando el problema sigue recurriendo - yo también lo estaría en tu posición. [Mostrar comprensión]

Déjame revisar rápidamente tu historial de tickets para ver qué se ha intentado hasta ahora. [Nivel factual]

Quiero resolver el problema de manera sostenible esta vez y me encargaré personalmente de ello. [Compromiso]

¿Puedes describir brevemente qué está pasando exactamente ahora mismo? [Capturar problema específicamente]

No pararé hasta que tengamos una solución permanente." [Compromiso]

</details>

### Ejercicio 2: Establecer Límites

Un usuario dice: "¡Todos ustedes son incompetentes! ¡Quiero hablar con tu jefe inmediatamente!"

¿Cómo respondes?

<details>
<summary>Mostrar Solución</summary>

**Respuesta en dos pasos:**

**1. Establecer límite (amable pero firme):**
"Entiendo que estás insatisfecho, y quiero ayudarte. Sin embargo, te pido que permanezcas factual para que podamos encontrar una solución juntos."

**2. Hacer una oferta:**
"Me gustaría intentar resolver tu problema directamente primero. Si todavía quieres hablar con mi supervisor después, con gusto puedo arreglar eso. ¿Cuál es exactamente el problema?"

**Si sigue insultando:**
"Estoy feliz de ayudarte, pero solo en una conversación respetuosa. Ahora estoy transfiriendo tu caso a mi líder de equipo, quien te contactará. ¿Tienes un momento?"

**Importante:** La escalación a supervisor no es una derrota - a veces es la mejor solución para todos los involucrados.

</details>

### Ejercicio 3: Aplicar Principio CALM

Asigna las siguientes declaraciones a la letra CALM correcta:

a) "Te contactaré en una hora"
b) Mantener la calma y ser factual aunque el usuario esté ruidoso
c) "Entiendo que esto es molesto para ti"
d) Dejar que el usuario termine de hablar primero

<details>
<summary>Mostrar Solución</summary>

a) **M - Manage** - Establecer expectativas y prometer seguimiento
b) **C - Control** - Controlar las propias emociones
c) **A - Acknowledge** - Reconocer problema y sentimientos
d) **L - Listen** - Escuchar activamente

</details>
