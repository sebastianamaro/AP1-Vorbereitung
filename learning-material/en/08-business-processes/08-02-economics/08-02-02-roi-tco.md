# ROI and TCO

## Learning Objectives

After this section, you will be able to:
- Calculate and interpret Return on Investment (ROI)
- Determine Total Cost of Ownership (TCO)
- Apply both metrics to IT investments
- Evaluate the economic viability of IT projects

---

## Core Content

### Return on Investment (ROI)

**ROI** measures the return on an investment - how much profit is generated relative to the capital invested.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ROI - RETURN ON INVESTMENT                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                         Profit                                       │
│         ROI  =  ─────────────────────  x  100%                       │
│                    Investment                                        │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  Or alternatively:                                                   │
│                                                                      │
│               Return  -  Investment                                  │
│         ROI  =  ─────────────────────────  x  100%                   │
│                      Investment                                      │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  EXAMPLE:                                                           │
│                                                                      │
│  Investment in new servers: $50,000                                 │
│  Savings through efficiency: $15,000/year                           │
│  Observation period: 3 years                                        │
│                                                                      │
│  Profit = ($15,000 x 3) - $50,000 = $45,000 - $50,000 = -$5,000    │
│                                                                      │
│  OR: Total return = $45,000                                         │
│                                                                      │
│         $45,000 - $50,000     -$5,000                               │
│  ROI = ─────────────────────  = ────────── = -10%                   │
│              $50,000            $50,000                              │
│                                                                      │
│  → Investment does NOT pay off in 3 years yet                       │
│                                                                      │
│  At 4 years: ($15,000 x 4) - $50,000 = $10,000                     │
│  ROI = $10,000 / $50,000 = +20%                                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### ROI Interpretation

| ROI Value | Meaning |
|-----------|---------|
| **> 0%** | Investment generates profit |
| **= 0%** | Investment is cost-neutral |
| **< 0%** | Investment generates loss |
| **> 20%** | Good return (depends on industry) |

### Payback Period (Amortization)

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PAYBACK PERIOD                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│                        Investment                                    │
│  Payback Period  =  ─────────────────────────                        │
│                     Annual Savings                                   │
│                                                                      │
│  ────────────────────────────────────────────────────────────────   │
│                                                                      │
│  EXAMPLE:                                                           │
│                                                                      │
│  Investment: $50,000                                                │
│  Annual savings: $15,000                                            │
│                                                                      │
│                     $50,000                                          │
│  Payback Period = ───────── = 3.33 years                            │
│                     $15,000                                          │
│                                                                      │
│  → After 3 years and 4 months, the investment has paid for itself  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Total Cost of Ownership (TCO)

**TCO** captures ALL costs that arise over the entire useful life of a product or system.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TCO - TOTAL COST OF OWNERSHIP                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  TCO = Acquisition Costs + Operating Costs + Disposal Costs         │
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │     ACQUISITION          OPERATION           END             │    │
│  │     ───────────          ─────────           ───             │    │
│  │                                                              │    │
│  │     • Purchase price     • Maintenance       • Disposal      │    │
│  │     • Installation       • Updates           • Data deletion │    │
│  │     • Training           • Support           • Dismantling   │    │
│  │     • Configuration      • Energy            • Migration     │    │
│  │     • Data migration     • Consumables                       │    │
│  │                          • Repairs                           │    │
│  │                          • Licenses                          │    │
│  │                                                              │    │
│  │  ←────────────────── Useful Life ─────────────────────→      │    │
│  │         (typically 3-5 years for IT hardware)                │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                      │
│  IMPORTANT: The purchase price is often only 20-30% of TCO!         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### TCO Example: PC Workstation

```
┌─────────────────────────────────────────────────────────────────────┐
│              TCO EXAMPLE: PC WORKSTATION (5 years)                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ACQUISITION COSTS (one-time):                                      │
│  ├── Hardware (PC, monitor, peripherals)       $1,200               │
│  ├── Software licenses                           $300               │
│  ├── Installation and configuration              $150               │
│  └── Employee training                           $200               │
│                                          ──────────                 │
│                               Total:          $1,850                │
│                                                                      │
│  OPERATING COSTS (per year x 5 years):                              │
│  ├── Maintenance and support                     $300               │
│  ├── Software updates                            $100               │
│  ├── Energy costs                                 $80               │
│  ├── Consumables                                  $50               │
│  └── Allocated IT personnel costs                $400               │
│                                          ──────────                 │
│                        Per year:              $930                  │
│                        x 5 years:           $4,650                  │
│                                                                      │
│  DISPOSAL COSTS:                                                    │
│  ├── Data deletion                                $50               │
│  └── Disposal/recycling                           $30               │
│                                          ──────────                 │
│                               Total:            $80                 │
│                                                                      │
│  ════════════════════════════════════════════════════════════════   │
│                                                                      │
│  TOTAL COST OF OWNERSHIP:                                           │
│  Acquisition:          $1,850                                       │
│  Operation (5 yrs):    $4,650                                       │
│  Disposal:                $80                                       │
│                   ──────────                                        │
│  TCO TOTAL:            $6,580                                       │
│                                                                      │
│  → Acquisition is only 28% of TCO!                                  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### ROI vs. TCO

| Metric | Question | Calculates |
|--------|----------|------------|
| **ROI** | Is the investment worthwhile? | Return / Profit |
| **TCO** | What does it cost in total? | All costs |

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ROI vs. TCO                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ROI answers:                      TCO answers:                     │
│  "How profitable is               "What does it cost               │
│   the investment?"                  in total?"                      │
│                                                                      │
│  Used for:                         Used for:                        │
│  → Investment decision             → Budget planning                │
│  → Project evaluation              → Comparison of alternatives     │
│  → Economic justification          → Cost forecasting               │
│                                                                      │
│  TCO can flow INTO ROI:                                             │
│                                                                      │
│         Savings from new solution - TCO of new solution             │
│  ROI  = ──────────────────────────────────────────────── x 100%    │
│                      TCO of new solution                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Key Terms

| Term | Explanation |
|------|-------------|
| **ROI** | Return on Investment - Return on an investment |
| **TCO** | Total Cost of Ownership - Total operating costs |
| **Amortization** | Period until cost recovery |
| **Payback** | Same as amortization period |
| **Break-Even** | Point at which profit is achieved |
| **Useful life** | Planned usage period (e.g., 5 years) |

---

## Exam Tips

1. **Remember ROI formula:**
   - ROI = Profit / Investment x 100%
   - Positive ROI = Investment is worthwhile

2. **Don't forget TCO:**
   - Purchase price is NOT everything!
   - Operating costs often higher than acquisition

3. **Typical exam tasks:**
   - Calculate ROI
   - Name TCO components
   - Determine payback period

4. **Common mistake:**
   - For TCO, multiply operating costs x useful life!

## Practice Exercises

### Exercise 1: ROI Calculation

ConSystem GmbH invests $30,000 in a new ticket system. Through more efficient processes, $8,000 per year is saved.

a) Calculate the ROI after 3 years.
b) Calculate the payback period.
c) Is the investment worthwhile?

<details>
<summary>Show Solution</summary>

a) **ROI after 3 years:**
- Savings: $8,000 x 3 = $24,000
- Profit: $24,000 - $30,000 = -$6,000
- ROI = -$6,000 / $30,000 x 100% = **-20%**

b) **Payback period:**
- $30,000 / $8,000 = **3.75 years**

c) **Assessment:**
- After 3 years: ROI negative → not yet worthwhile
- After 4 years: ($8,000 x 4 - $30,000) / $30,000 = $2,000 / $30,000 = **+6.7%**
- **From the 4th year, the investment is worthwhile.**

</details>

### Exercise 2: TCO Comparison

Compare two alternatives for an ERP system:

| Item | Solution A (On-Premise) | Solution B (Cloud) |
|------|------------------------|-------------------|
| Acquisition | $50,000 | $0 |
| Installation | $10,000 | $5,000 |
| Annual license | $5,000 | $15,000 |
| Maintenance/year | $8,000 | $0 (included) |
| Hardware/year | $2,000 | $0 |

Useful life: 5 years

Which solution has lower TCO?

<details>
<summary>Show Solution</summary>

**Solution A (On-Premise):**
- Acquisition: $50,000
- Installation: $10,000
- Licenses (5 years): $5,000 x 5 = $25,000
- Maintenance (5 years): $8,000 x 5 = $40,000
- Hardware (5 years): $2,000 x 5 = $10,000
- **TCO Solution A: $135,000**

**Solution B (Cloud):**
- Acquisition: $0
- Installation: $5,000
- Licenses (5 years): $15,000 x 5 = $75,000
- Maintenance: $0 (included)
- Hardware: $0
- **TCO Solution B: $80,000**

**Result: Solution B (Cloud) is $55,000 cheaper over 5 years.**

</details>

### Exercise 3: Combined Calculation

A company is considering replacing its old server infrastructure (TCO: $80,000/year) with a cloud solution (TCO: $50,000/year). The migration costs $60,000 one-time.

a) What is the annual savings?
b) After how many years does the migration pay off?
c) What is the ROI after 5 years?

<details>
<summary>Show Solution</summary>

a) **Annual savings:**
$80,000 - $50,000 = **$30,000/year**

b) **Payback period:**
$60,000 / $30,000 = **2 years**

c) **ROI after 5 years:**
- Total savings: $30,000 x 5 = $150,000
- Profit: $150,000 - $60,000 = $90,000
- ROI = $90,000 / $60,000 x 100% = **150%**

**Conclusion:** Very good investment - high ROI, fast payback.

</details>
