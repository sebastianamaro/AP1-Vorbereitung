## Core Content

### The EU Energy Label

The **EU energy label** shows the energy efficiency of electrical appliances at a glance.

```
┌─────────────────────────────────────────────────────────────────┐
│                    EU ENERGY LABEL                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌───────────────────────────────────────┐                    │
│   │  ████████████████████  A  (dark green)│  ← Very efficient  │
│   │  █████████████████  B  (light green)  │                    │
│   │  ██████████████  C  (yellow-green)    │                    │
│   │  ███████████  D  (yellow)             │                    │
│   │  ████████  E  (orange)                │                    │
│   │  █████  F  (dark orange)              │                    │
│   │  ██  G  (red)                         │  ← Less efficient  │
│   └───────────────────────────────────────┘                    │
│                                                                 │
│   Additional information:                                      │
│   • Power consumption in kWh/year or kWh/1000h                │
│   • Product-specific data                                      │
│   • QR code to product database                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

### Energy Efficiency Classes (since 2021)

Since March 2021, the new, simplified label from **A to G** applies (without plus classes).

| Class | Rating |
|-------|--------|
| **A** | Best efficiency |
| **B** | Very good |
| **C** | Good |
| **D** | Average |
| **E** | Below average |
| **F** | Poor |
| **G** | Worst efficiency |

**Note:** A is often empty initially - room for future improvements!

### Old vs. New Scale

```
┌────────────────────────────────────────────────────────────────┐
│            OLD vs. NEW ENERGY CLASSES                          │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   OLD (until 2021)             NEW (from 2021)                │
│                                                                │
│   A+++  ─────────────────►   B or C                           │
│   A++   ─────────────────►   C or D                           │
│   A+    ─────────────────►   D or E                           │
│   A     ─────────────────►   E or F                           │
│   B     ─────────────────►   F or G                           │
│   C-G   ─────────────────►   G (or no longer allowed)         │
│                                                                │
│   The old A+++ class is NO LONGER the best!                   │
│   Class A is reserved for future innovations                  │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

| Device Category | Relevant Data |

### TCO Certification for IT

Since computers don't have an EU energy label, **TCO certification** is important.

```
┌────────────────────────────────────────────────────────────────┐
│                 TCO CERTIFIED                                  │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   TCO Development (Sweden) certifies IT products:             │
│                                                                │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  • Energy efficiency                                    │  │
│   │  • Ergonomics (monitors, keyboards)                     │  │
│   │  • Emissions (electromagnetic, chemical)                │  │
│   │  • Durability and repairability                        │  │
│   │  • Social responsibility in production                  │  │
│   │  • Recyclability                                        │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                │
│   Products: Monitors, laptops, desktops, printers, servers   │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

---

### Energy Star

**Energy Star** is an energy efficiency program, originally from the USA.

| Aspect | Description |
|--------|-------------|
| **Focus** | Energy consumption in operation and standby |
| **Products** | Computers, monitors, printers, servers |
| **Requirement** | Meeting energy thresholds |
| **Standby** | Max. 1-2 Watts |

### Disposal of IT Equipment (WEEE)

IT equipment is **electronic waste** and must be disposed of properly.

```
┌────────────────────────────────────────────────────────────────┐
│              DISPOSAL OF IT EQUIPMENT                          │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│   ┌────────────────────────────────────────────────────────┐  │
│   │  [X]  NOT in household waste!                          │  │
│   └────────────────────────────────────────────────────────┘  │
│                                                                │
│   Correct disposal:                                           │
│                                                                │
│   ┌──────────────────┐  ┌──────────────────┐                 │
│   │ Recycling center │  │ Retailer take-   │                 │
│   │ (municipal)      │  │ back             │                 │
│   └──────────────────┘  └──────────────────┘                 │
│                                                                │
│   ┌──────────────────┐  ┌──────────────────┐                 │
│   │ Manufacturer     │  │ Donation/        │                 │
│   │ take-back        │  │ Reuse            │                 │
│   └──────────────────┘  └──────────────────┘                 │
│                                                                │
│   Important:                                                  │
│   • Delete data first!                                        │
│   • Dispose of batteries separately                           │
│   • For company hardware: Proof of disposal                  │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

| Regulation | Description |
|------------|-------------|
| **WEEE** | Waste Electrical and Electronic Equipment (EU) |
| **National laws** | E-waste laws in each country |

| Criterion | Look For |
|-----------|----------|
| **Energy label** | Prefer class A-C |

## Practice Exercises

### Exercise 1
What energy efficiency classes exist on the new EU label (since 2021)?

**Solution:** The new EU energy label has classes **A to G**:
- A = Best efficiency (often still empty)
- B = Very good
- C = Good
- D = Average
- E = Below average
- F = Poor
- G = Worst efficiency

The old classes A+++, A++, A+ were abolished.

### Exercise 2
A company wants to purchase new monitors. What certifications should be considered?

**Solution:**
- **EU energy label:** Prefer class A-C
- **TCO certification:** For energy efficiency, ergonomics, and environment
- **Energy Star:** For efficient standby consumption
- Additionally: Data on power consumption in kWh/1000h

### Exercise 3
How must old computers and monitors be disposed of?

**Solution:** IT equipment is **electronic waste** and must **NOT go in household trash**!

Correct disposal options:
1. **Recycling center:** Municipal collection points accept e-waste for free
2. **Retailer take-back:** Large retailers must take back devices
3. **Manufacturer take-back:** Many manufacturers offer take-back programs

**Important:** Securely delete data before disposal!
