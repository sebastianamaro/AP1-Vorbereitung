# Quality Management Systems (QMS)

## Learning Objectives

After this chapter you should:
- Know what a quality management system is
- Know the components of a QMS
- Be able to describe the steps for implementing a QMS
- Know the difference between QMS models

---

## Core Content

### What is a QMS?

A **Quality Management System (QMS)** is a structured approach to ensuring and continuously improving quality in organizations. It encompasses:

- **Processes:** Defined workflows
- **Policies:** Rules and guidelines
- **Responsibilities:** Clear accountabilities
- **Documentation:** Traceable records

#### The Five Building Blocks in Detail

| Building Block | Description | Example |
|----------------|-------------|---------|
| **Quality Policy** | Fundamental direction and intentions | "We strive for 100% customer satisfaction" |
| **Quality Objectives** | Concrete, measurable goals | "Error rate below 0.1%" |
| **Organizational Structure** | Structures and responsibilities | Organization chart, QM representative |
| **Operational Processes** | Defined processes and workflows | Process descriptions, work instructions |
| **Resource Management** | Provision of necessary resources | Personnel, IT systems, budget |

### Implementing a QMS

The implementation of a QMS takes place in several steps:

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 1. CURRENT  │ ─► │ 2. TARGET   │ ─► │ 3. QM       │
│    STATE    │    │    STATE    │    │    MANUAL   │
│    ANALYSIS │    │             │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 6. CERTI-   │ ◄─ │ 5. INTERNAL │ ◄─ │ 4. TRAINING │
│    FICATION │    │    AUDITS   │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
```

| Step | Activities |
|------|------------|
| **1. Current State Analysis** | Capture current processes and structures |
| **2. Define Target State** | Establish quality objectives and standards |
| **3. Create QM Manual** | Document processes, establish responsibilities |
| **4. Train Employees** | Familiarize all involved parties with the QMS |
| **5. Internal Audits** | Check compliance with requirements |
| **6. Certification** | External examination by certification body |

### Comparison of QMS Models

| Aspect | DIN EN ISO 9001 | EFQM Model |
|--------|-----------------|------------|
| **Origin** | International standard | European Foundation for Quality Management |
| **Approach** | Requirements-oriented | Self-assessment and excellence |
| **Certification** | Yes, by auditors | No, self-assessment |
| **Focus** | Processes and documentation | Holistic business management |
| **Philosophy** | Meet minimum requirements | Strive for excellence |

---

### The QM Manual

The **QM Manual** is the central document of a QMS:

**Typical contents:**
- Quality policy and objectives
- Organizational structure
- Process descriptions
- Responsibilities
- Procedure instructions
- Forms and checklists

## Key Terms

| Term | Explanation |
|------|-------------|
| **QMS** | Quality Management System - structured approach to quality assurance |
| **Quality Policy** | Fundamental direction of an organization regarding quality |
| **Quality Objectives** | Concrete, measurable specifications for quality |
| **QM Manual** | Central document with all QMS information |
| **Audit** | Systematic review of QMS compliance |
| **Organizational Structure** | Hierarchical structure (who is responsible for what) |
| **Operational Processes** | Process structure (how tasks are completed) |

## Practice Exercises

### Exercise 1
Put the following steps for implementing a QMS in the correct order:
- Certification
- Employee training
- Current state analysis
- Internal audits
- Writing the QM manual

**Solution:**
1. Current state analysis
2. Writing the QM manual
3. Employee training
4. Internal audits
5. Certification

### Exercise 2
Define the terms quality planning and quality policy.

**Solution:**
- **Quality planning:** Establishing quality objectives and measures to achieve these objectives
- **Quality policy:** The fundamental intentions and direction of an organization regarding quality, formulated by top management

### Exercise 3
Name three components of a QMS and briefly explain them.

**Solution:**
1. **Quality policy:** Fundamental quality direction of the organization
2. **Organizational structure:** Structures, roles and responsibilities
3. **Operational processes:** Defined processes and workflows
