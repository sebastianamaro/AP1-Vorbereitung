# Software Types

## Learning Objectives

After this chapter, you should be able to:
- Distinguish between system software and application software
- Name examples of different software types
- Distinguish between standard and custom software
- Know about Open Source and proprietary software

### System Software

**System software** controls and manages hardware and provides the foundation for application software.

| Type | Description | Examples |
|------|-------------|----------|
| **Operating System** | Manages hardware and resources | Windows, Linux, macOS |
| **Drivers** | Enables communication with hardware | Graphics card drivers, printer drivers |
| **Firmware** | Software directly on hardware | BIOS/UEFI, router firmware |
| **Utilities** | System tools | Defragmentation, virus scanner |

---

### Application Software

**Application software** fulfills specific tasks for the user.

| Category | Description | Examples |
|----------|-------------|----------|
| **Office Software** | Word processing, spreadsheets | Word, Excel, LibreOffice |
| **Communication** | Email, messaging | Outlook, Teams, Slack |
| **Industry Software** | Specialized for industries | ERP, CAD, Accounting |
| **Multimedia** | Image, video, audio editing | Photoshop, Premiere |
| **Browser** | Display websites | Chrome, Firefox, Edge |
| **Games** | Entertainment | - |

### Standard vs. Custom Software

```
┌────────────────────────────────────────────────────────────────┐
│          STANDARD vs. CUSTOM SOFTWARE                          │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  STANDARD SOFTWARE                 CUSTOM SOFTWARE             │
│  ┌────────────────────┐          ┌────────────────────┐      │
│  │                    │          │                    │      │
│  │ Developed for      │          │ Tailored for       │      │
│  │ many customers     │          │ one customer       │      │
│  │                    │          │                    │      │
│  │ • Microsoft Office │          │ • Company-         │      │
│  │ • SAP              │          │   specific         │      │
│  │ • Adobe Photoshop  │          │   solution         │      │
│  │                    │          │                    │      │
│  └────────────────────┘          └────────────────────┘      │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

| Aspect | Standard Software | Custom Software |
|--------|------------------|-----------------|
| **Target Group** | Many customers | One customer |
| **Cost** | Lower (distributed among many) | Higher (development) |
| **Customization** | Limited | Fully customizable |
| **Availability** | Immediate | Development time |
| **Updates** | From vendor | Individual |
| **Support** | Standardized | Individual |

### Software-as-a-Service (SaaS)

In addition to traditional installation, today there is **SaaS** - software from the cloud.

| Aspect | Traditional Software | SaaS |
|--------|---------------------|------|
| **Installation** | Local | None (in browser) |
| **Updates** | Manual | Automatic |
| **Payment** | One-time purchase | Subscription |
| **Access** | Only on installed device | From anywhere |
| **Examples** | MS Office (classic) | Microsoft 365 |

## Key Terms

| Term | Explanation |
|------|-------------|
| **System Software** | Manages hardware (OS, drivers) |
| **Application Software** | Fulfills user tasks |
| **Standard Software** | Developed for many customers |
| **Custom Software** | Tailored for one customer |
| **Open Source** | Source code open, freely usable |
| **Proprietary** | Source code closed |
| **SaaS** | Software as a Service |

## Exam Tips

### Common Exam Questions

- What is the difference between system and application software?
- Name advantages and disadvantages of standard software
- What is Open Source?

### Important to Remember

- **System software** = Operating system, drivers (hardware-related)
- **Application software** = Office, ERP, browser (user-related)
- **Standard software** = cheaper, quickly available
- **Custom software** = more expensive, tailored

## Practice Exercises

### Exercise 1
Categorize the following software: Windows 11, SAP, Graphics card driver, LibreOffice

**Solution:**
| Software | Category |
|----------|----------|
| Windows 11 | System software (Operating system) |
| SAP | Application software (ERP) |
| Graphics card driver | System software (Driver) |
| LibreOffice | Application software (Office software) |

### Exercise 2
A company needs accounting software. Name two advantages of standard software and two advantages of custom software.

**Solution:**

**Standard Software:**
1. **Cost:** Cheaper, distributed among many customers
2. **Availability:** Immediately usable, no development time

**Custom Software:**
1. **Customization:** Exactly tailored to company processes
2. **Flexibility:** Can be changed at any time

### Exercise 3
What is the difference between Open Source and Freeware?

**Solution:**
- **Open Source:** Source code is open, can be viewed and modified (e.g., Linux)
- **Freeware:** Software is free to use, but source code is closed (e.g., VLC)
