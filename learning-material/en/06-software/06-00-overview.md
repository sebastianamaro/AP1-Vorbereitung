## Topic Overview

### 06-01 Fundamentals
- Distinguish between system software and application software
- Understand programming paradigms
- Evaluate software according to criteria

### 06-02 Programming
- Variables and data types
- Control structures (conditions, loops)
- Functions and procedures
- OOP fundamentals (classes, objects)
- Writing and reading pseudocode
- **[NEW 2025]** Performing desk checks
- **[NEW 2025]** Finding errors in code

### 06-03 UML
- Creating and reading class diagrams
- Understanding use case diagrams
- **[NEW 2025]** Activity diagrams

### 06-04 Databases
- Understanding relational databases (WITHOUT SQL - SQL is in AP2!)
- Entity-Relationship diagrams
- Normalization and normal forms

### 06-05 Web Development
- HTML basics
- Understanding XML

### 06-06 Development Process
- Software Development Life Cycle
- Test types and test methods
- Debugging techniques

### 06-07 Licenses
- Copyright for software
- Understanding license models
