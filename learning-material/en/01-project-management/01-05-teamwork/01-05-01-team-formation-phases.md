# Team Formation Phases (Tuckman Model)

## Learning Objectives

After this chapter, you should be able to:
- Know the four team formation phases according to Tuckman
- Describe the characteristics of each phase
- Understand how teams develop
- Understand the role of leadership in the different phases

---

## Core Content

### The Tuckman Model

The **Tuckman Model** describes the development of teams in four phases. It was developed in 1965 by Bruce Tuckman and is the most well-known model for team development.

---

### The Four Phases

```
+-------------+    +-------------+    +-------------+    +-------------+
|   FORMING   | -> |  STORMING   | -> |   NORMING   | -> | PERFORMING  |
|             |    |             |    |             |    |             |
| Getting to  |    |  Conflicts  |    |   Rules     |    | Performance |
| know        |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+
```

---

### Phase 1: Forming (Orientation)

**Characteristic:** Getting acquainted and initial orientation

| Aspect | Description |
|--------|-------------|
| **Behavior** | Polite, cautious, wait-and-see |
| **Focus** | Getting to know each other, understanding goals |
| **Feelings** | Uncertainty, curiosity, expectation |
| **Productivity** | Low |
| **Conflicts** | Are avoided |

**Leadership Tasks:**
- Communicate clear goals
- Provide structure
- Promote getting to know each other
- Offer orientation

---

### Phase 2: Storming (Confrontation)

**Characteristic:** First conflicts and disputes

| Aspect | Description |
|--------|-------------|
| **Behavior** | Power struggles, resistance, criticism |
| **Focus** | Clarifying roles, power dynamics |
| **Feelings** | Frustration, disappointment, competition |
| **Productivity** | Low to medium |
| **Conflicts** | Occur openly |

**Leadership Tasks:**
- Moderate conflicts
- Introduce clear rules
- Define roles
- Be patient

**Important:** This phase is normal and necessary for team development!

---

### Phase 3: Norming (Organization)

**Characteristic:** Development of common rules and norms

| Aspect | Description |
|--------|-------------|
| **Behavior** | Cooperative, willing to compromise |
| **Focus** | Common standards, work methods |
| **Feelings** | Belonging, trust |
| **Productivity** | Increases significantly |
| **Conflicts** | Are resolved constructively |

**Leadership Tasks:**
- Support team rules
- Encourage delegation
- Strengthen personal responsibility
- Give feedback

---

### Phase 4: Performing (Performance)

**Characteristic:** Effective, goal-oriented collaboration

| Aspect | Description |
|--------|-------------|
| **Behavior** | Independent, flexible, supportive |
| **Focus** | Task completion, continuous improvement |
| **Feelings** | Pride, satisfaction, connection |
| **Productivity** | High |
| **Conflicts** | Rare, quickly resolved |

**Leadership Tasks:**
- Delegate
- Ensure framework conditions
- Let team celebrate
- Support development

---

### Overview of the Phases

| Phase | Indicators | Team Focus | Leadership Style |
|-------|------------|------------|------------------|
| **Forming** | Uncertainty | Orientation | Directive |
| **Storming** | Conflicts | Clarification | Moderating |
| **Norming** | Rules | Organization | Participative |
| **Performing** | Performance | Results | Delegative |

---

### Dynamics in the Model

**Important Notes:**
- Phases can last different lengths of time
- Teams can fall back into earlier phases (e.g., with new members)
- Not all teams reach the Performing phase
- With Scrum: Each Sprint can contain elements of all phases

```
                    New member?
                    New task?
                          |
                          v
Forming -> Storming -> Norming -> Performing
   ^          |          |            |
   +----------+----------+------------+
              Regression possible
```

## Key Terms

| Term | Explanation |
|------|-------------|
| **Forming** | Orientation phase, getting acquainted |
| **Storming** | Conflict phase, power struggles |
| **Norming** | Organization phase, developing rules |
| **Performing** | Performance phase, effective collaboration |
| **Tuckman Model** | Model of team development in four phases |

## Practice Exercises

### Exercise 1
Name the four team formation phases according to Tuckman in the correct order.

**Solution:**
1. Forming (Orientation)
2. Storming (Confrontation)
3. Norming (Organization)
4. Performing (Performance)

### Exercise 2
In which phase do typically the most conflicts occur?

**Solution:** In the Storming phase (Phase 2)

### Exercise 3
A new team member joins. What effect can this have on team dynamics?

**Solution:** The team can fall back into an earlier phase (usually Forming or Storming), as the group dynamics need to be renegotiated.
